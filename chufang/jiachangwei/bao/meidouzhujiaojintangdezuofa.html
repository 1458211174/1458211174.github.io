      眉豆猪脚筋汤的做法

      时下正值春天与夏交界的时候，人容易出现春困与亚健康状态，适合食用一些补气养血，强身健体，预防疾病的汤水。今天会用到广东家庭煲汤很喜欢的两种食材花生与眉豆，花生煲汤除了可以令到汤水更香浓，还有补肾健脾，补血等作用，并且富含优质蛋白；眉豆也叫白饭豆，它除了祛湿利水，还有健脾益气的作用，而红枣主要的作用是调和各种材料的味道，还有补气养血的作用；干的猪脚筋有补肾填精益髓的作用，而猪脊骨的作用只是起到带出肉香味，起到调味的作用，所以用量不需太多。 煲好的汤水给人的第一视觉感受是汤色很白很浓，入口味浓香甜，加上花生眉豆的植物蛋白，让你有喝豆浆的美妙错觉，而猪脚筋富含的胶原蛋白，更是将汤水的鲜美充分调动出来，喝一口醇香汤水，就好似喝下一口丰富蛋白，或者这正是广东煲汤的美妙之处，既取入口之美味，又取药补之功效

      一、主料

      主料用量
      花生80克
      眉豆80克
      猪脚筋15条
      猪脊骨100克
      红枣8个

      二、辅料

      辅料用量
      姜3片
      清水2500毫升

      三、做法

      步骤
      1.

      少许油入锅，爆香姜片

      |
      |
      2.

      猪脚筋洗净沥干水入锅干煸

      |
      |
      3.

      炒至微透明时，加入清水，关火，浸泡2小时

      |
      |
      4.

      花生与眉豆洗净用清水浸泡

      |
      |
      5.

      猪脊骨洗净用开水焯去血污

      |
      |
      6.

      所有食材放入砂锅中

      |
      |
      7.

      加入2500毫升清水，启动煲汤程序

      |
      |
      8.

      煲好后，可根据自家口味加入适量盐调味

      |

      四、功效

      抗衰老，润肺，止血，心血管病，补虚，保护心脏，促进发育，益智，健脾，治流鼻血。
