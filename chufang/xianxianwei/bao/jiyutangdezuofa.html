      鲫鱼汤的做法

      很喜欢这道汤，奶香浓郁，鲜美可口。

      一、主料

      主料用量
      鲫鱼400克
      纯牛奶100毫升

      二、辅料

      辅料用量
      色拉油40克
      料酒5毫升
      盐5克
      葱1棵
      姜片4片
      水量1200毫升

      三、做法

      步骤
      1.

      鱼腮，内脏一定要清洗干净。

      |
      |
      2.

      洗干净后用厨房用纸把鱼身上的水吸干，免得煎鱼的时候，有生水下锅油溅出来烫伤。

      |
      |
      3.

      鱼两边片几刀容易炖出味，因个人不喜欢吃生姜和葱，所以我都会切大块直接放里面，出锅时好捞出来扔掉。

      |
      |
      4.

      用砂锅提前把水烧开，有没有砂锅不重要，重要的一定要用开水，因为冷水很难把汤炖成奶白色。

      |
      |
      5.

      少放点油，以免汤炖出来混浊

      |
      |
      6.

      油温高一点下鱼，或等锅热了没倒油之前用生姜，擦拭锅底，或直接用不粘锅，主要是避免把鱼皮煎破。煎到鱼两边颜色微金黄。（ps煎的完整的鱼皮炖的汤更白）

      |
      |
      7.

      把生姜、葱稍煸一下，放少许料酒把鱼两边翻一下。

      |
      |
      8.

      把鱼直接倒入砂锅

      |
      |
      9.

      根据个人口味，在砂锅中放适量鲜牛奶，为什么放牛奶？一是汤更白，二是有淡淡的奶香。

      |
      |
      10.

      大火炖十分钟，改小火炖十五分钟，直到汤汁收到你喜欢的浓稠度即可。

      |
      |
      11.

      切记，盐，出锅前再放调口

      |

      四、功效

      增强体质，下奶，益眼，抗皱养颜，防癌，镇静助眠，美容养颜，减肥瘦身。
