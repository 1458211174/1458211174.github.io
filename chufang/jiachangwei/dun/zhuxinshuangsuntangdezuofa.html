      猪心双笋汤的做法

      现在很多妈妈都意识到要经常主动给宝宝添加含铁食物，比较常见效果好的是瘦肉、猪肝。给婴幼儿添加辅食时，只要加一勺肝泥或瘦肉泥，就能有效地供应铁元素，预防婴幼儿常见的缺铁性贫血。除了猪肝之外，鸡肝、鸭肝、鹅肝、羊肝也一样好，鸭胗、鸡心、猪腰子、猪心都很不错，只要是深红色的内脏，都有补铁的作用。猪肝做的比较多，源于它比较好咀嚼，今天的猪心汤，适合2岁半以上的宝宝食用，还嚼不动猪心的宝宝，换成猪肝、鸡肝、鸭肝、鹅肝、羊肝也一样。 肝脏能帮助部分人改善眼睛的健康，因为动物肝脏中维生素A的含量特别高，远远超过奶、蛋、肉、鱼等其他食物，可有效防治夜盲症、干眼病及角膜软化症等眼部疾病。枸杞也有明目、润肺的作用。 为了补血效果好，肝脏是不需要将血水泡掉的，那么很多麻麻反应做出来的汤会腥的喝不下。按照柚子妈今天的方法，用双笋炖猪心，炖出来的汤一点不腥，反而带着冬笋的鲜美，百喝不腻。

      一、主料

      主料用量
      猪心1个
      芦笋4根
      冬笋1个
      枸杞1小把

      二、辅料

      辅料用量
      盐2克
      姜2片
      葱2段

      三、做法

      步骤
      1.

      先将冬笋，切块加水入锅煮。大火煮开转小火。

      |
      |
      2.

      冬笋炖煮期间，将猪心洗洗，切片。在冬笋炖煮10分钟后，加入猪心和2片姜一起煮。
      柚子妈叨叨：
      吃深红色的动物内脏，比如肝脏、肾脏、心脏和脾脏，可以帮助缺铁性贫血的人补铁，因为内脏那紫红色的颜色，是“血红素”所带来的。“血红素”正是内脏、肌肉和血液呈现红色的原因，它的分子中含有铁元素。内脏的颜色深浓，是因为它所含的血红素比肉类中的铁含量更高，其中的铁元素吸收利用率非常高。（柚子妈妈说：这里决定了肝脏买回来是不需要把血水泡掉的，因为那是补铁的关键）

      |
      |
      3.

      猪心入锅10分钟，冬笋和猪心都已经好了。这时候加入一小把枸杞、芦笋、和2根葱。

      |
      |
      4.

      大火煮2分钟就好出锅了。

      |
      |
      5.

      鲜美、喝不厌的补血汤，一点不腥。没有放一滴油，由于猪心的加入，这款汤绝对不寡淡！

      |

      四、功效

      安神睡眠，强心，防癌，清热利尿，健脑，止血，排毒，养肝护肝，助消化，健脾，清热解毒，保护视力，保肝，降三高，抵抗力。
