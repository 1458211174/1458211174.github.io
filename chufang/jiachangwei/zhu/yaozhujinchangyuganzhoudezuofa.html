      瑶柱金鲳鱼干粥的做法

      从家里带来的食材，这些都是母亲给提前晒干的海鲜。 新鲜的的味道都被封存在食物体内，经过浸泡、炒制与熬制之后，释放出来的味道让味蕾从新感受属于湛江海鲜的滋味。 因为女朋友不在家，一个人吃饭，只是做了一人份。其实分量是勉强够两个人吃的。（掩脸）

      一、主料

      主料用量
      大米200克
      金鲳鱼1/3条
      瑶柱20颗

      二、辅料

      辅料用量
      生菜半颗
      花生油2茶匙
      精盐半茶匙
      鸡精少量
      沙姜5片
      葱段少量

      三、做法

      步骤
      1.

      先准备好主材，特别是冰冻的海鲜要先解冻。本人的金鲳鱼干就是冰冻的，有些许冰点在上面。
      当然，也可以用新鲜的金鲳鱼，或者你使用虾或者别的海鲜也是可以的，后面步骤是一样的。

      |
      |
      2.

      首先准备食材，让金鲳鱼干和瑶柱都先泡水，同时洗米，让米泡10来分钟，这样煮出来的米饱满。这段时候可以对食材进行加工。

      |
      |
      3.

      将大米和瑶柱一起放入锅中，大米与水量的比例是1：6，熬制时间久了之后入味还粘稠，不至于干水。

      |
      |
      4.

      这个时候可以准备些小的食材，沙姜切片，葱进行切段处理。前者带有特殊香味，没有普通姜的辣，却可以去腥，后者添香。
      生菜同样进行切块处理。

      |
      |
      5.

      在锅里加上花生油加热，之后把切块的金鲳鱼干进行炒制。翻炒至稍微金黄即可。

      |
      |
      6.

      将稍微金黄的金鲳鱼干连着油一起倒入煮沸的洲里进行熬制。

      |
      |
      7.

      在金鲳鱼熬制20分钟后可以加入生菜继续熬制5分钟即可。

      |
      |
      8.

      加入葱段，放入精盐还有鸡精，再稍微煮个2分钟后关火。
      如果你希望更加入味，可以不开锅再闷个三两分钟。
