      醪糟鸡蛋的做法

      醪糟也叫酒酿、糯米酒，常喝醪糟不仅对健康人有益，而且对一些慢性病有辅助医疗效能。患有慢性萎缩性胃炎的病人及消化不良的人，常喝醪糟可以促进胃液分泌，增加食欲，帮助消化。醪糟还有提神及解除疲劳的功效。 今天做了醪糟鸡蛋，是一道很有名气的小吃呢！

      一、主料

      主料用量
      江米（糯米）500克
      鸡蛋1个

      二、辅料

      辅料用量
      甜酒曲2克

      三、做法

      步骤
      1.

      将江米（糯米）浸泡6个小时以上，我是提前一晚浸泡的（夏季可缩短浸泡时间）蒸锅里放入足量的水，将屉布清洗干净（要保证无油的），放上浸泡好的江米，平铺开，蒸锅上汽后蒸30分钟；

      |
      |
      2.

      将蒸好的江米，倒回料理盆里自然晾凉后，用温水将甜酒曲稀释后，倒入凉凉的江米里；

      |
      |
      3.

      搅拌均匀，尽量让每粒米都沾上甜酒曲，中间挖个小坑，包好保鲜膜放在温暖处发酵24小时以上；

      |
      |
      4.

      发酵好后会有浓郁的、甜甜的米酒香，也可以直接食用了。

      |
      |
      5.

      煮锅里水冒小泡时打入一个鸡蛋（这时不要搅动鸡蛋），待逐渐冒大泡时关火，盖上锅盖焖熟鸡蛋；

      |
      |
      6.

      将煮好的鸡蛋放入碗里，再取适量的醪糟放入，放枸杞装饰一下即可食用了。

      |

      四、功效

      健脑益智，保肝，动脉硬化，防癌，延年益寿，美容护肤。
