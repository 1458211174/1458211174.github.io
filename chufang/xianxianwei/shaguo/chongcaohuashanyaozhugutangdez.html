      虫草花山药猪骨汤的做法

      说到“虫草花”不知你们是不是和我一样会想到声名赫赫的“冬虫夏草”，其实两者有着本质的区别。虫草花实质上是虫草子的实体属于真菌类,与常见的香菇、平菇等食用菌很相似，只是菌种、生长环境和生长条件不同。 为了跟冬虫草区别开来，商家起了一个美丽的名字，把它叫做“虫草花”，虫草花外观上最大的特点是没有了“虫体”，而只有橙色或者黄色的“草”。 虫草花性质平和，不寒不燥，对于多数人来说都可以放心食用。它所富含的虫草酸和虫草素能够综合调理人机体内环境，对增强和调节人体免疫功能、提高人体抗病能力有一定的作用。和山药一起煲汤有益肝肾、补精髓、特别适合经常加班熬夜的上班族用来滋补身体。

      一、主料

      主料用量
      虫草花10克
      猪骨头600克
      山药300克
      矿泉水1500毫升

      二、辅料

      辅料用量
      葱白1小段
      姜1小块
      料酒5毫升
      枸杞20粒左右
      盐1克

      三、做法

      步骤
      1.

      材料图。

      |
      |
      2.

      山药去皮后切段，用冷水浸泡待用。葱白切段，老姜切片，枸杞子冲洗干净控水备用。

      |
      |
      3.

      虫草花冲洗干净，用200ml的冷水浸泡15分钟。

      |
      |
      4.

      猪骨头斩件，冷水下锅，加姜片、葱段和料酒大火烧开，转中火煮15-20分钟，然后捞出用温水冲洗干净骨头表面的残渣和浮沫。

      |
      |
      5.

      将焯水后的汤骨、葱段、姜片、虫草花和泡虫草花的水一起放进电炖锅内，加入1500ml的水，选择“大骨汤”功能，按下启动键。

      |
      |
      6.

      待剩余时间提示为30分钟时加入山药段，调入约1g的盐，继续煲。

      |
      |
      7.

      待电炖锅发出“滴滴”的提示音后撒上冲洗过的枸杞子即可饮用。

      |

      四、功效

      骨质疏松，滋肾益精，降血糖，延年益寿，助消化。
