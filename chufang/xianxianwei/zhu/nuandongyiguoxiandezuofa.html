      暖冬一锅鲜的做法

      冬天真的只要一锅汤汤水水的菜就可以了，暖暖的，大家围坐在一起吃就是一种幸福

      一、主料

      主料用量
      鹌鹑蛋15个
      贡丸130克
      虾仁100克
      粉丝100克
      泡发黑木耳100克
      枸杞10克
      青菜100克
      姜10克
      葱10克
      料酒15克

      二、辅料

      辅料用量
      盐8克
      生抽2克
      白胡椒3克
      油15克

      三、做法

      步骤
      1.

      准备好美的电磁炉和食材

      |
      |
      2.

      鹌鹑蛋放入美的电磁炉锅中开启烧汤功能煮5分钟

      |
      |
      3.

      鹌鹑蛋煮好后剥壳粉丝冷水泡发，葱姜切丝

      |
      |
      4.

      鹌鹑蛋煮好后剥壳

      |
      |
      5.

      在美的电磁炉锅中倒入食用油，放入葱姜丝炒下

      |
      |
      6.

      加入虾仁炒制

      |
      |
      7.

      加入水，葱，枸杞煮沸

      |
      |
      8.

      加入各种食材，调料，粉丝放中间就不会糊
