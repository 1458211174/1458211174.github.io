      墨鱼萝卜丸子汤的做法

      冬天是吃白萝卜的季节，且白萝卜中含有丰富的维生素A、维生素C等各种维生素，特别是叶子中维生素c的含量是根茎的4倍以上。维生素c能防止皮肤的老化，阻止黑色色斑的形成，保持皮肤的白嫩。配墨鱼和丸子煲汤很好喝哟。

      一、主料

      主料用量
      墨鱼(干)2条
      猪肉丸600克
      白萝卜1条

      二、辅料

      辅料用量
      盐适量

      三、做法

      步骤
      1.

      白萝卜切成滚刀块

      |
      |
      2.

      把猪肉丸洗干净，这个是在超市买的。

      |
      |
      3.

      把白萝卜，猪肉丸子，墨鱼全部放进汤煲里，放入适量的水

      |
      |
      4.

      加几片姜，去腥味儿

      |
      |
      5.

      大火把汤煲开后转小火慢煲。

      |
      |
      6.

      大火煲开后，转小火，一共煲了30分钟左右

      |
      |
      7.

      汤煲好了哦，放点盐，调味即可享用
