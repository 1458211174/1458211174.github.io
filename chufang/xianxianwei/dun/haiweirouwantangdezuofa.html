      海味肉丸汤的做法

      儿子女儿周末休息，做点爱吃的。

      一、主料

      主料用量
      猪肉(肥瘦)750克
      馒头末一小盘

      二、辅料

      辅料用量
      白菜（不要帮）少许
      紫菜少许
      虾皮少许
      葱、姜少许
      花椒水少许
      香菜一根
      榨菜少许
      盐、糖少许
      香油几滴

      三、做法

      步骤
      1.

      碎的颗粒要均匀。

      |
      |
      2.

      用温开水泡半小时。

      |
      |
      3.

      白瘦三七分肉馅分别依次放葱姜末、料酒少许、生抽少许、白糖一勺、盐适量，顺一个方向搅拌，白糖然后分两次把花椒水倒入，让肉馅逐渐上劲儿。

      |
      |
      4.

      充分搅拌均匀。

      |
      |
      5.

      手沾点凉白开水，团搅好的肉馅，肉丸大小自定。

      |
      |
      6.

      烧开水，依次下入白菜丝、紫菜、肉丸，开锅三次关火，依次放入榨菜碎、香菜末，出锅装汤碗，倒几滴香油，上桌！

      |

      四、功效

      补肾，润肺。

      • 炖
      • 咸鲜味
