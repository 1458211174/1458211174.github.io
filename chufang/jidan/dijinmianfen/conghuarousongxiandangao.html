      葱花肉松咸蛋糕

      对官老师而言却是在从事餐饮业多年后才有机会在一次海线班上课时… 学员买了当地知名的梧栖小镇咸蛋糕给我才知道有这样的产品^.^ 第一口的感觉是…………………很特别………….我想我还是认为蛋糕就本就应该是甜的吧^.^ 但我想在我的食谱中可是没有国界之分更何况这只是县界之分吧! 于是呢?今天就来教大家如何用家里的电锅做出美味的咸蛋糕吧!

      一、材料

      鸡蛋6ea

      细砂糖200g

      低筋面粉160g

      沙拉油40ml

      肉松100g

      葱花50g

      开水60ml

      油葱酥30g

      二、做法

      步骤
      首先要准备制作蛋糕体的材料如下: 鸡蛋 / 细砂糖 / 低筋面粉 / 沙拉油 / 肉松 / 葱花 / 开水 / 油葱酥
      将蛋白加入砂糖打发至浓稠状态如左图/
      慢慢的加入水使其融蛋白中
      待打发至蛋白周围形成不易愈合的纹路时加入蛋黄
      待蛋黄加入完毕后再加入沙拉油拌匀
      拌入过筛后的低筋面粉
      形成完美的流线形面糊状态
      取一蒸气盘并铺上烤焙纸倒入面糊 /撒上一半的红葱头酥 / 葱花 / 肉松
      再加入另一半的面糊 /同样的做法再重复一次
      放入电锅上层 / 记得盖上锅盖喔!时间 / 25分
