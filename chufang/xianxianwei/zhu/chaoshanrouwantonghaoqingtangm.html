      潮汕肉丸茼蒿清汤面的做法

      正宗的潮汕人家家常清汤面

      一、主料

      主料用量
      潮汕猪肉丸一人份
      潮汕猪肉丸汤一人份
      生面条适量
      茼蒿菜适量

      二、辅料

      辅料用量
      炸蒜泥适量
      盐适量
      辣酱适量
      食用油适量

      三、做法

      步骤
      1.

      潮汕猪肉丸和肉丸汤倒入锅中并煮至沸腾

      |
      |
      2.

      茼蒿菜洗干净，晾干水备用

      |
      |
      3.

      等肉丸汤沸腾后放入生面条煮到水再次沸腾后倒入凉白开，再次煮至沸腾。这样煮面条才会筋道

      |
      |
      4.

      水再次沸腾后加入茼蒿菜，并撒入盐调味，加入食用油使油包裹茼蒿菜就可以盛出来，可以开吃了！

      |
      |
      5.

      在盛出的面条上撒上炸蒜泥，辣酱增加口味就可以了
