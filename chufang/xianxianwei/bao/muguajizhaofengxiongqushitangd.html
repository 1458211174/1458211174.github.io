      木瓜鸡爪丰胸去湿汤的做法

      在单位摘了两个木瓜，想着回来煲汤。木瓜和鸡爪最配了。

      一、主料

      主料用量
      木瓜500克
      鸡爪6只

      二、辅料

      辅料用量
      芡实10克
      薏米10克
      姜2片
      盐1勺

      三、做法

      步骤
      1.

      鸡爪对半切开，洗净焯水

      |
      |
      2.

      木瓜去皮切块

      |
      |
      3.

      备好芡实，薏米

      |
      |
      4.

      姜切片

      |
      |
      5.

      高脚汤锅里放入鸡爪，芡实，薏米，姜片，大火煮开

      |
      |
      6.

      倒入木瓜，小火煲60分钟

      |
      |
      7.

      加盐，出锅

      |

      四、功效

      健脾消食，驱虫，防癌，强身健体，高血压，养颜护肤。
