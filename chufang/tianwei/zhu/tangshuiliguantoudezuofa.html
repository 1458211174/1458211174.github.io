      糖水梨罐头的做法

      买了好多梨，吃不了快要坏了。赶紧做罐头吧！ 自己做的罐头纯天然，没有添加剂，浓浓的梨香味儿。过年的时候打开盛出来，给家宴的桌上摆个饭后甜品，那种感觉很有范儿啊！

      一、主料

      主料用量
      梨5个
      冰糖100克

      二、辅料

      辅料用量
      纯净水1瓶

      三、做法

      步骤
      1.

      梨子去皮去核切块

      |
      |
      2.

      装入用开水煮过消了毒的罐头瓶里

      |
      |
      3.

      从梨块缝隙之间把冰糖放进去。放不进去的糖放在上面就可以

      |
      |
      4.

      倒入纯净水

      |
      |
      5.

      放入凉水锅里，盖子浮盖，不要拧住盖子。开火煮20分钟

      |
      |
      6.

      关火，拧紧盖子，倒扣在台面上。

      |
      |
      7.

      放凉后翻过来，用手指戳一下盖子中间，如果凹下去就是真空已经形成，可以存放半年没有问题。反之，就要尽快吃完。

      |

      四、功效

      降压，润肺化痰，开胃，防癌，润肠通便，清热解暑，化痰止咳，清热去火，抵抗力，高血压。
