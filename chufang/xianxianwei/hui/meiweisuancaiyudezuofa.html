      美味酸菜鱼的做法

      美味酸菜鱼，随心搭配，满足味蕾！

      一、主料

      主料用量
      鲜鱼1条

      二、辅料

      辅料用量
      酸菜1袋
      生粉少许
      千张2张
      金针菇300克
      葱段若干
      花椒若干
      干辣椒若干
      食用盐10克
      胡椒粉少许
      十三香少许

      三、做法

      步骤
      1.

      鱼切块状，适当大小，加入盐，胡椒粉，十三香，少许生粉，拌匀腌制10～15分钟左右

      |
      |
      2.

      葱切段备用，少许葱切碎备用，花椒，干辣椒少许备用（依个人口味适量加入）。

      |
      |
      3.

      千张切宽带状，金针菇根部用温水浸泡。（配菜可根据个人喜好随意添加）

      |
      |
      4.

      可选择酸菜包，亦或是自家炮制酸菜。

      |
      |
      5.

      锅热倒油，加入葱碎，花椒，干辣椒微微煸炒出味，加入酸菜翻炒，直至飘出香味。

      |
      |
      6.

      酸菜煸出香味，加入清水，以能够淹没鱼块为宜。开大火。

      |
      |
      7.

      水开后加入腌制好的鱼块，继续炖。

      |
      |
      8.

      水煮沸后加入葱段（可依个人口味继续加入花椒和干辣椒）大火三分钟后转小火慢炖。

      |
      |
      9.

      加入配菜，配菜熟后即可关火出锅。
