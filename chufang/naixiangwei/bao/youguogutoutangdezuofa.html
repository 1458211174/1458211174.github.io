      油果骨头汤的做法

      油果骨头汤，十分美味

      一、主料

      主料用量
      油果10个

      二、辅料

      辅料用量
      猪排骨2条
      红枣5个
      枸杞8粒
      姜3片
      香菇(干)8个
      米酒1小勺
      白萝卜1个
      盐看个人
      桂圆肉4个

      三、做法

      步骤
      1.

      买回来的菜洗一次

      |
      |
      2.

      砍好的猪排骨和切好的白萝卜放到煲汤锅里

      |
      |
      3.

      把油果，枸杞，姜，米酒，盐，桂圆肉，香菇，红枣也放到煲汤锅里，然后用自己的碗来按加水，（我是3个人吃，我放5碗水）

      |
      |
      4.

      最后开始煲汤，按30分钟

      |
      |
      5.

      半小时后就可以吃啦！

      |
