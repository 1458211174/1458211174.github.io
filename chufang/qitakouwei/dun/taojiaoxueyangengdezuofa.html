      桃胶雪燕羹的做法

      桃胶，又名桃花泪，具有清热，止渴，养颜抗衰的功效；雪燕，即天山雪莲花分泌的汁液，可以预防呼吸道感染，抑制癌细胞生长。炖上一盅桃胶雪燕羹，滋阴养肺，又清血降脂，满满的胶原蛋白，美颜嫩肤，特别的适合这个雾霾的冬日。

      一、主料

      主料用量
      桃胶20克
      雪燕10克
      冰糖适量

      二、辅料

      辅料用量
      椰奶300毫升
      水1碗
      清水2000毫升

      三、做法

      步骤
      1.

      桃胶20克

      |
      |
      2.

      雪燕10克

      |
      |
      3.

      桃胶雪燕分别加入1000毫升清水，泡发24小时，期间更换3次水，并把杂质挑出

      |
      |
      4.

      泡发好的桃胶和雪燕倒入炖盅，加入一小碗水，搅拌均匀

      |
      |
      5.

      而后盖上盖子，电蒸锅蒸炖功能，定时一小时

      |
      |
      6.

      炖煮20分钟后掀开盖子，加入适量冰糖，继续炖煮

      |
      |
      7.

      炖好的桃胶雪燕羹盛碗食用

      |
      |
      8.

      也可加入椰奶调配食用

      |

      四、功效

      清热解暑，润肺化痰，化痰止咳，清热去火，抵抗力，高血压。
