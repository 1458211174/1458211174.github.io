      清淡冬瓜汤的做法

      冬瓜，不仅清热，还是减肥及美容的很好食材，所以也比较适合现在的我. 这碗冬瓜汤，制作超级简单，也有人可能就要来吐槽我了，但是，您知道的，不一定其他人都知道怎么做，所以我还是要分享出来。

      一、主料

      主料用量
      冬瓜200克

      二、辅料

      辅料用量
      葱5克
      香菜3克

      三、做法

      步骤
      1.

      材料。

      |
      |
      2.

      冬瓜切片、大葱和香菜切碎备用。

      |
      |
      3.

      平底锅刷层油，下入葱煸炒出香味。

      |
      |
      4.

      倒入冬瓜煸炒一下。

      |
      |
      5.

      然后加入适量热水煮。

      |
      |
      6.

      待冬瓜煮成透明，撒入少量盐、放入香菜即可。

      |

      四、功效

      减肥瘦身，滋养肾气，化痰，防癌，润肤美容，降血糖，降胆固醇，降血脂，软化血管，助减肥，降压。
