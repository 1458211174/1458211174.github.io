      水果糙米粥的做法

      糙米中米糠和胚芽部分含有丰富的B族维生素和维生素E,提高人体免疫功能，促进血液循环，还能帮助人们消除沮丧烦躁的情绪。 糖尿病患者适宜常食糙米，因为糙米中的碳水化合物被粗纤维组织包裹，人体消化吸收速度较慢，因而能很好地控制血糖； 此粥人群均可食用，尤适于肥胖，胃肠功能障碍、贫血、便秘者食用。

      一、主料

      主料用量
      糙米100克
      白糯米50克
      梨30克
      苹果30克
      猕猴桃30克
      葡萄干10克

      二、辅料

      辅料用量
      红枣10克
      黄冰糖30克
      水800毫升

      三、做法

      步骤
      1.

      食材准备好；

      |
      |
      2.

      糙米和白糯米、红枣、葡萄干分别洗净；

      |
      |
      3.

      将糙米和白糯米倒入砂锅中，加入水，盖上盖子；

      |
      |
      4.

      以大火烧开后，转中小火煮30分钟；

      |
      |
      5.

      苹果、梨、猕猴桃分别去皮，切成小块备用；

      |
      |
      6.

      煮30分钟后，再加入冰糖、红枣、葡萄干煮20分钟；

      |
      |
      7.

      最后加入切好的苹果、梨、葡萄干煮10分钟即可。

      |
      |
      8.

      四、功效

      脂肪肝，散瘀止痛，治贫血，助睡眠，减肥瘦身，防癌，开胃，心血管病，防过敏，糖尿病，降压，润肺化痰，润肠通便，中风，抵抗力，降血脂，强身健体，防贫血，安神助眠。
