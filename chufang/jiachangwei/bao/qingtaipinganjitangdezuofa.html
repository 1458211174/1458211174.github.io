      清泰平安鸡汤的做法

      我们老广的团年饭一定要有鸡，寓意吉利。鸡年煲个应景应节的传统鸡汤。 这个汤清甜可口，有补脾养血的桂圆肉和红枣，再加上苹果、百合的寓意，象征着“清泰平安”。

      一、主料

      主料用量
      鸡400克
      苹果半个
      干百合12克
      红枣5个
      桂圆肉10克

      二、辅料

      辅料用量
      盐1茶匙

      三、做法

      步骤
      1.

      备好材料

      |
      |
      2.

      百合、红枣、桂圆洗净稍浸泡一会

      |
      |
      3.

      鸡肉焯水

      |
      |
      4.

      锅中加水1600ml，放入鸡肉、百合、红枣、桂圆，大火烧开后转中小火煲半小时

      |
      |
      5.

      苹果去核切块

      |
      |
      6.

      加入苹果再煲半小时就可以加盐喝靓汤了

      |

      四、功效

      脂肪肝，防癌，中风，润肠通便，抵抗力，减肥瘦身，治流鼻血，健脑。
