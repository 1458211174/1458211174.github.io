      酸萝卜魔芋鸭的做法

      冬天来了 要腌制今年的新泡萝卜了！ 发现泡菜坛子里面还有陈萝卜！ 突然就想炖鸭子了！ 为了给新萝卜腾地方！ 也为了美味！ 呵呵 就是这样！

      一、主料

      主料用量
      鸭肉500克
      魔芋300克
      泡萝卜250克

      二、辅料

      辅料用量
      泡辣椒70克
      泡姜100克
      料酒40克
      花椒20克
      胡椒1小勺
      味精1茶匙
      生抽酱油20克

      三、做法

      步骤
      1.

      准备材料 这里特别要说的是魔芋 现在很多菜市场卖的魔芋都不够纯真颜色偏淡 我买的这种是一家老店卖的颜色黝黑味道纯真！

      |
      |
      2.

      先把魔芋切成小长方条 放在锅里加水点火汆水 水沸后煮3分钟即可 这样做事为了减少魔芋的碱味 增加口感

      |
      |
      3.

      鸭子改刀切块 洗净 放入锅中加水 加入少量老姜 料酒 不要焯太长时间 以免影响肉质

      |
      |
      4.

      把焯完水的鸭肉洗净（多习2次把血沫洗干净） 备用

      |
      |
      5.

      泡萝卜切一指厚的片 泡生姜切薄片 备用

      |
      |
      6.

      泡椒剁碎 备用

      |
      |
      7.

      点火热锅 转中大火倒油烧至7成热 倒入泡椒煸炒30秒

      |
      |
      8.

      倒入鸭肉 转中小火翻炒3分钟 不要翻的太过频繁

      |
      |
      9.

      到入泡萝卜 泡姜继续翻炒2分钟 然后关火 转入炖锅

      |
      |
      10.

      加适量的水 加入味精 胡椒 生抽（因为泡菜是咸的的所以不加盐 如果你的泡菜不咸 可以加入适量的盐）

      |
      |
      11.

      点火转中大火煮沸后 再转小火炖煮1小时（如果是老鸭子的话那炖煮时间至少从1个半小时起 不然不容易炖软 肉质会很硬 不好吃）
