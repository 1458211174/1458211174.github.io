      金针百页羊肉粥

      澄黄的金针就像秋日太阳暖和 ( ( ( 老爸的爱 ) ) ) 也很暖和

      一、材料

      米半碗

      干金针适量

      百页豆腐一条

      秋葵适量

      炒羊肉适量

      盐少许

      二、做法

      步骤
      干金针泡水 / 秋葵去蒂 / 百页豆腐切片 / 米洗净备用
      秘密武器是老爸为了出门在外打拼的我，将事先炒好的羊肉冷冻后拿给我的 ( ( ( 老爸的爱 ) ) ) ^.^
      白粥滚沸后，先将百页豆腐呈放射状铺陈，中间圆心的部份放入炒好的羊肉
      再将秋葵等距间隔呈放射状摆上，间隔中再放入泡好的金针，泡过金针的水也倒入锅中再一同慢火滚沸后，撒点盐即可
