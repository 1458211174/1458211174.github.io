      消食萝卜汤的做法

      微信公众号“宝宝营养辅食”，收录了我给宝宝做过的500道营养辅食，有了这500道辅食菜谱，不再为宝宝吃啥而发愁。小思妈妈会在这个公众号分享6-24个月的宝宝辅食。 宝宝添加辅食后，除了担心宝宝不爱吃饭，也会担心宝宝积食。如果宝宝出现大便干燥、口臭等现象，就有可能是积食了。 虽说宝宝不爱吃饭不一定代表积食，但积食的宝宝确实也有不爱吃饭这个症状呢？在宝宝积食初期，最好以食疗的方法来调理，比如用上促进消化的三种食材来煮一份消食萝卜汤。 白萝卜、山楂都是非常好的食疗食材。白萝卜能加速肠道蠕动，增进食欲，帮助消化。山楂有助于宝宝的消化，可以治疗肠胃积食、消化不良，能开胃健脾，增加食欲。

      一、主料

      主料用量
      白萝卜60克
      陈皮3克
      山楂片10克

      二、辅料

      辅料用量
      清水600毫升

      三、做法

      步骤
      1.

      准备食材：白萝卜60g、陈皮3g、山楂片10g。

      PS：虽然在配料表里面没有加冰糖，但如果宝宝觉得成品有些酸的话，可以少加一点点冰糖来调节。

      |
      |
      2.

      白萝卜切片 。

      PS：白萝卜切成薄薄的小片。

      |
      |
      3.

      陈皮浸泡20分钟，去掉陈皮和山楂表面的浮灰，然后把陈皮清洗干净。

      PS：陈皮含有类柠檬苦素，这种类柠檬苦素味平和，易溶解于水。此外，陈皮还含有挥发油、橙皮甙、维生素B、C等成分，可促进宝宝消化液的分泌，排除肠管内积气，增加食欲。

      |
      |
      4.

      山楂片也是跟陈皮相同的处理方法。

      PS：清洗山楂跟陈皮的方法一样。如果买不到新鲜的山楂，可以到药店去看看，是否能够买到晒干的山楂片和陈皮。

      |
      |
      5.

      将所有的食材都倒入锅中。

      |
      |
      6.

      加600ml左右的清水。

      PS：虽说这个加水量并不是固定的，但我也是反复尝试过几遍，觉得这个水量比较适中。如果水太少，熬出的山楂萝卜水会比较浓，宝宝们不太喜欢喝。

      |
      |
      7.

      大火烧开后，转小火继续煮10－15分钟。

      |
      |
      8.

      这个做法是不是非常地简单呢，熬好之后就可以取出来给宝宝喝啦。这道饮品只适合喝汤，里面的食材不要吃哦。

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体。
