      鲫鱼火腿汤的做法

      家里有密云野生鲫鱼还有金华火腿，做过鲫鱼豆腐汤、鲫鱼蘑菇汤，第一次吃金华火腿，突发奇想做了一个鲫鱼火腿汤，汤很浓，因为放了味极鲜，卖相不太好，味道凑合，感觉有些暴殄天物 了-_-#

      一、主料

      主料用量
      鲫鱼1条
      金华火腿3片
      香菇3朵

      二、辅料

      辅料用量
      葱1段
      木耳若干
      小平菇3朵
      蒜1头
      姜1块
      花椒若干
      八角若干
      植物油若干
      料酒若干
      醋若干
      味极鲜若干

      三、做法

      步骤
      1.

      洗净备用，鱼用料酒，盐，姜片腌制30分钟。

      |
      |
      2.

      火腿用焯水去咸味。

      |
      |
      3.

      少许油煎鱼

      |
      |
      4.

      锅内的油炒制准备的配料，加醋、料酒、味极鲜。建议用白醋米醋，不要放酱油、我用了香醋，味极鲜，汤料黑了，影响色泽。

      |
      |
      5.

      所有配料加入砂锅，加热水，大火烧开后转小火炖60分钟。

      |
      |
      6.

      出锅加入香葱，味精和香油

      |
