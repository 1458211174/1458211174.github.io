      补肾六黑粥的做法

      黑色入肾，今天一款六黑粥选用了六种黑色食物，其妙处嘛嘿嘿嘿～坏笑

      一、主料

      主料用量
      黑米30克
      黑豆10克
      黑花生10克
      黑木耳3朵
      黑糖5克
      熟黑芝麻10克

      二、辅料

      辅料用量
      清水2碗

      三、做法

      步骤
      1.

      黑木耳提前泡发，洗净后切成丝

      |
      |
      2.

      黑米，黑豆洗净备用

      |
      |
      3.

      黑豆黑米黑花生黑木耳丝黑糖一起备好

      |
      |
      4.

      锅中烧开水，将步骤3中的五黑一起入锅熬制

      |
      |
      5.

      在熬五黑的时候，把熟黑芝麻捣碎备用

      |
      |
      6.

      五黑熬好后出锅，把捣好的芝麻撒上即可。

      |

      四、功效

      美容养颜，防癌，保护心脏，心血管病，延年益寿，补钙，排毒，健脑，养发，软化血管，保肝，抵抗力，强身健体，减肥瘦身。
