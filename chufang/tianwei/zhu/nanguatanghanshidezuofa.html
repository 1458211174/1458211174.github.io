      南瓜汤 韩式的做法

      家里有好多南瓜…我只喜欢这种吃法

      一、主料

      主料用量
      南瓜泥600毫升
      开水300毫升
      糯米粉少量

      二、辅料

      辅料用量
      碎冰糖少量

      三、做法

      步骤
      1.

      南瓜洗干净去子

      |
      |
      2.

      切块

      |
      |
      3.

      放电饭煲隔水蒸30分钟，一定要熟透

      |
      |
      4.

      煮好，怕烫就冷却会，不怕烫就去皮

      |
      |
      5.

      去皮，勺子挖就好，很轻松的去皮

      |
      |
      6.

      用勺子压碎

      |
      |
      7.

      加入80ml-100ml的水，不要太多

      |
      |
      8.

      打汁机打成泥

      |
      |
      9.

      过滤，比较费事，耐心啊耐心

      |
      |
      10.

      过滤完成

      |
      |
      11.

      放入150ml-200ml的水(分三次搅拌均匀)，让南瓜汤变滑顺

      |
      |
      12.

      开中小火煮个两分钟转小火，分次放入少量的糯米粉要搅拌均匀哦(量的话感觉勺子在搅拌过程中变重了变稠了，就可以了)

      |
      |
      13.

      因为南瓜本来就甜放入少量的碎冰糖调下味就可以了，凭个人口味，南瓜汤小火煮滚就好喝了

      |
      |
      14.

      完工
