      西兰花浓汤的做法

      花儿印象里，小时候吃的菜花，一直是白色的。直到上高中后，才知道有种菜花是绿色的，名为西兰花。西兰花为1-2年生草本植物，原产于地中海东部沿岸地区，现已成为日常主要蔬菜之一。西兰花营养丰富，含蛋白质、糖、脂肪、维生素和胡萝卜素，营养成份位居同类蔬菜之首，被誉为“蔬菜皇冠”。 　　在凉拌沙拉、蒜蓉素炒，都少不了用到西兰花。而在西方，尤其是欧洲国家，父母经常让孩子食用的一种醇香又芳香的浓汤，就是用西兰花做的。

      一、主料

      主料用量
      西兰花500克
      淡奶油150克
      黄油10克

      二、辅料

      辅料用量
      盐适量
      黑胡椒粉少许

      三、做法

      步骤
      1.

      西蓝花掰成小块，洗净。

      |
      |
      2.

      锅烧热，放入黄油至黄油融化成液态，放西蓝花翻炒片刻。

      |
      |
      3.

      锅内加水，水量没过西蓝花即可。沸腾后，转小火煮10分钟。

      |
      |
      4.

      把西蓝花连汤带水放入搅拌机或料理机，搅拌成稀糊。

      |
      |
      5.

      再把汤糊放回锅里，加入淡奶油，小火加热2分钟即可。

      |
      |
      6.

      再放入适量盐，研磨少许黑胡椒粉，拌匀即可。

      |

      四、功效

      防癌，瘦身减肥，促进生长，软化血管，清热解渴，抗衰老，强身健体，促发育。
