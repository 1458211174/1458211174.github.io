      双肉双蔬粥的做法

      一个人给自己懒惰的理由

      一、主料

      主料用量
      米1小碗
      瘦肉1块
      西芹1根
      蘑菇6个
      火腿肠1根

      二、辅料

      辅料用量
      盐1勺
      姜片4片
      香葱2根
      料酒1勺

      三、做法

      步骤
      1.

      瘦肉洗净，切小条，倒料酒。生姜片切末，加盐，抓匀。静置

      |
      |
      2.

      淘米，加适量水，熬白粥，熬至20分钟左右，加肉

      |
      |
      3.

      西芹切丁，火腿肠切丁

      |
      |
      4.

      煮10分钟

      |
      |
      5.

      粥里加火腿丁、加蘑菇。煮至粘稠，加盐、加葱花

      |

      四、功效

      补肾，润肺，肝火，镇静助眠，利尿，防癌，贫血，散瘀止疼，安神助眠，止咳化痰，通便排毒，促进食欲，抵抗力。
