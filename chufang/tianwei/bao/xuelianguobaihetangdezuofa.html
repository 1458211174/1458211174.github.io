      雪莲果百合汤的做法

      这是一道清甜不油腻的汤。

      一、主料

      主料用量
      雪莲果500克
      龙骨400克

      二、辅料

      辅料用量
      胡萝卜100克
      莲子30克
      干百合30克
      清水1000毫升

      三、做法

      步骤
      1.

      清洗干净雪莲果和胡萝卜，然后切成块。

      |
      |
      2.

      老公也用清水清洗干净。

      |
      |
      3.

      莲子百合用清水稍微洗一下，放入锅里。

      |
      |
      4.

      把所有材料一起放入电饭锅内胆，准备煲汤。

      |
      |
      5.

      按煲汤功能，大约一个小时之后汤好了稍等几分钟就可以开锅喝汤了。

      |

      四、功效

      通便，抵抗力，去火，糖尿病，调理血液，解毒。
