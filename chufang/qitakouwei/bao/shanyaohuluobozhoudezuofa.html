      山药胡萝卜粥的做法

      淮山药很少吃，但是它对身体非常好，所以干脆熬粥来吃

      一、主料

      主料用量
      大米100克
      淮山药一根
      胡萝卜1到2根

      二、辅料

      辅料用量
      食盐3克
      水砂锅的三分之一处

      三、做法

      步骤
      1.

      泡50分钟到一个小时， 这样是为了让大米快速软糯。也可以下午吃中午就泡上

      |
      |
      2.

      淮山药一根，削皮，如果碰到粘液发痒，削山药可以戴上一次性手套，胡萝卜同样去皮，两根胡萝卜是做菜剩下的，所以就一起放了，略多，您做的时候其实一根就够了

      |
      |
      3.

      将胡萝卜和山药切丁，胡萝卜比较慢切，小心手，可以先给圆的胡萝卜侧切一块，然后再切大概6毫米左右厚块，不容易因为胡萝卜滚动而伤手，山药可以稍微大一些，避免熬住时间过长消失

      |
      |
      4.

      大概这样大小即可

      |
      |
      5.

      我是没等水开就将大米捞出（剩下的米汤女士可以拿来当爽肤水或者做面膜，很好用）然后大火煮开

      |
      |
      6.

      水开后，换成小火

      |
      |
      7.

      用小火熬煮，用木勺不断搅动粥，避免溢出

      |
      |
      8.

      继续煮5到10分钟

      |
      |
      9.

      继续熬煮

      |
      |
      10.

      30分钟后，米软烂，锅内米汤呈糊状即可，加入少量食盐，就可以出锅了

      |

      四、功效

      适合婴儿，强身健体，抵抗力，降糖，保护视力，助消化。
