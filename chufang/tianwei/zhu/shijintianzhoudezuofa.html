      什锦甜粥的做法

      冬日饮食重在“滋养温补”，肠胃保护首当其冲，喝粥无疑是最佳选择。经常喝粥不但可以减轻消化系统负担，还能达到调胃气、补脾虚、壮筋骨、和五脏、通血的脉养生效果，尤其以晨起空腹食用效果最佳。今天这道甜粥，原料多样，营养丰富，味道香甜，家人和孩子都特别喜欢~~

      一、主料

      主料用量
      大米50克
      糯米25克
      蜜枣50克
      无花果（干）50克

      二、辅料

      辅料用量
      桃胶15克
      皂角米15克
      陈皮1片
      冰糖40克
      蔓越莓干15克

      三、做法

      步骤
      1.

      准备原料

      |
      |
      2.

      桃胶、皂角米、无花果干洗净后浸泡一天至完全泡发

      |
      |
      3.

      大米和糯米淘洗干净

      |
      |
      4.

      将所有原料倒入美的电饭煲内胆（冰糖和蔓越莓干除外）

      |
      |
      5.

      倒入大半锅清水

      |
      |
      6.

      电饭煲通电，选择煮粥功能

      |
      |
      7.

      粥煮到快好时，加入冰糖和蔓越莓干

      |
      |
      8.

      继续熬至程序结束，香喷喷的粥就可以喝了

      |

      四、功效

      适合婴儿，强身健体，增强体质，健脾胃，缓解尿频，养血安神，补虚养颜，补血养气，抵抗力，健脾养胃，润肺，健脾。
