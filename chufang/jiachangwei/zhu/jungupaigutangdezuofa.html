      菌菇排骨汤的做法

      不懒床早点起，给小宝还有婆婆煲一款汤，然后用这汤再来给全家人配煮一份面。是得呢朋友们这几天可以看到好多款这样子的汤，其实我是想说生活只需一点点的改变，那就可以成全许多我们漏想到的地方。我不是夸自己，我真的是按着自己的方法，用最走心的家常菜来给家人带来一天最美的感觉。

      一、主料

      主料用量
      排骨750克
      干菌菇10克

      二、辅料

      辅料用量
      面条200克
      细盐适量
      酱油适量

      三、做法

      步骤
      1.

      备好所用食材，锅里加清水，将泡洗好的什锦菌菇下锅，再另起锅将排骨焯洗好备用。

      |
      |
      2.

      锅里加少许油将焯好的排骨下锅炒制出味，再加入适清水煮至排骨出鲜。

      |
      |
      3.

      将排骨连同汤汁一并转至在汤锅里，再加入适量姜提味，煮至汤底大开再转小火煮40分钟使汤底出味，加入适量细盐及酱油提味即可。

      |
      |
      4.

      再锅里加入开水煮开，将面下名煮熟，一半的汤汁用于配面。

      |
      |
      5.

      这样子一份排骨什锦菌菇汤就完成。

      |
      |
      6.

      早餐也可以美美吃到鲜汤面了。

      |

      四、功效

      补肾，润肺。
