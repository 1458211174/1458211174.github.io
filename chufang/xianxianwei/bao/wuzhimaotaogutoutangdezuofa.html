      五指毛桃骨头汤的做法

      最喜欢五指毛桃的清香了，还没有煲汤，光温材料就一股椰子香迎面而来。 《中华药典》记载，五指毛桃味辛甘、性平、微温，其功能：具有益气补虚、行气解郁、壮筋活络、健脾化湿、止咳化痰等功效。

      一、主料

      主料用量
      猪扇骨450克
      五指毛桃200克

      二、辅料

      辅料用量
      盐5克
      清水1500毫升

      三、做法

      步骤
      1.

      清洗五指毛桃，然后用清水浸泡半小时。

      |
      |
      2.

      猪扇骨清洗干净。

      |
      |
      3.

      把所有材料放入电饭锅，准备煲汤，加入适量清水和盐。按煲汤功能即可。这款骨头汤本身清香甘甜，无需加鸡精调味。

      |
      |
      4.

      一个小时后，汤煲好了。表面泛着的可是扇骨的骨油哦，看着油腻，其实喝起来可香了。骨油可以补钙哟！

      |

      四、功效

      骨质疏松，止咳化痰。
