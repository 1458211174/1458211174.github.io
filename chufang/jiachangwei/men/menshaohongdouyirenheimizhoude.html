      焖烧 红豆薏仁黑米粥的做法

      懒人养生

      一、主料

      主料用量
      薏仁20克
      黑米50克
      红豆20克
      花生10克

      二、辅料

      辅料用量
      蜂蜜1勺

      三、做法

      步骤
      1.

      泡豆四小时

      |
      |
      2.

      刚开的热水温杯三分钟

      |
      |
      3.

      高火转2分钟

      |
      |
      4.

      焖烧一夜，约七小时

      |
      |
      5.

      黑米薏仁红豆都烂了，但是花生还有点脆。比不上电饭煲煮的软糯口感，但胜在适合懒人养生。
