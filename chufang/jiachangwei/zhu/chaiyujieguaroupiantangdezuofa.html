      柴鱼节瓜肉片汤的做法

      小时候很喜欢妈妈煲的柴鱼花生粥，我相信每个广州人都吃过这个粥，很香很好吃。我这个人比较好吃，但又吃不胖，所以从小就到厨房看妈妈做菜，二年级就可以做一桌子的菜等妈妈下班一起吃，当然都是家常菜，慢慢就喜欢上了做菜，而且很喜欢做给家人和朋友吃，看到别人喜欢吃自己做的菜很有自豪感，而这道菜是一次上餐馆吃饭时看到的，所以回来自己做，方法很简单，上网看过也有这道菜的介绍，是一道非常简单又好吃的家常菜，对脾胃虚弱的人很有好处哦

      一、主料

      主料用量
      柴鱼干适量
      节瓜一条
      猪肉片适量

      二、辅料

      辅料用量
      食盐适量
      花生油适量
      豆粉适量
      糖适量
      酱油适量
      料酒适量
      姜片适量

      三、做法

      步骤
      1.

      节瓜用刀背刮去毛皮清洗干净切片备用.

      |
      |
      2.

      柴油用手撕开小块用清水浸泡十五分钟.

      |
      |
      3.

      肉先用酱油盐豆粉油糖料酒腌制十分钟，分量可以偏多一点，因为汤底不再放其他调味

      |
      |
      4.

      姜去皮切成片备用

      |
      |
      5.

      热锅下油，放入姜片爆香，放入切好的节瓜翻炒至出味，倒入两人份量的净水，再将浸泡好的柴鱼一同放下，盖上锅盖大火煮开转中火煮十分钟，煮至汤底有点发白即可，然后到入腌过的肉片捞匀，盖上锅盖再次煮开等一两分钟关火，将汤倒入陶瓷锅里就可以享用了。

      |
      |
      6.

      选用陶瓷锅是因为陶瓷能很好的保温，等你做好其他送菜时，汤还是热热的

      |

      四、功效

      去火，抵抗力，利尿。
