      黑木耳白菜肉骨汤的做法

      花鱼儿家的菜菜来了。。。

      一、主料

      主料用量
      黑木耳10克
      白菜325克
      肉骨610克

      二、辅料

      辅料用量
      料酒2汤勺
      盐2勺
      味精0.5勺

      三、做法

      步骤
      1.

      食材：黑木耳（已泡发清洗）、白菜（已清洗切开）、肉骨（已清洗焯水）

      |
      |
      2.

      将已清洗焯水好的肉骨和黑木耳下入高压锅中，加入一些清水。

      |
      |
      3.

      加2汤勺料酒。

      |
      |
      4.

      接着，盖上盖用中小火烧煮20分钟后关火。

      |
      |
      5.

      焖至没气后开盖。

      |
      |
      6.

      然后，合入切好的白菜翻动一下煮开，煮至白菜熟透。

      |
      |
      7.

      最后，加2勺盐。

      |
      |
      8.

      加0.5勺味精。

      |
      |
      9.

      调味翻匀即成。

      |

      四、功效

      强身健体，排毒，减肥瘦身，抵抗力，解渴利尿，润肠通便，消食。
