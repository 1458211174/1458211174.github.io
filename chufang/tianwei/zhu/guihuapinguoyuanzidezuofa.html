      桂花苹果圆子的做法

      这次做的圆子里，放入了有芝麻馅的小汤圆，加上自己腌的糖桂花，特别的好吃。

      一、主料

      主料用量
      芝麻馅小汤圆20个
      小圆子70克
      苹果1个

      二、辅料

      辅料用量
      冰糖适量
      桂花1勺

      三、做法

      步骤
      1.

      准备好两种不同的圆子。

      |
      |
      2.

      锅里放入适量水，冰糖。

      |
      |
      3.

      把圆子放入锅内，盖上锅盖，煮开。

      |
      |
      4.

      苹果去皮，切丁。

      |
      |
      5.

      等圆子全部浮起来。（浮起来的圆子说明已经煮熟了）

      |
      |
      6.

      放入一勺桂花。

      |
      |
      7.

      放入切好的苹果丁。

      |
      |
      8.

      关火，搅拌均匀即可。
