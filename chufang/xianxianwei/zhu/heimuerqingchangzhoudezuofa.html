      黑木耳清肠粥的做法

      黑木耳可是公认的肠道清道夫，想要排毒排垃圾就要多吃黑木耳哦～～ 白菜含有大量水分，多吃更是好处多多。这道粥只加了少许盐调味，但是非常清淡可口，很适合大餐过后的亲哦～～制作也非常简单！

      一、主料

      主料用量
      白菜200克
      大米150克
      黑木耳10小朵

      二、辅料

      辅料用量
      盐适量
      清水700克

      三、做法

      步骤
      1.

      提前1小时将黑木耳泡发。

      |
      |
      2.

      将大米淘洗干净，放入苏泊尔养生壶中，加入5倍左右的清水。启动养生汤模式，定时60分钟。

      |
      |
      3.

      将泡发好的黑木耳洗净，切成丝。

      |
      |
      4.

      待粥煮开10分钟后，加入黑木耳，一起煮。

      |
      |
      5.

      将白菜洗净，切成末。

      |
      |
      6.

      待养生汤模式还剩10分钟时，加入白菜末一起煮。

      |
      |
      7.

      最后加入适量盐调味即可。

      |

      四、功效

      抵抗力，解渴利尿，润肠通便，消食，适合婴儿，强身健体，排毒，减肥瘦身。
