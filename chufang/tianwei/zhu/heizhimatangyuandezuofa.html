      黑芝麻汤圆的做法

      没过正月十五都是年。所以今天给大家带来一道既有颜值又有爱的汤圆。 还记得那首“每逢佳节倍思亲”吗，虽然已经上班了，但是心里却还是惦念着佳节多么希望能和家人一起围在桌边，说说笑笑，包汤圆、吃汤圆呀。 坦言现在的年味和小时候相比似乎逊色了好多，没有此起彼伏的鞭炮声了，有的只是隔着冰冷的电视屏幕看着电视中满天空的烟花；没有各色灯笼高高挂了，更没有猜灯谜，玩游戏，有的只是和三五朋友小聚，穿行在各大商圈，看着琳琅满目的商品，却没有了忆昔的期待。 妈妈总是告诉我，无论时间怎么变迁，传统还是要有的，元宵节就要吃汤圆，吃猪尾巴（我们老家的习俗）。而且每年元宵节的馅儿都是妈妈亲手制作的，每到这个时候我们家就像举行仪式一样，一点也不马虎。买食材，准备糯米粉，炸油渣，拌馅，包汤圆，有条不紊的进行。那场景丝毫不逊于国家大厨。 趁着年味还未散去，赶紧做了一碗记忆中的黑芝麻汤圆，趁着这碗热气，抓住2017年年的尾巴吧。

      一、主料

      主料用量
      糯米粉200克

      二、辅料

      辅料用量
      抹茶粉10克
      紫薯粉10克
      桂花1勺
      猪板油80克
      黑芝麻粉100克
      白糖80克

      三、做法

      步骤
      1.

      1.将锅烧热倒入黑芝麻小火煸炒出香味，放凉后用料理机打磨成粉备用

      |
      |
      2.

      2.将锅烧热，把猪板油切成小块放入锅中不停翻炒，直至颜色发黄，油基本渗出成为油渣，捞出放凉备用

      |
      |
      3.

      3.将炸好的油渣、备好的黑芝麻粉、白糖均匀混合扮成馅儿，直至油渣、白糖、黑芝麻粉均匀混合，然后搓成小圆球备用

      |
      |
      4.

      4.开始和面，分别把抹茶粉、紫薯粉放入温水中调均，然后一边倒糯米粉一边搅拌均匀

      |
      |
      5.

      5.将表面揉搓光滑的面团静置30分钟

      |
      |
      6.

      6.取适量的面团在手中搓匀，捏成小碗状，然后将准备好的馅儿放入面团中，慢慢的收口，两手揉至表面光滑即可。

      |
      |
      7.

      7.将揉好的汤圆放入带有糯米干粉的盘中（以免粘盘）备用

      |
      |
      8.

      8.锅内倒入适量的清水煮沸，将汤圆逐个下入锅中，待汤圆完全浮起后，再煮2分钟即可出锅。吃的时候撒上自己腌制的桂花，那滋味更美妙。

      |

      四、功效

      抵抗力，健脾。
