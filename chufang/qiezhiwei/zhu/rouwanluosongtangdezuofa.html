      肉丸罗宋汤的做法

      小睿哥：老婆，我要吃肉～ 汐汐：小睿哥同学，你在减肥…… 小睿哥：我要吃肉

      一、主料

      主料用量
      洋葱半个
      西红柿1个
      土豆1个
      胡萝卜1个
      猪肉200克
      西生菜3片

      二、辅料

      辅料用量
      精盐10克
      酱油适量
      鸡精5克
      法香适量
      干香茅3片
      番茄酱50克
      黑胡椒2克
      鼠尾草适量(没有也可以)
      鸡蛋1个
      面粉20克

      三、做法

      步骤
      1.

      土豆，洋葱，胡萝卜，西红柿，切块炒一下，加盐，鸡精

      |
      |
      2.

      加入番茄酱炖熟

      |
      |
      3.

      加入香茅

      |
      |
      4.

      加入鼠尾草煮10min

      |
      |
      5.

      榨汁机打碎

      |
      |
      6.

      调制肉馅：盐，鸡精，鸡蛋，面粉加到猪肉馅里。

      |
      |
      7.

      鸡蛋

      |
      |
      8.

      清水，汆成丸子。煮的丸子油少

      |
      |
      9.

      将打好的罗宋汤，倒入锅中加热，加入丸子，大概炖10-20分钟，入味出锅

      |
      |
      10.

      加入法香～还可以加入芝士粉

      |

      四、功效

      抵抗力，消食，降血压，补钙，防癌，延年益寿，高血压，祛斑，血栓，美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，降糖，保护视力，助消化，减肥，助睡眠，利尿消肿。
