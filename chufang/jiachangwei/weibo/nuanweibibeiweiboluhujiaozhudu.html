      暖胃必备——微波炉胡椒猪肚煲的做法

      大过年的，煲一锅热热的胡椒猪肚煲来暖胃暖心吧！

      一、主料

      主料用量
      猪肚1个
      猪肉200克

      二、辅料

      辅料用量
      姜一小块
      蜜枣1个
      白胡椒粒1勺
      水1000克

      三、做法

      步骤
      1.

      猪肉切粒备用；猪肚里外用淀粉和盐反复清洗3遍，焯热水后，把瘦肉粒、白胡椒粒塞入猪肚中，用牙签封好。

      |
      |
      2.

      微波适用盘中加入蜜枣、姜片、猪肚和水，加盖留孔，入微波高火10分钟。用手把猪肚里的空气挤出，这样可以有利于猪肚浸没在水中。

      |
      |
      3.

      转文火继续加热50分钟，揭盖，把猪肚捞出切条。

      |
      |
      4.

      连同瘦肉粒和白胡椒放回汤中继续用高火加热5分钟，最后根据个人口味调入食盐即可。
