      酥肉汤的做法

      家常菜

      一、主料

      主料用量
      鸡胸肉250克
      黄心菜4棵
      香菇1个
      豆干2片

      二、辅料

      辅料用量
      食盐适量
      生抽1勺
      红薯淀粉3勺
      葱1棵
      芝麻油1勺
      辣椒油1勺

      三、做法

      步骤
      1.

      选择带有脆骨的下面部分，洗净切小块

      |
      |
      2.

      不需要剁太细腻，保留些脆骨的颗粒感更好

      |
      |
      3.

      鸡肉馅里加盐，生抽，淀粉拌匀

      |
      |
      4.

      小块的放入已经烧热的油锅里，炸至微黄即可

      |
      |
      5.

      锅里加盐煸香葱花后加水煮开

      |
      |
      6.

      炸好的鸡肉酥肉与豆干先放锅里煮3分钟

      |
      |
      7.

      待鸡肉煮可以了放入黄心菜，芝麻油即可出锅

      |

      四、功效

      抵抗力，养肾，消食，延年益寿，防癌，强身健体，降压，降血脂，降胆固醇，软化血管，补钙，缓解更年期，预防疾病，防乳腺癌，有益大脑。
