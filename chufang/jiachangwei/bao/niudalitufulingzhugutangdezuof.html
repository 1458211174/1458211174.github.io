      牛大力土茯苓猪骨汤的做法

      牛大力土茯苓猪骨汤是一道传统药膳，属于粤菜系。祛湿清热，活络筋骨，止咳。广州比较潮湿，常喝有利祛湿利尿，若有水肿的朋友，常喝有益。身上常长湿疹，湿气重，痰多的人，喝这个汤也很好的，土茯苓的主要功效就是去湿，清热解毒。汤里的云苓和赤小豆均是春天袪湿必备材料。

      一、主料

      主料用量
      土茯苓150克
      牛大力80克
      赤小豆70克
      云苓50克
      蜜枣2个
      猪骨350克

      二、辅料

      辅料用量
      水2.5L

      三、做法

      步骤
      1.

      土茯苓和牛大力洗净，土茯苓已请档主去外皮，牛大力用力刮下外皮

      |
      |
      2.

      土茯苓用刀剁小块

      |
      |
      3.

      赤小豆，云芩，蜜枣洗净

      |
      |
      4.

      猪骨焯水洗净

      |
      |
      5.

      瓦煲里将2.5升水煲沸，加入上述所有材料，开始20分钟中大火煲，转中小火继续煲1小时即可。

      |

      四、功效

      通便排毒，消除水肿，健脾，下奶，抵抗力，甲状腺，降压，养血安神，补虚养颜，补血养气，强身健体，健脾养胃，骨质疏松。
