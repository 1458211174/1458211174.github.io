      野苋菜头袪湿汤的做法

      广东的春天3-5月份最烦人了，绵绵春雨下不停，空气中的湿度大，人也容易湿气重。体内湿气过重，会让人觉得困倦、身体四支沉重、没有食欲、手脚冰冷、皮肤起疹、甚至出现肠胃炎现象，口腔症状比较明显，舌苔白厚。野苋菜头有清热解毒、袪湿的功效，很适合广东春天的潮湿天气喝，希望2018年的年历里有我的广东春天菜谱，谢谢

      一、主料

      主料用量
      野苋菜头400克
      猪骨400克
      薏米50克
      扁豆50克
      蜜枣2个
      姜10克

      二、辅料

      辅料用量
      水2.5L

      三、做法

      步骤
      1.

      野苋菜头就是长这个样子的，根也一起煲汤

      |
      |
      2.

      野苋菜头剁短，粗的干剁短纵向开边

      |
      |
      3.

      准备扁豆和薏米。买回来在锅里炒赤黄了，晾凉装起备用，这样能改变它寒性的性质，而且不易生虫子。

      |
      |
      4.

      猪骨瘦肉焯水洗净

      |
      |
      5.

      瓦煲里将2.5升水煮沸，放入所有材料

      |
      |
      6.

      再放入两颗蜜枣和姜片，盖盖中大火煲20分钟，转中小火煲40分钟至1小时即可

      |

      四、功效

      骨质疏松，防癌，美白美容，抵抗力，糖尿病，解热镇痛，治脚气病，清热去火，助消化，强身健体，消食开胃，贫血，养血安神，补虚养颜，补血养气，健脾养胃，预防感冒，散瘀止疼，治肠炎，开胃。
