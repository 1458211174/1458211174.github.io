      猪肉洋葱馅饼

      一、材料

      面粉200g / 水100g / 酵母1g

      里脊肉200g / 紫洋葱1个 / 盐3g / 生抽10g

姜末少许 / 蒜末少许 / 蚝油5g / 酱油10g

      二、做法

      步骤
      1.面粉和酵母装和面盆中，一边加水一边用筷子搅动，待看不到干粉时，用揉手成光滑面团，静放20分钟左右
      2.里脊肉用清水冲洗过后，剁成肉馅，剁好的肉馅加入盐，姜蒜末，蚝油，生抽，酱油拌匀。如果觉得馅有些干，加入少许食用油和清水继续拌匀
      3.洋葱洗干净，切成细末，加入肉馅中搅拌均匀
      4..取一块面团，搓成圆柱状，再切成剂子。剂子上撒些干粉防粘连
      5.取其中一个面团，擀制成圆形面皮，在面皮中央放入猪肉洋葱馅
      6.捏合边缘，褶皱尽量捏小，包好的馅饼平放在案板上，轻拍几下，用擀面杖轻轻擀制成圆形薄饼
      7.锅中放油，放入包好的馅饼，收口朝上，在锅中按扁按圆，小火，一面金黄翻面,馅饼鼓起可出锅
