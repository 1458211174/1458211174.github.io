      花胶石斛煲竹丝鸡汤的做法

      小试身手～

      一、主料

      主料用量
      花胶20克
      石斛5克
      竹丝鸡（乌鸡）半只

      二、辅料

      辅料用量
      姜片2-3片
      盐适量

      三、做法

      步骤
      1.

      按人数准备花胶，我用的是50头鳕鱼胶！

      |
      |
      2.

      用清水清洗浸泡一下

      |
      |
      3.

      注意锅要无油噢

      |
      |
      4.

      花胶放入滚水，焯约1分钟左右

      |
      |
      5.

      准备一个无油保鲜盒，装入冷水或冰水，放入焯好的花胶和姜片，泡发12小时以上。期间勤换水和姜片（2-3次）

      |
      |
      6.

      铛铛，泡发好了!

      |
      |
      7.

      食材一同倒入锅中

      |
      |
      8.

      食材倒入汤锅，炖煮2个小时左右。

      |
      |
      9.

      完工

      |

      四、功效

      缓解疲劳，润肺。
