      韩式海带汤的做法

      韩剧中，每当生日必喝海带汤。在韩国，海带汤是韩国孕妇产后的补品，同时代表着韩国人对母亲的敬意，纪念母亲生育的痛苦，韩国人认为藻类有益产后的调理，于是有食海带的传统，海带汤被视为滋补圣品，价格便宜又营养丰富。 海带含丰富的蛋白质，维他命a，矿物质，易于吸收，对身体有莫大的益处。另外，海带含可溶性纤维，比一般纤维更容易消化吸收，今天就来做一锅海带汤吧。

      一、主料

      主料用量
      海带50克
      肉馅（瘦）50克
      葱适量
      盐少许
      生抽适量
      料酒适量
      食用油适量
      姜末适量

      二、辅料

      辅料用量
      家乐热炒鲜露
      适量

      三、做法

      步骤
      1.

      准备食材，干海带提前泡发，清洗干净。

      |
      |
      2.

      猪肉馅用料酒生抽腌制10分钟。

      |
      |
      3.

      锅内倒入适量食用油，下入腌好的肉馅翻炒。

      |
      |
      4.

      肉变色后加入葱姜末爆香。

      |
      |
      5.

      加入海带翻炒均匀。

      |
      |
      6.

      滴入适量家乐热炒鲜露。

      |
      |
      7.

      加入适量水，熬煮20分钟即可，出锅前撒盐调味。

      |

      四、功效

      利尿，防病，御寒，防癌，防甲亢，抗辐射，促进食欲，老年痴呆，抵抗力，感冒，降三高，延年益寿，散瘀止痛，排毒。
