      老鸭汤的做法

      天气冷了，每天下班回到家就想喝一晚汤。今天在家试试老鸭汤。这个汤老人小孩都喜欢，而且还可以多吃一些菜呀！

      一、主料

      主料用量
      鸭肉1000克
      萝卜500克

      二、辅料

      辅料用量
      老鸭汤包1包

      三、做法

      步骤
      1.

      把材料准备。

      |
      |
      2.

      把鸭肉焯水，去掉血水。

      |
      |
      3.

      把萝卜去皮，改刀成小块状。

      |
      |
      4.

      把焯水好的鸭肉放入电压力锅里。

      |
      |
      5.

      把萝卜放在鸭肉的表面。

      |
      |
      6.

      往电压力锅加入清水，然后放入老鸭糖。

      |
      |
      7.

      电压力锅调好烹饪时间。

      |
      |
      8.

      一个小时侯就可以食用。

      |

      四、功效

      清热去火，消食，保护肠胃，预防疾病，强身健体。
