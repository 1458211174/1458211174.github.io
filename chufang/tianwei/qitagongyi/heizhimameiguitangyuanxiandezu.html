      黑芝麻玫瑰汤圆馅的做法

      团团圆圆，甜甜蜜蜜！过年必做汤圆，黑芝麻汤圆馅最经典，什么馅儿也比不了黑芝麻香吧。先做好馅儿备用，添加了一些自制玫瑰酱，玫瑰香的甜香混合着黑芝麻的纯香美味！汤圆馅的材料和用量不固定根据自家的口味搭配哦，喜欢桂花口味的就放桂花糖，或添加花生和各种坚果做成不同风味的汤圆。芝麻和各种材料碾碎混合，加入糖、蜂蜜、油和少许的熟糯米粉捏成团就OK！当然用猪油更香啦，我家喜欢素油版的。

      一、主料

      主料用量
      黑芝麻100克
      玫瑰酱50克
      细砂糖100克
      糯米粉（熟）20克
      玉米油或香油20克

      二、辅料

      辅料用量
      蜂蜜适量

      三、做法

      步骤
      1.

      黑芝麻和糯米粉炒熟，可以多炒一些备用。

      |
      |
      2.

      将黑芝麻倒入破壁机内打碎（或放食品袋中用擀面杖碾碎），倒入容器中。

      |
      |
      3.

      倒入容器中加入细砂糖拌匀

      |
      |
      4.

      加入玫瑰酱、糯米粉

      |
      |
      5.

      加入油，把材料抓匀（戴上一次性手套方便操作）

      |
      |
      6.

      最后加入适量的蜂蜜调一下

      |
      |
      7.

      能捏成团状态就可以了

      |
      |
      8.

      然后分成10一个的小剂子

      |
      |
      9.

      全部做好放冰箱冷冻保存，随吃随取非常方便。

      |
      |
      10.

      好香啊！

      |
