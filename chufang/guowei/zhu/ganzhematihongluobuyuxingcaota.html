      甘蔗马蹄红萝卜鱼腥草汤的做法

      天气干燥，喉咙不舒服，用当季的新鲜鱼腥草、甘蔗、马蹄、红萝卜和瘦肉煲个汤，清肺润肺，甚好。

      一、主料

      主料用量
      鱼腥草新鲜鱼腥草50克
      甘蔗2小段
      红萝卜1个
      马蹄带皮马蹄6个
      瘦肉100克

      二、辅料

      辅料用量
      纯净水1000毫升

      三、做法

      步骤
      1.

      瘦肉汆水洗净去血水，新鲜鱼腥草（折耳根)洗净待用。

      |
      |
      2.

      甘蔗、马蹄洗净；红萝卜去皮切滚刀状。

      |
      |
      3.

      依次将鱼腥草、瘦肉、甘蔗、马蹄、红萝卜放入炖盅，注入纯净水。

      |
      |
      4.

      用电炖锅，加盖隔水炖3小时。

      |
      |
      5.

      加适量盐花调味即可。

      |

      四、功效

      清热解毒，甲状腺，抵抗力，健脾，通便，促发育，降压，通便排毒，补肾，润肺。
