      玉米山药猪蹄汤的做法

      一入初冬，特别是南方，天气就变得很是干燥，在这个时候尽量多煲点汤水喝，不仅可以滋补营养，还可调节口味，增强体质，提高免疫力等。 新鲜玉米上市了，味道清甜，含有丰富的植物纤维素，具有刺激胃肠蠕动、加速排泄的特性，可防治便秘、肠炎等；山药，味甘，性平，润肺、健脾、益肾，它含有淀粉酶、多酚氧化酶等物质，有利于脾胃消化吸收功能，是平补脾胃的药食两用之品。两种食材加上肉类，煮点养胃健脾的汤水喝喝，既能清凉解暑，又能补养身体，储备能量，预备过冬。 处理好食材，电炖锅通上电，定不会辜负你几小时的等待，美味自来！肉不柴不腻，汤鲜中带点甜，山药软糯糯的，玉米甜脆脆的，一碗下肚还嫌不够，呵呵~麻麻，能再添一碗吗？

      一、主料

      主料用量
      猪蹄1000克
      甜玉米300克
      山药150克

      二、辅料

      辅料用量
      生姜20克
      干辣椒4个
      八角1个
      料酒20毫升
      盐3克

      三、做法

      步骤
      1.

      猪蹄半只剁小块，清洗杂毛，杂质。

      |
      |
      2.

      锅中加入，滴少许料酒，几片姜，入猪蹄焯水。

      |
      |
      3.

      煮出血沫子。

      |
      |
      4.

      焯水后的猪蹄，在流动的水下冲洗干净，沥水备用。

      |
      |
      5.

      电炖锅中，加入生姜七八片，干辣椒4个，八角一个。

      |
      |
      6.

      沥水后的猪蹄，一次性加足开水，通电用慢火炖。

      |
      |
      7.

      甜玉米两根，山药一根，切段备用。

      |
      |
      8.

      两小时后加入甜玉米，山药，再炖两小时。

      |
      |
      9.

      加入盐调味，断电出锅即可。

      |

      四、功效

      美容养颜，抗衰老，促进生长，防冠心病，防癌，滋肾益精，降血糖，延年益寿，助消化。
