      胡萝卜鸡蛋面的做法

      既营养又美味，宝贝们很喜欢吃

      一、主料

      主料用量
      面条（生）200克
      鸡蛋2个
      胡萝卜半根
      瑶柱（元贝干）50克
      虾仁（干）50克

      二、辅料

      辅料用量
      水70毫升
      芝麻油适量
      精盐适量

      三、做法

      步骤
      1.

      准备食材，胡萝卜切丝

      |
      |
      2.

      把已泡发的元贝干虾米熬大约半小时成高汤

      |
      |
      3.

      鸡蛋打散备用

      |
      |
      4.

      用另外一个锅烧水，水开了放入面条煮熟捞起

      |
      |
      5.

      放油炒鸡蛋，备用

      |
      |
      6.

      放油炒胡萝卜，

      |
      |
      7.

      胡萝卜炒熟再把已炒好的鸡蛋放入一起，放适量盐

      |
      |
      8.

      装盘最后淋点芝麻油

      |

      四、功效

      抵抗力，健脑益智，保肝，动脉硬化，防癌，延年益寿，美容护肤，降糖，保护视力，助消化。
