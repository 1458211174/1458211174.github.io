      腊八粥的做法

      一般不提醒，我是记不住农历的。前天与我母亲电话，她说后天是腊八节，她去庙里捐点钱喝腊八粥，还要给不能去的一位邻居带回一碗。嘱咐我记得喝腊八粥。今天早上一家三口吃了自己煮的腊八粥。昨晚用了家里现有的食材，电炖锅提前煲上30分钟，然后保温一宿。早上起来洗漱完了就能吃，超简单。喝了热乎乎的腊八粥出门上班很舒服。

      一、主料

      主料用量
      粳米25克
      麦仁25克
      薏米25克
      莲子15克
      红豆25克
      枣(干)8颗
      桂圆干20克
      冻玉米粒100克
      西米25克

      二、辅料

      辅料用量
      冰糖25克

      三、做法

      步骤
      1.

      家里没有糯米，还有自己剝的老玉米粒冻在冰箱里的，用上了。

      |
      |
      2.

      将食材淘洗干净倒入电压力锅内胆中，红枣和桂圆干先不加入

      |
      |
      3.

      盖上盖子，调整好气阀，按下”豆类”键，开始煲煮。时间到，会有滴滴滴声提醒。等气消投入红枣、桂圆干和冰糖，盖上改下保温即可。

      |
      |
      4.

      早上关电源。盛出就可喝腊八粥了。
