团团圆圆爆浆芝心丸子的做法

假期快到了，春节是团圆的节日，传统要是些团团圆圆的食物，可以使名字上有意思或者外形是圆的，不管什么食物都寄寓着希望一家人团团圆圆。我家的年夜饭必备一道菜是丸子，圆滚滚的丸子，今年也不例外，往年都是买现成的。今年打算自己给家人试试新口味的丸子，自己动手做做。下面的实验品，试试做着，做出来还不错，用了烤箱，非油炸，更健康。有兴趣的厨友可以学做看看。表面焦脆，内软芝心香浓，一口一个。做的是2-3人份。

一、主料

主料 用量
车打芝士酱 （Ricos浓缩）60克
猪肉 （16个丸子份量）400克
鸡蛋 1个
面包屑 100克

二、辅料

辅料 用量
肉豆蔻 1 / 4茶匙
鼠尾草 3 / 4茶匙
干迷迭香碎 1 / 4茶匙
干芥末 3 / 4茶匙
香菜3 / 4茶匙 3 / 4茶匙
食用盐 3 / 4茶匙
辣椒粉 1 / 4茶匙
白胡椒 1 / 4茶匙
番茄酱 适量
烧烤酱 适量

三、做法

步骤
1.烤箱预热到190度。烤盘裹上铝箔纸，刷上食用油备用。Ricos芝士酱装在保鲜盒里，放入冰箱负18度1小时，取出切小粒备用。
2.大碗里加入面包屑，调味料，香草，猪肉碎和鸡蛋，然后充分搅拌均匀。
3.将肉搓成16个肉丸，每个丸子塞入Ricos芝士酱，然后捏紧封口，用手将球滚。
4.然后放在准备好的烤盘里。烤18-20分钟，直至熟透。
5.烤好的丸子可以直接吃，也可以蘸酱吃。

四、功效

。

  • 烤
  • 奶香味
