      萝卜丸子汤的做法

      小时候奶奶很爱吃萝卜丸子，那时候因为每天都吃萝卜，所以对它喜欢不起来，但是萝卜丸子却是小时候的最爱，Q弹有嚼劲，萝卜的清香弥漫在嘴里，这是一种幸福的味道！

      一、主料

      主料用量
      萝卜100克
      红萝卜50克
      瘦肉30克

      二、辅料

      辅料用量
      地瓜粉100克
      盐5克
      青菜20克
      鸡蛋1个

      三、做法

      步骤
      1.

      将白萝卜与红萝卜剁碎、瘦肉切丁

      |
      |
      2.

      加入少许盐腌5分钟，将多余的水分过滤掉，然后加入盐、鸡蛋一个抓匀，随后再加入地瓜粉继续搅拌

      |
      |
      3.

      拌好之后捏成丸子状备用

      |
      |
      4.

      锅内加适当的水，放入一点生姜提鲜

      |
      |
      5.

      待水开后加入捏好的丸子，大火煮10分钟

      |
      |
      6.

      然后加一点青菜提鲜，出锅前加入少许的盐即可

      |
