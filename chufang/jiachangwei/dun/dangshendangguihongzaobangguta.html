      党参当归红枣棒骨汤的做法

      炖汤对我们这个膳食食疗家族简直太简单了、因为我爸妈他们卖了大半辈子的食疗药材！今天我给大家推荐一款简单方便的女神棒骨汤！我个人认为女人最好的补品闺密之三就是当归、党参、红枣！

      一、主料

      主料用量
      党参8-10克
      当归8-10克

      二、辅料

      辅料用量
      红枣10克

      三、做法

      步骤
      1.

      准备好当归、党参、红枣、冲洗干净放在盘中。

      |
      |
      2.

      取出4-6块的棒骨

      |
      |
      3.

      放在锅中焯水

      |
      |
      4.

      把焯好了棒骨放入碗中

      |
      |
      5.

      接着把食材放在盛有棒骨的碗中

      |
      |
      6.

      加适量水到锅中、再把炖汤铁三角架放在锅中

      |
      |
      7.

      加三小碗半水到盛有棒骨汤碗中，然后把准备好的配料食材棒骨汤放在带有炖汤铁架的上面

      |
      |
      8.

      旺火到小火炖共40-60分钟

      |
      |
      9.

      开吃吧各位宝宝们！

      |

      四、功效

      降压，抵抗力，贫血，通便，心血管病，抗休克，强心，安神助眠，神经衰弱，血栓，补血。
