      素•罗宋汤的做法

      要口感～要视觉挑逗～要低脂健康～要实惠管饱(o^^o)那我们就开始吧！

      一、主料

      主料用量
      胡萝卜1根
      紫皮洋葱1个
      土豆1个
      圆白菜1个
      西红柿1个
      纯牛奶250毫升

      二、辅料

      辅料用量
      甜辣酱2勺
      黑胡椒粉适量
      欧芹碎适量
      盐适量
      无盐黄油适量

      三、做法

      步骤
      1.

      啦啦～食材明细在此～我们马上开始吧！

      |
      |
      2.

      为了口感更好～食材还是需要切成大小均匀的小块儿滴～(o^^o)

      |
      |
      3.

      锅热后再下入黄油～黄油慢慢融化的过程～飘出阵阵香味～特别幸福！(^_^)v

      |
      |
      4.

      下料顺序：黄油完全融化后，先下入洋葱块，爆香后，再放入胡萝卜块，翻炒均匀后，再放入西红柿块，翻炒成西红柿酱

      |
      |
      5.

      以后下入圆白菜继续翻炒

      |
      |
      6.

      最后下入土豆块，放入清水末过菜地，大火炖开

      |
      |
      7.

      大火炖开后，放入甜辣酱，菜量大放2勺，喜欢辣口可以再多放一些，不吃辣，可以放番茄酱哈(o^^o)之后再撒些盐，调味～

      |
      |
      8.

      再次大火炖开后，放入牛奶，搅拌均匀，调小火炖5分钟后出锅，盛盘后撒些黑胡椒粉和欧芹叶碎～味道更鲜美！喜欢奶味浓的可以再多放一些(o^^o)

      |

      四、功效

      抵抗力，降糖，保护视力，助消化，美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，防癌，消食，高血压，祛斑，延年益寿，血栓，镇静助眠，减肥瘦身。
