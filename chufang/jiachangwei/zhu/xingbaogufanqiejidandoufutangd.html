      杏鲍菇番茄鸡蛋豆腐汤的做法

      好久没做菜了。 这道菜适合一个人吃饭的朋友。 一个人自己做饭吃总是很麻烦，做多了吃好几天，做少了又犹豫天天这样忙活半天到底有没有必要。 其实可以选既可以吃到菜，又可以喝到汤的类型。 煮上一大锅菜。既可以有清单的清汤，又可以做麻麻辣辣的红汤。

      一、主料

      主料用量
      杏鲍菇1个
      番茄1个
      鸡蛋1-2个
      内酯豆腐1个

      二、辅料

      辅料用量
      姜3片
      盐1勺
      鸡精1勺
      猪油一筷子头

      三、做法

      步骤
      1.

      准备材料。
      豆腐建议用内酯豆腐，用来煮汤合适，口感好，特别是含在嘴里一下吸掉的感觉。
      杏鲍菇可以尝试换成其他菌类。

      |
      |
      2.

      番茄切片

      |
      |
      3.

      蘑菇切丝

      |
      |
      4.

      放锅中煮。
      加盐，加姜片。

      |
      |
      5.

      哦，还加点鸡精。

      |
      |
      6.

      煮开之后打个蛋进去。
      我直接整个蛋打进去的，这样搅了之后有白的有黄的好看。
      也可以先搅匀之后倒进去。

      |
      |
      7.

      蛋掉进去了。
      (其实这是上次做的，我先加了豆腐，这样是不好的，因为搅合蛋的时候会把豆腐搅烂。)

      |
      |
      8.

      立刻搅一搅掉进去的蛋。
      火啊，一直是大火烧着啊！烧得滚滚的，蛋白蛋黄冲出来之后就不搅了。

      |
      |
      9.

      这个豆腐划道道，看上去一小块一小块的。
      其实是一条一条的。

      |
      |
      10.

      咵，倒进锅里。
      注意就让他这样煮了，不要去搅了，这个豆腐太嫩，会烂的。
      当然，你喜欢看和喜欢喝支离破碎的豆腐，可以使劲搅。

      |
      |
      11.

      烧开后就开中火等着吧。
      做你的其他菜去。

      |
      |
      12.

      当你其他菜做完之后。
      就关火准备汤装盆了。

      |
      |
      13.

      这是今天做的，先放的蛋，后放的豆腐。
      不过还是煮烂了一点。
      可能豆腐要再切粗点吧。

      |
      |
      14.

      装，装盆吧。

      |
      |
      15.

      不对呀，忘了放点猪油。
      完工。

      |
      |
      16.

      我一个人可以吃3顿。
      说多都是泪。
      这个用来泡饭吃，别提多爽了。
