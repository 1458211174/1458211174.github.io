      红枣花生二米粥的做法

      一碗好粥，总能带给人温暖的感觉，居家生活久了，做的饭菜多了，粥，却始终是心头爱。对于粥里的食材，总是随着自己的心意和想要得到的营养功效而添加。 小时候煮粥，都是在炉火上用锅慢熬，有一个人必须守着，大火烧开，小火慢煮，边煮还要边搅拌，以免糊了锅底。现在方便多了，因为有电饭煲，把所有的食材放进去，通电，启动程序，就可以做别的事情去了，时间到自有一锅好粥等着自己。 妈妈寄来了她自己种的花生，剥好的一粒一粒的放在布袋里，颗颗饱满。花生中钙含量比较高，对孩子来说可以促进儿童骨骼发育，还能改善血液循环，增强记忆，花生中的锌元素还能够促进儿童大脑发育，延缓人体衰老，另外花生还有一个重要的作用，它能滋血通乳，对于还在哺乳期的我来说，很是合适。 既然是粥，就不能缺米，大米和小米都是家里常备的食材，再配上些红枣，有一些枣的香甜味，这样味道更好一些，不仅大人喜欢喝，孩子也更喜欢喝。

      一、主料

      主料用量
      大米45克
      小米35克
      红枣50克
      花生50克

      二、辅料

      辅料用量
      清水适量

      三、做法

      步骤
      1.

      准备好大米、小米、红枣、花生；

      |
      |
      2.

      花生洗净后，用刀切碎；红枣洗净后去核掰碎；

      |
      |
      3.

      大米小米洗净，连同处理好的花生和红枣一起放进电饭煲内；

      |
      |
      4.

      加入适量的清水；

      |
      |
      5.

      电饭煲通电，启动“煮粥”程序，时间可以调整，我一般选1小时30分钟，如果想要更黏稠的口感，可以选2小时以上；

      |
      |
      6.

      90分钟后，待提示音响起，一锅飘着枣香的热粥就可以准备食用了；

      |
      |
      7.

      米和花生都已经软烂，枣味已经融入枣里，香甜可口，孩子们也都爱喝。

      |
      |
      8.

      这款粥补气养血，营养丰富，可以说是老少皆宜。

      |
      |
      9.

      不管是早餐还是晚餐，来这么一晚营养好味道好的粥，总是很能安抚自己的胃！

      |

      四、功效

      适合婴儿，强身健体，开胃消食，抗感染，预防流产，维持性功能，祛斑美容，促进生长，助胎儿发育，治流鼻血，抗衰老，润肺，止血，心血管病，补虚，保护心脏，促进发育，益智。
