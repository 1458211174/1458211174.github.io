      野淮山玉米猪蹄汤的做法

      淮山不仅营养丰富且具有很多药用价值，经常食用，可促进抗体形成，增强免疫力，还能滋养肝肾，补虚健脾，治疗身体虚弱，疲倦无力等症，是大众化的滋补食品。 猪蹄含有丰富的胶原蛋白质，脂肪含量也比肥肉低，能防治皮肤干瘪起皱、增强皮肤弹性和韧性，是老幼皆宜的健康食品，有美容抗衰老的作用。

      一、主料

      主料用量
      猪前脚1个
      野淮山1根
      甜玉米两根

      二、辅料

      辅料用量
      茶籽油15克
      盐1.5克

      三、做法

      步骤
      1.

      猪脚买的时候烧干净猪毛，剁好，拿回来用菜刀全部刮一遍，再冲洗干净，

      |
      |
      2.

      焯水

      |
      |
      3.

      捞起冲洗一遍，再刮干净，特别是猪蹄上的那层白色的皮，焯水后很容易刮下来

      |
      |
      4.

      切成小块，冲洗干净

      |
      |
      5.

      猪脚放入电饭锅里，倒入足够的清水，选择煮汤键，煮一个小时

      |
      |
      6.

      野淮山放盐水里浸泡半小时，再去皮，洗净切滚刀块

      |
      |
      7.

      甜玉米洗净切段

      |
      |
      8.

      一个小时后放淮山、玉米下锅再煮一个小时

      |
      |
      9.

      倒点茶籽油

      |
      |
      10.

      撒点盐调味，即可出锅
