      猪骨煲苓薯茅根汤的做法

      苓薯这种食材清热下火，本地人都喜欢买来去火煲汤喝，实际本身没有什么味道，我觉得只是拿它跟猪骨煲的话味道很淡，我老公也不太爱喝的，所以我又加入了一把茅根，茅根有丝丝的甜味，这汤的味道就好很多了，而且茅根也是利尿，清火的，对于广东人爱上火，煲这汤喝很合适的。

      一、主料

      主料用量
      猪骨200克
      苓薯100克
      白茅根50克

      二、辅料

      辅料用量
      油1汤匙
      盐1汤匙

      三、做法

      步骤
      1.

      猪骨放入水中焯烫一下。

      |
      |
      2.

      茅根适量清洗干净。

      |
      |
      3.

      苓薯洗净，剥掉外皮后对半切块。

      |
      |
      4.

      焯烫好的猪骨倒入砂锅内，加入适量水。

      |
      |
      5.

      把茅根倒入砂锅中。

      |
      |
      6.

      把苓薯倒入砂锅中。

      |
      |
      7.

      盖上锅盖大火煮开后改小火煲1个小时。

      |
      |
      8.

      加盐调味即可关火。

      |

      四、功效

      骨质疏松，去火，利尿，治流鼻血。
