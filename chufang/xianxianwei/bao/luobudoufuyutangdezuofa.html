      萝卜豆腐鱼汤的做法

      冬日里来一道清新暖胃的鱼汤，能量满满的又健康！

      一、主料

      主料用量
      鱼八条
      白萝卜一根
      嫩豆腐两块

      二、辅料

      辅料用量
      枸杞子十五颗
      皮蛋一颗
      葱一根
      生姜半块
      盐适量

      三、做法

      步骤
      1.

      鱼处理干净后入锅连生姜小火煎 入锅前用猪油润锅

      |
      |
      2.

      煎至两面熟后开大火加黄酒热水葱白煮五分钟

      |
      |
      3.

      转中火下入切好的白萝卜豆腐煮十分钟

      |
      |
      4.

      萝卜熟后转大火加入切好的皮蛋粒 撒入枸杞葱花白胡椒盐三分钟关火 鲜香无比
