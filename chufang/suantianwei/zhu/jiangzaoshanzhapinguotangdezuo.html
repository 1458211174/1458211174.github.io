      姜枣山楂苹果汤的做法

      偶然看到一道大枣山楂苹果汤，忽然一亮。做蛋糕剩下了3/4个苹果，一堆山楂待用，这汤遇得真是时候。 转念之间，新的主意又蹦出来。曾经的那个姜枣苹果汤正好可以与这大枣山楂苹果汤结合起来。温中散寒，且又活血化瘀。正适合这阴雨的天气。 没加太多红糖，苹果大枣都够甜，山楂好像也并不那么酸。。。。

      一、主料

      主料用量
      苹果3/4个
      山楂6颗
      大枣2颗
      生姜10克
      枸杞10克

      二、辅料

      辅料用量
      红糖1勺
      水800毫升

      三、做法

      步骤
      1.

      用料：苹果3/4个，大枣2颗，山楂6颗，生姜10克，枸杞10克，红糖1勺，水800毫升

      |
      |
      2.

      山楂洗净去果核蒂把。

      |
      |
      3.

      生姜洗净切片。

      |
      |
      4.

      大枣洗净去核，

      |
      |
      5.

      苹果洗净切片。

      |
      |
      6.

      将水和山楂大枣放入锅中，煮沸。

      |
      |
      7.

      加入苹果，煮10分钟。

      |
      |
      8.

      再加入姜片，枸杞，红糖，煮15分钟。

      |
      |
      9.

      熄火，趁热食用。

      |

      四、功效

      脂肪肝，防癌，中风，润肠通便，抵抗力，减肥瘦身，降血脂，开胃，止疼，腹泻，抗衰老，产后补虚，养肝护肝，补钙，软化血管，预防结石，降压，降胆固醇，补血，预防感冒，散瘀止疼，强身健体，治。
