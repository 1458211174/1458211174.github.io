      流水素面和酱油煎猪排

      和吉儿在一起每天都会有新意，她口中常常提起的流水素面就是我们今天的午餐。

      传说王羲之流觞曲水便是这流水素面的由来。日本人见诗人们将酒觞(杯)置于清流之上，任其飘流，停在谁的前面，谁就即兴赋诗，而出发奇想把竹子劈成半月形，接成水渠行，热爱美食的人们拿着筷子分别站在竹子的两旁，水中有面条顺水而下，想吃的人把筷子一放，面条就上来了，面放到蘸料里面稍微蘸一蘸，一口气吸进肚里，清凉入骨。非常适合夏季食用。

      后来一个叫井上的人改良了，如今吃流水素面，就是一个圆形的透明的水流机，放入面条，好玩又好吃。

      据说在鹿儿岛，吃流水素面是一道夏季风景，再配上烤鱼，今天我们在家做了一碗没有流水的流水素面，配上我们家常的酱油煎猪排。

      一、材料

      【食材】细面 // 酱油、糖、味霖// 柴鱼 // 黄瓜 // 柠檬 //生姜 // 里脊肉 // 卷心菜 //

      二、做法

      步骤
      1、先腌制猪排，里脊肉切两大片，半厘米厚，撒些生抽和糖，这次我使用的是蒸鱼豉油，腌制1-2小时；
      2、卷心菜切成细丝，越细越好，浸泡在清水里清洗干净再滤干水分；
      3、做素面的蘸汁，水烧开，大约200毫升，用一个漏勺盛着柴鱼在沸水里小火慢熬柴鱼；
      4、等柴鱼熬干扁了，倒掉，加入生抽（我还是用蒸鱼豉油）、糖和味霖煮开，盛在一个杯子里
      5、因为流水素面是冷着吃更好，所以这个时间我们可以煎猪扒，等酱汁稍微凉的时候，可以擦些姜泥和黄瓜泥，最后再挤些柠檬，这样蘸汁就做好了。
      6、现在说说煎猪排。锅烧热下少许油，拿厨房纸巾把腌制的猪排擦干，下煎锅，小火慢煎；
      7、其实猪排很快就会熟，猪排两面变色，筷子可以擦透即可；
      8、将猪排切成小块，在盘子上撒上卷心菜丝，把切好的猪排放在卷心菜丝上，最后撒上worcestershire sauce辣酱油；
      9、最后烧开水煮素面，素面大约10分钟就好，然后用冷水冲凉，最好是矿泉水或是冷开水；
      10、冷却的素面放在碗里，夹一筷子面，蘸着酱汁，一口吸进，可以试试，味道真的很不错
      细面 酱油 味霖 柴鱼 黄瓜 柠檬 生姜 里脊肉 卷心菜
