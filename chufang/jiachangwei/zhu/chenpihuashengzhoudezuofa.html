      陈皮花生粥的做法

      用陈皮搭配花生米，再添加些糯米、大米煮了粥，口感味道都很不错。春季很容易出现消化不良的情况，这个时候喝点陈皮粥，健胃消食的效果很好，对消化不良很有疗效，还有着润肺止咳等功效。

      一、主料

      主料用量
      陈皮10克
      花生米100克
      糯米50克
      大米50克

      二、辅料

      辅料用量
      枸杞10克

      三、做法

      步骤
      1.

      花生米洗干净用冷水浸泡30分钟

      |
      |
      2.

      陈皮洗干净用冷水浸泡30分钟

      |
      |
      3.

      糯米、大米洗干净用冷水浸泡30分钟

      |
      |
      4.

      煮锅加水浸泡后的花生米入锅煮

      |
      |
      5.

      开锅后，陈皮入锅煮

      |
      |
      6.

      糯米、大米入锅煮。其间适时搅拌，以免糊锅。

      |
      |
      7.

      煮至食材熟，枸杞入锅关火。盖上锅盖焖5、6分钟后即可享用

      |

      四、功效

      增强体质，健脾胃，缓解尿频，适合婴儿，强身健体。
