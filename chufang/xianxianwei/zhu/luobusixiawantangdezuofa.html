      萝卜丝虾丸汤的做法

      白萝卜是特别适合秋冬季常吃的好食材，其具有具有清热生津、下气宽中、消食化滞、开胃健脾、顺气化痰的功效。 简简单单的一道汤，味道却特别鲜美，近似透明的汤色，萝卜丝也晶莹剔透，淡粉色的鲜美虾丸，是尤为养生又健康的一款汤品。

      一、主料

      主料用量
      虾200克
      白萝卜50克
      鸡蛋清1个
      姜2片
      葱丝适量

      二、辅料

      辅料用量
      盐1小勺
      料酒1勺
      淀粉适量
      白胡椒粉少许

      三、做法

      步骤
      1.

      虾仁或去皮鲜虾用刀背剁成虾蓉，不要剁的太碎。

      |
      |
      2.

      虾蓉加入适量的蛋清、料酒和盐，上劲搅拌，并逐渐加入适量淀粉，筷子要沿着一个方向搅拌，使得虾蓉更有弹性。

      |
      |
      3.

      取适量的虾蓉，用勺子挖成一勺，滑入沸水，汆烫到虾丸变色就捞出。

      |
      |
      4.

      萝卜去皮切细丝。

      |
      |
      5.

      另起锅加水烧开，放入姜片、萝卜丝，煮到沸腾且萝卜丝变得透明后加入虾丸，加入少许盐，煮2分钟即可。

      |
      |
      6.

      起锅前加入少许白胡椒粉，搅拌后去掉姜片，捞出即可。

      |

      四、功效

      下奶，安神助眠，动脉硬化，抵抗力，助睡眠，消食，保护肠胃，预防疾病，强身健体，清热解毒，美容养颜，延年益寿，预防感冒，散瘀止疼，治肠炎，开胃，预防中暑，缓解腹痛，治吐泻。
