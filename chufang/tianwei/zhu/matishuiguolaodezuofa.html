      马蹄水果捞的做法

      微信公众号“宝宝营养辅食”，收录了我给宝宝做过的500道营养辅食，有了这500道辅食菜谱，不再为宝宝吃啥而发愁。小思妈妈会在这个公众号分享6-24个月的宝宝辅食。 自从上次发了马蹄蒸肉饼的菜谱，留言区就被大家攻陷了，很多麻麻们纷纷贡献出了各种马蹄的制作方法，有用糖炒的，有做饺子馄饨的，还有做成马蹄糕，或者煲汤清炒的，我拿着小本本一样一样记了下来，非常感谢大家的分享呦~ 其中有一位麻麻告诉我可以将马蹄和雪梨一起熬煮，不用加糖，煮出的汤汁味道清甜，宝宝非常喜欢喝。于是我就“马不停蹄”地尝试了一下，做好后真的超级赞！马蹄和雪梨煮好后，咬在嘴里咔呲咔呲地非常清脆，再配上微甜的汤汁，让宝宝的小胃暖暖的。

      一、主料

      主料用量
      马蹄70克
      雪梨180g
      哈密瓜少许

      二、辅料

      辅料用量
      蔓越莓少许

      三、做法

      步骤
      1.

      准备食材：马蹄70g、雪梨180g、哈密瓜少许、蔓越莓少许。

      PS： 蔓越莓干味道酸酸的，非常开胃，所以我会经常用它来点缀一些小甜品或者夹在吐司中给宝宝吃。如果没有的话也可以不加，或者用葡萄干代替。

      |
      |
      2.

      将雪梨去皮。

      |
      |
      3.

      然后将雪梨切成小块。

      PS： 雪梨和马蹄的大小都可以根据宝宝的咀嚼能力调整。

      |
      |
      4.

      再将马蹄切成小块。

      PS： 我买的是已经切成片的冷冻的马蹄，如果麻麻可以买到新鲜的，直接去皮切成小块就可以了。

      |
      |
      5.

      哈密瓜去皮去瓤，切成小块。

      PS： 哈密瓜中β-胡萝卜素含量高达920微克/100克，堪称水果之最。

      |
      |
      6.

      将雪梨和马蹄倒入锅中。

      PS： 这道马蹄水果捞也非常适合全家人一起享用，所以我做的比较多，如果只做宝宝份的可以适当减少食材量。

      |
      |
      7.

      添加适量清水。

      PS： 我大概添加了700ml的清水，这个比例刚刚好，可以供大家参考一下哈~

      |
      |
      8.

      先大火煮沸，然后小火慢炖。

      |
      |
      9.

      期间用勺子撇去浮沫。

      |
      |
      10.

      大概煮30分钟左右就可以出锅啦。

      |
      |
      11.

      煮好后，将马蹄雪梨和汤汁倒入碗中，加一勺哈密瓜，最后撒上一点蔓越莓干。

      |
      |
      12.

      给小一点的宝宝食用，可以将煮好的马蹄和雪梨倒入搅拌机中搅拌成汁。

      PS： 添加的水量可以根据宝宝的喜好随意掌握，稀一点稠一点都是可以哒！

      |
      |
      13.

      一碗甜甜的马蹄水果捞就做好啦，味道甜甜的非常好喝，嚼起来脆脆的还能锻炼宝宝的咀嚼能力呦！

      |

      四、功效

      促发育，降压，通便排毒，清热解毒，去火，神经衰弱。
