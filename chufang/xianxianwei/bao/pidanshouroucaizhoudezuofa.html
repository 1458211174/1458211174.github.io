      皮蛋瘦肉菜粥的做法

      过年回家吃的太油腻。今天下午回来实在不知道吃什么，白粥又太单调，所以做了锅皮蛋瘦肉粥。

      一、主料

      主料用量
      大米150克
      瘦肉150克
      皮蛋2只
      青菜4棵

      二、辅料

      辅料用量
      生姜一小块
      葱1把
      精盐3克

      三、做法

      步骤
      1.

      淘米

      |
      |
      2.

      将淘好的米放一点水泡半小时

      |
      |
      3.

      170g纯精肉

      |
      |
      4.

      将精肉切成小丁

      |
      |
      5.

      开水里焯一下

      |
      |
      6.

      焯到沸腾，去血水去腥味

      |
      |
      7.

      将焯好的肉丁放入碗中

      |
      |
      8.

      锅中放油并爆炒葱姜，炒出香味后放入焯好的肉丁再翻炒一两分钟。随后放入适量开水煮十几分钟。

      |
      |
      9.

      肉汤煮好后将葱姜捞出，连汤带肉一起放入泡好的米中。开始煮粥。

      |
      |
      10.

      乘煮粥的时间煮皮蛋二十分钟

      |
      |
      11.

      煮好后切成丁

      |
      |
      12.

      青菜三四棵洗净

      |
      |
      13.

      菜叶切碎，菜梗去掉

      |
      |
      14.

      粥煮了一个小时后放入皮蛋煮十几分钟

      |
      |
      15.

      最后将电饭煲开大功率将菜叶放入锅中再放入3g左右盐，用勺子来回翻一下粥煮两三分钟即可食用。放入菜叶后锅盖打开不要闷着，不然菜叶会变黄。

      |
      |
      16.

      做好啦

      |
      |
      17.

      开吃，这么多天胡吃海塞后来一顿皮蛋瘦肉菜粥很是清口。

      |

      四、功效

      适合婴儿，强身健体，补肾，润肺，软化血管，防癌，润肺化痰，抵抗力，美容养颜，延年益寿。
