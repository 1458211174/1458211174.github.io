      山药薏米银耳羹的做法

      银耳味甘、淡、性平、无毒，既有补脾开胃的功效，又有益气清肠、滋阴润肺、美容嫩肤的作用，银耳含有蛋白质、脂肪和多种氨基酸、矿物质及肝糖。富有天然植物性胶质，对皮肤有良好保护作用，被称为“穷人的燕窝”。 大家都知道银耳补充胶原蛋白、滋润肌，薏米祛湿美白、细嫩肌肤、抗老化，多数以食疗来美容瘦身的提倡者，她们的食谱中都少不了它们。 山药健脾益胃、助消化，益肺止咳，延年益寿，辅助治疗妇女白带多、小便频数等症。 寒冷而又干燥的冬季，每天来一碗养生银耳羹再好不过了，滋润了肌肤，也温暖了心~~

      一、主料

      主料用量
      银耳8克

      二、辅料

      辅料用量
      铁棍山药60克
      薏米30克
      冰糖20克

      三、做法

      步骤
      1.

      准备材料，银耳提前2个小时泡开

      |
      |
      2.

      银耳洗去杂质，去根，撕成小片放入压力锅内，放入洗净的薏米

      |
      |
      3.

      加适量水

      |
      |
      4.

      选择“粥”程序煮15分钟

      |
      |
      5.

      山药去皮切块

      |
      |
      6.

      山药放入锅内

      |
      |
      7.

      放入冰糖，再煮十分钟

      |
      |
      8.

      山药变得粉糯即可

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体。
