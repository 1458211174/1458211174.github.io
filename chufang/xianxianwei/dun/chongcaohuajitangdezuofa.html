      虫草花鸡汤的做法

      虫草花不是虫草其实是真菌类的一种，最近咳嗽感冒，鸡汤可以增强免疫力，虫草花可以止咳，一顿吃不完，还可以买点面条做鸡汤面

      一、主料

      主料用量
      小母鸡700克
      虫草花15克
      黑木耳15克

      二、辅料

      辅料用量
      姜2片
      盐3克
      味精3克
      料酒2毫升

      三、做法

      步骤
      1.

      虫草花，网上买的，买了一大包，其实只要15克就好了，如果家里没称的，就手抓一小把就可以了

      |
      |
      2.

      黑木耳，15克左右，依个人喜好，喜欢多吃的可以多放一点

      |
      |
      3.

      虫草花和黑木耳用自来水浸泡半小时，黑木耳长大了，泡软了就可以了，烧之前用干净的水再冲洗一下

      |
      |
      4.

      鸡我选择小母鸡炖汤，鸡肉比较嫩。放一片生姜和适量黄酒，初学者黄酒的量不好把握的话就倒在汤勺里，一汤勺黄酒就够了，冷水下锅，水开后把鸡撩出来冲洗干净，另放一锅。

      |
      |
      5.

      把所有食材倒入锅中，加盐，煤气上炖1小时，压力锅炖半小时

      |
      |
      6.

      出锅时加适量鸡精调味，其实已经很鲜美了，依个人口味，如果淡了再加点盐。我喜欢清淡的，这样已经很美味了，我一个人喝了三碗汤

      |

      四、功效

      强身健体，排毒，减肥瘦身。
