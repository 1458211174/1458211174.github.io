      蒜苗豆腐羹的做法

      豆腐，作为家常必备食材，深得广大人民群众的喜欢，已有两千多年的历史。豆腐的营养价值不必多说，想必大家也众所周知。豆腐的做法更是种类繁多，例如，麻婆豆腐，红烧豆腐，白菜炖豆腐等等。今天推荐给大家一款更适合老人，妇女，儿童的做法，营养丰富，口感爽滑，味道咸香。

      一、主料

      主料用量
      内酯豆腐1盒
      蒜苗1把
      胡萝卜半根
      香菇2朵
      鸡蛋1个

      二、辅料

      辅料用量
      鸡精少许
      盐少许

      三、做法

      步骤
      1.

      豆腐切小块，蒜苗，胡萝卜，香菇切小丁，鸡蛋打散

      |
      |
      2.

      锅中放油，油热放入蒜苗，胡萝卜，香菇，翻炒

      |
      |
      3.

      青菜炒香加入水

      |
      |
      4.

      水开放入豆腐

      |
      |
      5.

      水再次开放入鸡蛋液，加入鸡精，盐调味即可

      |

      四、功效

      抵抗力，清热解暑，降血脂，防癌，降糖，保护视力，助消化，延年益寿，强身健体，降压，降胆固醇，健脑益智，保肝，动脉硬化，美容护肤。
