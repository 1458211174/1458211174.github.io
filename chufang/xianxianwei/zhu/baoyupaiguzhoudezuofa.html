      鲍鱼排骨粥的做法

      之前在揭阳，姐夫带我们去吃了砂锅版的鲍鱼排骨粥，那粥吃了真的是不会忘。潮汕地区的粥品，一般是用砂锅来煮，砂锅以文火来煮，这样煮出的潮汕粥米香四溢。那真的是是极费工时的哦。现在我也找到了一个好的方法，就是用电饭煲来煮制，用水量比平时煮粥多那么五分之一，这样子的一个配水方法，那煮出来的粥也是米香四溢，鲜香满屋的哦。

      一、主料

      主料用量
      鲍鱼干8只
      排骨350克
      大米1杯

      二、辅料

      辅料用量
      冬菜适量
      芹菜适量
      细盐适量

      三、做法

      步骤
      1.

      备好排骨，虾干，鲍鱼干。

      |
      |
      2.

      再备多半杯的大米。

      |
      |
      3.

      将所的食材混合在一起，泡制一会。

      |
      |
      4.

      再将所用食材，洗好控水备用。

      |
      |
      5.

      将食材入电饭煲，加入的清水比砂锅煮锅更多的清水。

      |
      |
      6.

      选择煮粥模式，用时差不多一个半小时，如图我是预约的时间。

      |
      |
      7.

      .如图就是煮好的鲍鱼排骨粥。

      |
      |
      8.

      再加入适量细盐及冬菜提味。

      |
      |
      9.

      最后再加入芹菜末点缀提鲜即可。

      |
      |
      10.

      这样子一份鲍鱼排骨粥就完成了。

      |

      四、功效

      补肾，润肺，适合婴儿，强身健体。
