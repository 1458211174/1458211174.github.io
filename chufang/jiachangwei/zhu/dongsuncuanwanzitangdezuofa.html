      冬笋汆丸子汤的做法

      现是吃冬笋的好时节，虽说已是春天了，还会时不时的有阴雨微寒之感，给家人做一份热乎浓香的冬笋汆丸子汤，爽脆的冬笋，满口含香的肉丸，小抿一口食物在碰撞中的浓汤，感受味蕾带来的美好体验。

      一、主料

      主料用量
      冬笋1个
      猪肉馅200克

      二、辅料

      辅料用量
      黑木耳适量
      鸡蛋1个
      玉米淀粉少许
      大蒜苗2根
      生姜一小块
      大葱适量
      酱油1勺
      白胡椒粉少许
      食用油少许
      香油2滴
      盐少许

      三、做法

      步骤
      1.

      冬笋去皮

      |
      |
      2.

      大葱、生姜、鸡蛋、胡椒粉、食用油、酱油、盐、玉米淀粉放入肉馅中，一个方向搅拌上劲，冬笋切片

      |
      |
      3.

      冬笋在锅里焯一下水 捞出

      |
      |
      4.

      锅洗干净加入清水煮开，加入焯过水的冬笋，木耳烧开转小火，用小勺挖出拌好的肉糜，一勺一勺的放入锅中

      |
      |
      5.

      中火慢慢煮开，继续小火2分钟，洒上大蒜苗碎，胡椒粉，盐，2滴香油即可出锅

      |

      四、功效

      止血，排毒，养肝护肝，助消化，健脾，清热解毒。
