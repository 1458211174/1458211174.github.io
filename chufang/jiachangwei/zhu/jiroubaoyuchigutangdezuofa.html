      鸡肉煲鱼翅骨汤的做法

      鱼翅骨是剥过鱼翅的骨头，不过营养价值还是很高的，用它煮汤非常的鲜美，看到汤都已经浓稠变乳白了，就知道非常的好喝了，加入适量鸡肉去煲，又甘甜又营养，宝宝大人喝都非常的好哦。

      一、主料

      主料用量
      鱼翅骨50克
      鸡肉200克

      二、辅料

      辅料用量
      油1汤匙
      盐1汤匙
      北芪30克
      姜2片

      三、做法

      步骤
      1.

      鱼翅骨用清水浸泡一个晚上。

      |
      |
      2.

      把鸡肉在锅中先焯烫一下。

      |
      |
      3.

      锅中加入适量水，加适量切丝的瘦肉。

      |
      |
      4.

      加入几片姜。

      |
      |
      5.

      加入几片北芪。

      |
      |
      6.

      把焯烫好的鸡肉倒入锅中。

      |
      |
      7.

      鱼翅骨沥干水分倒入锅中。

      |
      |
      8.

      盖上锅盖大火煮开改小火慢炖两个小时以上。

      |
      |
      9.

      喝之前加盐调味即可。

      |

      四、功效

      抵抗力，养肾，消食。
