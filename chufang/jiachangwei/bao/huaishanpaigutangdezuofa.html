      淮山排骨汤的做法

      淮山益肾气，健脾胃，止泄痢，化痰涎，润肤养发。加入胡萝卜、排骨一起煲汤，味道更鲜美。这个春天，你需要这样一碗汤。

      一、主料

      主料用量
      淮山200克
      排骨150克
      姜适量
      胡萝卜150克

      二、辅料

      辅料用量
      盐适量

      三、做法

      步骤
      1.

      准备好食材：胡萝卜，淮山，排骨，姜。

      |
      |
      2.

      胡萝卜去皮切滚刀块。

      |
      |
      3.

      姜去皮切片。

      |
      |
      4.

      淮山去皮，切段。

      |
      |
      5.

      锅里加适量的水，加入姜片，倒入洗干净后的排骨，大火煮开，焯去血水。

      |
      |
      6.

      水开后继续煮一分钟，关火，把排骨捞出，用清水冲洗掉血沫。

      |
      |
      7.

      把淮山、胡萝卜、排骨、姜片放入砂锅中。

      |
      |
      8.

      倒入适量的清水，盖过食材。

      |
      |
      9.

      大火煲开，改小火继续煲40分钟左右。

      |
      |
      10.

      食用前加入适量的盐调味。

      |

      四、功效

      补肾，润肺，预防感冒，散瘀止疼，强身健体，治肠炎，开胃，预防中暑，缓解腹痛，治吐泻，抵抗力，降糖，保护视力，助消化。
