      苹果燕麦粥的做法

      苹果富含多种维生素，含有的果胶对于身体排毒很有好处，还有很好的延缓衰老的作用。燕麦中富含可溶性膳食纤维，对预防动脉粥样硬化、高血压、冠心病等病的效果都很好。两种利好又美味的食材搭配，再添加同样利好又美味的桂圆、糯米煮的粥，非常好喝，也非常养生保健。

      一、主料

      主料用量
      苹果200克
      燕麦80克
      糯米60克

      二、辅料

      辅料用量
      桂圆干30克

      三、做法

      步骤
      1.

      备好燕麦、糯米

      |
      |
      2.

      桂圆洗干净用冷水浸泡20分钟

      |
      |
      3.

      燕麦、 糯米洗干净用冷水浸泡30分钟

      |
      |
      4.

      煮锅加水，燕麦入锅煮

      |
      |
      5.

      开锅后，糯米入锅同煮。其间适时搅动

      |
      |
      6.

      20分钟后，桂圆入锅同煮。其间还要适时搅动

      |
      |
      7.

      苹果洗干净切成块儿

      |
      |
      8.

      苹果入锅同煮。其间还要适时搅动，煮至食材熟，粥粘稠即可关火。焖5、6分钟后即可盛出享用

      |

      四、功效

      脂肪肝，防癌，中风，润肠通便，抵抗力，减肥瘦身，糖尿病，通便，补钙，延年益寿，美白，预防疾病，增强体质，健脾胃，缓解尿频。
