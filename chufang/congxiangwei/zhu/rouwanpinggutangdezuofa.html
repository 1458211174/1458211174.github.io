      肉丸平菇汤的做法

      自制肉丸，加上平菇的清甜，这汤真是香甜无比。

      一、主料

      主料用量
      肉丸200克
      平菇350克

      二、辅料

      辅料用量
      食盐4克

      三、做法

      步骤
      1.

      准备500ml开水，把锅洗干净。

      |
      |
      2.

      把平菇洗干净，一朵一朵分开，大的可以对半撕开。

      |
      |
      3.

      把自制的猪肉丸放入锅里，稍微加热一下。（猪肉丸是自己用新鲜猪肉做的，口感比外面的好吃太多啦！而且我还把已经做肉丸食谱上传了。有兴趣的朋友可以看我发布的食谱）

      |
      |
      4.

      倒入适量开水，待开水继续沸腾时，放入洗干净的平菇。

      |
      |
      5.

      等汤再次烧开，加适量食盐调味即可。。

      |

      四、功效

      抵抗力，散瘀止疼，延年益寿，益智。
