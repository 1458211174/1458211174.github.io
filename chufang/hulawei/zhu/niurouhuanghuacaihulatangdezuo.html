      牛肉黄花菜胡辣汤的做法

      黄花菜营养很丰富，含有丰富的钙、蛋白质、糖、脂肪、氨基酸，胡萝卜素，维生素C等人体必须的养分。黄花菜含磷丰富，具有较佳的健脑抗衰功能。黄花菜可以和木耳等一起吃。或者和鸡蛋，肉，鸡等做汤吃或炒食，营养都很丰富。不过，黄花菜使用前应先用温水泡20分钟，在用冷水泡10分钟。务必要炒熟或者煮熟吃，保证它所含的有毒物质秋水仙碱被破坏掉。

      一、主料

      主料用量
      牛肉150克
      黄花菜50克

      二、辅料

      辅料用量
      五花肉50克
      盐适量
      白糖少量
      淀粉适量
      食用油适量
      香菜1颗
      菠菜100克
      西红柿1个
      干木耳3-5个
      指天椒1个
      葱1根
      胡椒适量
      料酒适量
      鸡蛋2个

      三、做法

      步骤
      1.

      牛肉洗干净切小粒，五花肉洗干净切小片，分别装在碗里备用。

      |
      |
      2.

      西红柿洗干净去皮切小块，干木耳泡发后撕成小片，泡好去梗的黄花菜切成两段，菠菜洗干净切成两段，指天椒洗干净切小块，葱洗干净，葱白葱叶分开切断。把这所有材料分别装在碗里备用。

      |
      |
      3.

      胡椒放入舂子，舂碎。

      |
      |
      4.

      五花肉放入锅里，炒至五花肉的油出来，颜色呈金黄色就可以了。

      |
      |
      5.

      葱白和辣椒放入锅里翻炒出香味，烹入料酒，拌匀。

      |
      |
      6.

      锅里加水，烧开。

      |
      |
      7.

      牛肉、黄花菜和西红柿丁放入锅里，烧开。

      |
      |
      8.

      鸡蛋磕入碗，打散，备用。

      |
      |
      9.

      把舂碎的胡椒倒入锅里，加入少许白糖，加入盐，搅拌均匀。

      |
      |
      10.

      木耳放入锅里，翻炒均匀。

      |
      |
      11.

      把调好的水淀粉倒入锅里勾芡，拌匀。

      |
      |
      12.

      水开后，鸡蛋液均匀地倒入锅里。

      |
      |
      13.

      菠菜段、香菜段和葱叶段一起放入碗里，大火烧开即可。

      |
      |
