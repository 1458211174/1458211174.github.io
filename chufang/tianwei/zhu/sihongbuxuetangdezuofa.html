      四红补血汤的做法

      这是一款经典的补血养颜汤水，在寒冷的冬天喝上这样的一碗甜汤，整个人都会感觉温暖起来。

      一、主料

      主料用量
      红豆80克
      花生仁(生)60克
      红枣10克
      桂圆10克
      红糖10克

      二、辅料

      辅料用量
      水1500毫升

      三、做法

      步骤
      1.

      准备食材（冰糖换成红糖）；红糖是纯甘蔗熬制的，效果要比白冰糖好

      |
      |
      2.

      将红豆和花生清洗干净，浸泡一夜，红枣去核备用。

      |
      |
      3.

      锅中加入1.5L水，放入红豆和花生，大火烧开，改用小火煲煮20分钟

      |
      |
      4.

      加入龙眼肉和红糖，继续炖煮15分钟，最后加入红枣再煮10分钟即可

      |

      四、功效

      化痰，治流鼻血，健脑，防癌，痛经，经期保健，止痛，驱寒暖身，滋补强身，心血管病。
