      西米花蛤汤的做法

      相信这道菜很少人知道吧，哈哈，我也是第一次听到西米可以打汤的，以前西米都是用来做甜点，打汤也是很不错的哟，晶莹剔透的，美观又美味，所以今天拿出给大家一起分享~

      一、主料

      主料用量
      西米60克
      海带(浸)100克
      花蛤300克

      二、辅料

      辅料用量
      葱10克
      食盐3克
      食用油10克
      姜6克

      三、做法

      步骤
      1.

      食材准备；

      |
      |
      2.

      煮西米（详情步骤图）；

      |
      |
      3.

      煮好的西米如图，用冷水冲一下过虑备用；

      |
      |
      4.

      锅中烧沸水，到入花蛤氽水去淤泥及杂质，煮至花蛤裂开过虑备用；

      |
      |
      5.

      姜洗净，切成薄片，海带、葱切成小段；

      |
      |
      6.

      炒锅烧热，倒入油，放入姜片爆香，依次加入海带、花蛤、西米煮10分钟；

      |
      |
      7.

      最后加入盐、香葱即可。

      |

      四、功效

      健脾，止咳平喘，高血压，清热去火，助消化。
