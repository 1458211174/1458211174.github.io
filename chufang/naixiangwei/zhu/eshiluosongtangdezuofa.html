      俄式罗宋汤的做法

      罗宋汤的做法有很我。收集参考了很多资料后，做了一款最正宗的罗宋汤。

      一、主料

      主料用量
      牛肉300克
      西芹10克
      洋葱2个
      土豆2个
      胡萝卜2个
      包菜5片
      西红柿1个

      二、辅料

      辅料用量
      黄油1小盒
      番茄酱2勺
      牛奶大半盒
      面粉20克
      盐8克
      糖8克
      胡椒粉10克
      味精5克

      三、做法

      步骤
      1.

      新鲜牛肉洗干净。

      |
      |
      2.

      切成牛肉粒，大约3厘米块最好。

      |
      |
      3.

      冷水下锅煮。

      |
      |
      4.

      烧开后撇去浮沫。

      |
      |
      5.

      准备食材：西芹切成粒。（最好不要用普通的芹菜）

      |
      |
      6.

      准备食材：洋葱切成粒。

      |
      |
      7.

      准备食材：土豆切成粒。

      |
      |
      8.

      准备食材：胡萝卜切成粒。

      |
      |
      9.

      准备食材：卷心菜或者包菜撕成小块。

      |
      |
      10.

      准备食材：西红柿用开水泡一下，这样很容易就能撕去皮。

      |
      |
      11.

      准备食材：撕掉皮的西红柿切成粒，和其他蔬菜分开放。

      |
      |
      12.

      各种蔬菜。

      |
      |
      13.

      煸炒洋葱，炒出香味，然后把切成小粒的：卷心菜、胡萝卜、土豆、西芹全放进去煸炒，使其脱水。

      |
      |
      14.

      炒好后一起倒入之前的牛肉汤锅里。

      |
      |
      15.

      黄油。

      |
      |
      16.

      番茄酱。

      |
      |
      17.

      把切好的西红柿放入锅里翻炒，放入两勺番茄酱，和点黄油，煮化后也倒入汤锅里。

      |
      |
      18.

      在汤锅内倒入大半盒牛奶。顿时锅内浓香四溢。

      |
      |
      19.

      锅内下油，在放入面少，炒均匀后也倒入汤锅内。

      |
      |
      20.

      最后加入盐、糖、味精和胡椒粉调味提鲜即成。

      |
