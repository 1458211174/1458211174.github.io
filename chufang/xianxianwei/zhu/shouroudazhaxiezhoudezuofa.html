      瘦肉大闸蟹粥的做法

      海鲜粥咯

      一、主料

      主料用量
      大闸蟹6个
      瘦肉200克

      二、辅料

      辅料用量
      葱50克
      姜少许

      三、做法

      步骤
      1.

      把大闸蟹打开，去腮等不可以吃的部位。清洗干净，备用

      |
      |
      2.

      蟹壳

      |
      |
      3.

      瘦肉切丝，用生抽，料酒抓，可以使肉松软有味道

      |
      |
      4.

      当生抽盐油少许

      |
      |
      5.

      葱葱切碎，姜切片。

      |
      |
      6.

      切碎哈

      |
      |
      7.

      准备的同时把粥熬好，少开后小火炖着，切记不要焦底

      |
      |
      8.

      可以跟粥同时进行，熬蟹汤的时候记得放入姜片，驱寒用

      |
      |
      9.

      放少许盐，煮好装盘。可以直接吃

      |
      |
      10.

      把砂锅的白粥放入蟹汤里，煮开后放入瘦肉。再次开锅。

      |
      |
      11.

      成品出锅啦

      |
      |
      12.

      黄澄澄
