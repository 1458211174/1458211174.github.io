      腊鸭煲笋干汤的做法

      春天的气候，往往阴晴不定，是流感，感冒的高峰期。这个时候除了做足保暖工作以外，更重要的是提升身体的免疫力。南方的冬天，都没有让人感觉到寒冷，到是这倒春寒，彻底让我们折服了，呵呵。。。 办公室的同事，大多都中了流感的招，精灵这易生病的体质，整日提心吊胆的，真心不容易。平时工作忙，下班后又要带娃，运动量严重不足，呜呜。。。咱就先从食补开始下手，身体搞好了，免疫力是不是会提高一些呢？反正家里一堆食材，总得想着法子来消耗，这会不会给自已找个想吃的理由。 汤汤水水的，总是上班一族，周末必休的功课；腊鸭，笋干，都是公司的朋友送的。腊鸭是朋友的朋友自家养的，然后腌制晒干的，然后转送了一支给我，一直挂阳台的墙壁上，进进出出的总能看见它，似乎总在像我招手，让我好好宠爱它；笋干，是闺密给的，之前用它来炖猪蹄，烧排骨，都很好味道。两者结合，加一些蚝豉，清甜的萝卜，汤鲜，咸，香，甜，好滋味，喝上两碗都不嫌腻，嘻嘻。。。

      一、主料

      主料用量
      腊鸭600克
      笋干50克
      蚝豉30克
      胡萝卜50克

      二、辅料

      辅料用量
      生姜15克
      料酒20毫升
      盐3克
      鸡粉1克
      干辣椒1个

      三、做法

      步骤
      1.

      各式食材准备。

      |
      |
      2.

      腊鸭剁小块，加水浸泡8小时。（精灵提前一晚剁的，浸泡了一晚上）

      |
      |
      3.

      笋干，蚝豉加水浸泡2小时。

      |
      |
      4.

      浸泡好的腊鸭，加姜片焯水。

      |
      |
      5.

      泡发好的笋干，清先杂质后下锅焯水。

      |
      |
      6.

      焯水好的腊鸭，笋干倒入电炖锅中。

      |
      |
      7.

      加姜片，干辣椒，泡发好的蚝豉。

      |
      |
      8.

      加足热水，料酒20ml。

      |
      |
      9.

      接通电源，炖2小时。

      |
      |
      10.

      加入胡萝卜，加盐调味，再炖1小时。

      |
      |
      11.

      加鸡粉调味。

      |
      |
      12.

      最后再炖10分钟后，断电装碗即可食用。
