      虫草花炖藕的做法

      德寿素厨房为您推荐美味营养的虫草花炖藕

      一、主料

      主料用量
      藕550克
      虫草花200克
      花生40克

      二、辅料

      辅料用量
      盐2小匙

      三、做法

      步骤
      1.

      根据人数准备好适量的蔬菜，藕切大块

      |
      |
      2.

      藕先放入电压力锅底部，再放入花生和虫草花，加入适量的清水、2小匙盐

      |
      |
      3.

      加入1汤匙植物油

      |
      |
      4.

      按煲汤键，坐等开吃（根据各家电压力锅品牌不同，根据实际情况设置时间）

      |

      四、功效

      防治胃病，贫血，开胃消食，治流鼻血，抗衰老，润肺，止血，心血管病，补虚，保护心脏，促进发育，益智。
