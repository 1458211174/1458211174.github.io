      桃胶皂角米银耳羹的做法

      桃胶、银耳富含胶质，平民界的美容圣品。这一道糖水羹献给大家。

      一、主料

      主料用量
      银耳半朵
      桃胶40克
      皂角米40克

      二、辅料

      辅料用量
      冰糖60克
      枸杞少许
      红枣2颗
      百合20克

      三、做法

      步骤
      1.

      桃胶泡发12小时以上；皂角米泡发6小时以上，我一般是跟桃胶一起泡发；银耳提前半小时泡发即可。

      |
      |
      2.

      清洗桃胶上的黑色杂质

      |
      |
      3.

      将所有食材放入炖锅中，炖1.5-2小时。快煮好前投入冰糖

      |
      |
      4.

      出炉啦，是不是美美的。

      |
      |
      5.

      是不是很美味呢～

      |
      |
      6.

      按个人喜好添加鲜牛奶或者椰子汁

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体。
