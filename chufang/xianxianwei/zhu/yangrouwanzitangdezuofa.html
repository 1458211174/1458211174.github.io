      羊肉丸子汤的做法

      煮了些羊肉丸子汤，如果有蔬菜可以放些小白菜或者冬瓜，都很好喝

      一、主料

      主料用量
      羊肉300克

      二、辅料

      辅料用量
      葱5克
      姜5克
      食盐5克
      生抽5克
      淀粉15克
      料酒5克

      三、做法

      步骤
      1.

      准备一块有肥有瘦的羊肉

      |
      |
      2.

      我没有料理机所以我用剁的，越细腻丸子越光滑

      |
      |
      3.

      放姜末继续剁，让味道融合

      |
      |
      4.

      放盐，花椒粉，生抽，老抽继续剁

      |
      |
      5.

      放淀粉继续剁

      |
      |
      6.

      剁到肉馅细腻而且上劲就可以了

      |
      |
      7.

      烧一小锅水，水冒小泡下丸子，从虎口挤出来

      |
      |
      8.

      全都挤进去了，撇去浮沫

      |
      |
      9.

      放入葱姜和盐调味

      |
      |
      10.

      小火慢煮，煮到丸子飘起来就好了

      |

      四、功效

      健胃，温补肝肾，温经止痛，保护视力，抵抗力。
