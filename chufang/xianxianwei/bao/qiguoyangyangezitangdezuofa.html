      汽锅养颜鸽子汤的做法

      汽锅养颜鸽子汤，一听名字就知道有滋补美颜的功效

      一、主料

      主料用量
      鸽子一只

      二、辅料

      辅料用量
      大红枣一只
      葱一把
      香菇(干)若干
      姜三片
      料酒适量

      三、做法

      步骤
      1.

      将鸽子洗净，切块，放在一边备用，葱洗净扎成一个圈备用，大红枣冲洗干净，切成小块，姜切片备用，干香菇泡开洗净，然后把葱，红枣，香菇，装入汽锅内

      |
      |
      2.

      热锅热油，把姜片先爆香，放入鸽子翻炒几下，放少许料酒，翻炒均匀

      |
      |
      3.

      装锅，汽锅内装满水，不喜欢汤的可以少放点，再拿一个与汽锅差不多大的锅装8成水，把汽锅叠上去，看看是不是贴合，不贴合的话可以用纱布缠绕把缝隙填满

      |
      |
      4.

      盖上盖子，大火煮沸，然后小火煲一个小时至一个半小时，注意下面的水烧干，发现水要烧干了，可以加开水进去继续煮

      |
