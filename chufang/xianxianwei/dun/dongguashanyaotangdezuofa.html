      冬瓜山药汤的做法

      德寿素厨房，为您推荐春季养生的冬瓜山药汤。春季多吃山药、冬瓜，有利健康。

      一、主料

      主料用量
      冬瓜430克
      山药250克
      胡萝卜130克
      枸杞10克

      二、辅料

      辅料用量
      盐1.5小匙
      植物油半汤匙

      三、做法

      步骤
      1.

      根据人数准备好适量的蔬菜

      |
      |
      2.

      切块

      |
      |
      3.

      把所有食材放入电压力锅内，加入适量清水、盐1.5小匙（少加盐，汤鲜、不加味精，汤自然甜）

      |
      |
      4.

      加入半汤匙植物油

      |
      |
      5.

      按汤类键，坐等开吃（根据各家电压力锅品牌不同，依据实际情况设置时间）

      |

      四、功效

      减肥瘦身，滋养肾气，化痰，防癌，润肤美容，降血糖，降胆固醇，降血脂，软化血管，助减肥，降压，滋肾益精，延年益寿，助消化，抵抗力，降糖，保护视力，保肝，降三高。
