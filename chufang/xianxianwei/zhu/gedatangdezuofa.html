      疙瘩汤的做法

      冬日有雾霾的天气，不想出门遭罪，利用家里仅有的食材做了疙瘩汤，饱饱的一碗，很暖。

      一、主料

      主料用量
      面粉80克
      大头菜四分之一个
      小菠菜6棵
      鸡蛋1个
      青椒三分之一个
      长绿豆芽少许

      二、辅料

      辅料用量
      大葱少许
      盐1茶匙
      香油1滴
      生抽1汤匙
      花生油适量
      水适量

      三、做法

      步骤
      1.

      面粉和少许清水，顺一个方向搅拌，搅成差不多一小坨一小坨的样子。面疙瘩别搅得太大，小点的更容易熟；水一次别加太多，一点一点的加，我在搅面疙瘩的时候磕了一个鸡蛋，所以面疙瘩看起来偏黄色，这样的面疙瘩吃起来也比较筋道（听妈妈说的）。

      |
      |
      2.

      大头菜切洗净切丝；小菠菜择好洗净；青椒洗净切丁；瘦猪肉切丁；葱白洗净切丁。豆芽我是临出锅的时候顺手捞了一把加的，所以图上没拍到。

      |
      |
      3.

      锅内倒油烧热，加肉丁、葱花碎、青椒丁炒香，加少许生抽翻炒出香味后加水（水量视个人情况而定，喜欢喝稀点的就多加一点，喜欢喝稠一点就少加水；后面下完疙瘩和蔬菜后觉得水加少了可以再添的，多煮一会就是了）。

      |
      |
      4.

      汤水烧开后下面疙瘩，记得中间要不断搅动，避免面疙瘩都粘到一起。

      |
      |
      5.

      疙瘩汤煮至汤水冒泡时下蔬菜（大头菜、小菠菜、豆芽），再次冒泡时加一茶匙盐，滴一滴香油，继续煮2～3分钟后关火，出锅。

      |

      四、功效

      安神，润肠通便，防癌，防治胃病，止咳化痰，健脑益智，保肝，动脉硬化，延年益寿，美容护肤，缓解疲劳，清热去火，散瘀止疼，心血管病，消食，助消化。
