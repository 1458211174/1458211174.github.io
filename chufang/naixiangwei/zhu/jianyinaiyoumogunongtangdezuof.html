      简易奶油蘑菇浓汤的做法

      每次吃牛扒都要来一碗，外面三四十一小碗，贵的不要不要的啊，在家自己做，想要多香浓就有多香浓，简易不耗时

      一、主料

      主料用量
      口蘑3两
      淡奶油10毫升

      二、辅料

      辅料用量
      盐少许
      鸡精少许
      油3毫升

      三、做法

      步骤
      1.

      7、8颗蘑菇这样，还有淡奶油

      |
      |
      2.

      蘑菇去蒂，直接一拗就断了，特别容易处理

      |
      |
      3.

      蘑菇切丁

      |
      |
      4.

      热锅，加入少量油，不能加太多，蘑菇吸油的，会让汤特别油腻

      |
      |
      5.

      蘑菇爆香，将蘑菇炒软即可，加入开水，切记开水哟，还有少量奶油

      |
      |
      6.

      煮沸之后，倒入榨汁机里面搅碎，注意有热气，压紧盖子，大概20秒即可

      |
      |
      7.

      搅碎之后的汤倒回锅中，一边加热一边加入调料，否则调料会不均匀，加入盐和鸡精，自己尝试一下味道，咸淡因人而异嘛

      |

      四、功效

      抵抗力，保肝，美容瘦身。
