      百里香猪尾汤

      这汤实际就是家庭版罗宋汤的演变，只是加了少许百里香，味道立马更加西式。
      汤汁很浓稠，颜色完全是番茄色的，千万别加番茄沙司，那里完全没有番茄都是一些化学元素配置的不利于健康；
      一碗汤，一碗米饭，再来一盘绿色蔬菜，沐浴着春天的阳光开动我美美的午餐，如阳光般温暖的幸福啊！

      一、材料

      【食材】猪尾巴 2个// 小土豆 5个 // 小西红柿若干 // 洋葱 半个 // 胡萝卜 半个// 生姜 1片 // 盐、糖和百里香

      二、做法

      步骤
      1、猪尾巴切块，加一片生姜冷水煮开，去浮沫；
      2、煮猪尾巴的空挡可以准备其他食材；
      3、将洋葱却片，西红柿切片，胡萝卜切片；
      4、热锅下少许油炒香洋葱，等洋葱完全变软，倒入西红柿，加入盐糖和百里香，翻炒至烂；
      5、再下胡萝卜一起翻炒，和煮开的猪尾汤一起加入煲锅煲至2.5 — 3小时，我煲三小时；
      6、在煲至1小时时，将小土豆削皮，整个土豆加入汤锅即可。
      猪尾巴 小土豆 西红柿 洋葱 胡萝卜 生姜 百里香
