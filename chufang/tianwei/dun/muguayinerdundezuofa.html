      木瓜银耳炖的做法

      “木瓜银耳炖“ 爽滑的银耳，香甜的木瓜，做法也是超级简单哟！ -银耳- 几乎成了美容养颜，滋补的代名词啦！ 对菇凉也是超级好的哟！ 而且，大叔偷偷地告诉你哦~银耳可是绝对的 “膨化食品” 一颗银耳，最后就可以膨胀出好多好多， 学会了这道甜品，就可以一次性吃个饱啦！

      一、主料

      主料用量
      木瓜400克
      银耳100克
      冰糖40克

      二、辅料

      辅料用量
      百合10克
      枸杞15克

      三、做法

      步骤
      1.

      -食材-

      木瓜
      银耳
      冰糖
      百合
      枸杞

      |
      |
      2.

      干木耳用水泡发，去除根部，切成小块

      |
      |
      3.

      木瓜去皮，切成小块

      |
      |
      4.

      干百合，泡水撕碎

      |
      |
      5.

      银耳加水，放入锅中，

      如果小伙伴不能很好地掌握水的比例，请看图哟

      |
      |
      6.

      银耳加水烧开，加入适量冰糖
      并小火烧20分钟。让银耳软化，膨胀。

      |
      |
      7.

      加入木瓜，继续小火炖煮到银耳粘稠

      胶质炖入汤汁里即可~

      |
      |
      8.

      最后出锅前加入适量冰糖，
      加入百合，枸杞点缀，即可

      |
      |
      9.

      如果喜欢吃冰的，也可以盖冷藏到冰冰凉，

      口感会完全不一样哟！~上保鲜膜，

      |

      四、功效

      健脾消食，驱虫，防癌，强身健体，保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，清热解暑，润肺化痰，化痰止咳，清热去火，高血压。
