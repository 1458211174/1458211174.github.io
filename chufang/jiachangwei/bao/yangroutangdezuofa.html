      羊肉汤的做法

      冬季很多人开始吃羊肉。 以前我基本不吃羊肉的，今年我们家隔壁开了一家羊肉馆。每天都现杀羊，所以羊肉很新鲜，生意火爆。 我去吃了好几次，发现羊肉汤真的很喝 经过学习后，我也会做羊肉汤了。 清淡的，很好喝。

      一、主料

      主料用量
      羊肉500克

      二、辅料

      辅料用量
      葱三棵
      食盐2勺
      黑胡椒粉2勺
      姜2块
      花椒适量
      桂皮1个
      八角4个

      三、做法

      步骤
      1.

      羊肉洗干净后，加盐，姜，除水。去掉白沫。

      |
      |
      2.

      除水后的羊肉用清水洗一下，备用

      |
      |
      3.

      姜切片。羊肉和姜片一起放入锅里

      |
      |
      4.

      准备后好佐料

      |
      |
      5.

      把佐料放入纱布里包好。一起放入锅里。加水加盐开始炖。我用的是高压锅。上气后压20分钟关火。

      |
      |
      6.

      炖羊肉时，开始切葱。

      |
      |
      7.

      在碗里放入胡椒粉，然后盛入羊肉汤。在上面撒上葱花即可！美味的羊肉汤就做好了。

      |

      四、功效

      健胃，温补肝肾，温经止痛，保护视力，抵抗力。
