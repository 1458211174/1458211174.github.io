      粉葛萝卜煲猪横脷的做法

      节后，越来越多的人加入了加班熬夜的“队伍”，加之不良的饮食习惯，许多人的肝火都非常的旺盛，此时，粉葛就成为舒缓各种“火”的利器。 而猪横脷是猪的脾脏，有健脾胃，助消化，养肺润燥，泽颜面色，去肝火的功效，常被用来煲汤。二者合而为汤，清热泻火还能让我们保持心情的宁静，拥有一个健康的好睡眠。

      一、主料

      主料用量
      粉葛300克
      胡萝卜1根
      猪骨200克
      猪横脷1条

      二、辅料

      辅料用量
      姜2片
      盐适量

      三、做法

      步骤
      1.

      准备好材料，粉葛洗干净，削去外皮，再切成小方块。胡萝卜去皮切大小相当的块。

      |
      |
      2.

      将猪横脷刮去苔；和猪骨一起洗净飞水（放入凉水中，大火煮开沸滚1~2分钟，然 后将水倒掉，冲洗干净）；

      |
      |
      3.

      所有材料和姜片一起下瓦煲，加水2500毫升（约10碗量）按自己家庭情况；大火滚沸后改小火煲约1 个半至2小时.

      |
      |
      4.

      出锅前加入食盐即可。

      |

      四、功效

      血栓，清热去火，降糖，健脑益智，高血压，解酒醒酒，养颜护肤，清热解暑，抵抗力，保护视力，助消化，骨质疏松。
