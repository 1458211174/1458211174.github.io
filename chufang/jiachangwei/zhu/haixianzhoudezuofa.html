      海鲜粥的做法

      家乡寄来的新鲜虾、鱼和猪肉卷，想吃潮汕海鲜粥了……

      一、主料

      主料用量
      大米三人份
      活虾适量
      鱼片适量
      潮汕猪肉卷适量

      二、辅料

      辅料用量
      姜丝少许
      盐少许
      芹菜适量
      蒜少许

      三、做法

      步骤
      1.

      先淘米煮饭

      |
      |
      2.

      猪肉卷切丁，芹菜洗干净切粒，虾洗干净待炒出香味，鱼洗干净待蒸熟

      |
      |
      3.

      把鱼放进锅里蒸15分钟，把鱼肉撕下来

      |
      |
      4.

      先爆蒜葱然后下虾翻炒几下之后焖三分钟，加盐翻炒再出锅

      |
      |
      5.

      把米饭放入锅中加水，把猪肉卷、虾、鱼肉、芹菜放进去煮成粥

      |
      |
      6.

      10分钟左右煮好可以吃啦

      |

      四、功效

      适合婴儿，强身健体。
