      茶树菇炖鸡的做法

      逛街时无意看到有新鲜的茶树菇，这么冷的天当然要炖汤来补冬咯

      一、主料

      主料用量
      鸡1/2只
      茶树菇100克

      二、辅料

      辅料用量
      食盐适量
      料酒2勺
      姜1块
      葱1颗

      三、做法

      步骤
      1.

      半片鸡洗净，剁成小块

      |
      |
      2.

      与冷水，料酒一起下锅煮开后煮五分钟，捞出洗净血沫

      |
      |
      3.

      加入葱姜，料酒炖一个小时

      |
      |
      4.

      等到满屋都是鸡汤的香味时加入茶树菇炖20分钟左右

      |
      |
      5.

      汤鲜肉嫩，菌菇爽口

      |

      四、功效

      防癌，抗衰老，治尿床。
