      南瓜奶油汤的做法

      南瓜奶油汤，是一道简单易作的家常汤菜。因为南瓜含有丰富的维生素B，奶油中则蕴含钙质和蛋白质，这些都是儿童成长不可或缺的成分，可以帮助宝宝健康生长。可爱的奶黄色汤，加上奶油的奶香味，让这一款南瓜汤成为色香味俱全的营养早餐，而且作为午后甜点也不不错。

      一、主料

      主料用量
      老南瓜500克
      大米50克
      奶油适量
      纯净水适量

      二、辅料

      辅料用量
      纯净水适量

      三、做法

      步骤
      1.

      南瓜去皮洗净切块、大米洗净，倒入破壁机。

      |
      |
      2.

      加入适量清水，水位刚刚淹到南瓜就是，按下米糊功能键。

      |
      |
      3.

      制成的南瓜汤细腻柔滑，原汁原味。

      |
      |
      4.

      倒入盘里后加入奶油，用牙签在汤的表面挑出自己喜欢的花纹，美美的南瓜奶油汤就做好了。
