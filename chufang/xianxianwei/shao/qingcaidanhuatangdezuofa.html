      青菜蛋花汤的做法

      青菜蛋花汤～平淡有营养

      一、主料

      主料用量
      青菜300克

      二、辅料

      辅料用量
      盐20克
      花生油5毫升

      三、做法

      步骤
      1.

      把青菜洗干净

      |
      |
      2.

      把青菜切段

      |
      |
      3.

      锅里倒油，把葱放进去炒炒

      |
      |
      4.

      把青菜放进锅里炒至变软

      |
      |
      5.

      往锅里加水，放适量的盐

      |
      |
      6.

      把鸡蛋打在碗里

      |
      |
      7.

      用筷子把鸡蛋搅拌开

      |
      |
      8.

      把鸡蛋倒进锅里

      |
      |
      9.

      汤开了就可以出锅啦

      |
