      美龄粥的做法

      相传，有段时间宋美龄茶饭不思，于是府里的大厨用香米和豆浆等食材熬了一锅粥，宋美龄吃了胃口大开，后来这道粥流传到民间，名曰“美龄粥”。 自己在家煮美龄粥最好选择糯米和大米两种米，糯米煮完口感比较软糯，而且半透明的米粒浮在豆浆上的样子莫名会有些萌，而大米的口感则会更佳的饱满有弹性，两种米的不同口感赋予了美龄粥更多层次感。浓香四溢的豆浆香气中一颗颗饱满Q弹的米粒在舌尖跳跃着，偶尔还可以吃到绵软的山药颗粒，一碗下肚香气依旧留在舌尖难以消退。 山药健脾开胃、美容养颜，还有诸多的食疗效果。大米和糯米搭配，补中益气，还增加粘稠度。豆浆的加入是亮点，豆浆加入正好谷豆结合，蛋白质互相补充，也提高了蛋白质的利用率。宋美龄能活到106岁，这美龄粥也起到了很大的作用。

      一、主料

      主料用量
      黄豆60克
      糯米40克
      大米20克
      山药150克
      水900毫升
      冰糖25克

      二、辅料

      辅料用量
      枸杞子适量
      蔓越莓干适量

      三、做法

      步骤
      1.

      材料图。

      |
      |
      2.

      黄豆、糯米、大米淘洗干净冷水浸泡4小时以上。

      |
      |
      3.

      泡好的黄豆捞出放进破壁料理机中，加入900ml的水，选择豆浆功能打成浆。

      |
      |
      4.

      等待料理机工作的时间将山药冲洗干净，带皮上锅蒸20分钟至熟透。然后取出晾凉后撕掉山药皮，用勺子将山药捣成泥，不用太细，有一点颗粒感喝的时候更有层次。

      |
      |
      5.

      打好的豆浆转至汤锅中，加入泡好的糯米、大米，加入冰糖，中火煮开转小火煮30分钟（锅盖留条小缝防溢锅），中途记得不时搅拌一下预防糯米沉底糊锅。

      |
      |
      6.

      30分钟后粥基本就煮好了，这时下山药泥和冲洗干净的枸杞子，搅拌均匀即可关火饮用了。

      |

      四、功效

      软化血管，益智，糖尿病，降脂，美白，防癌，脂肪肝，更年期，抵抗力，增强体质，健脾胃，缓解尿频，适合婴儿，强身健体，滋肾益精，降血糖，延年益寿，助消化，美容养颜，清热解暑，润肺化痰。
