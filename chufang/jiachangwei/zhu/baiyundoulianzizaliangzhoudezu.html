      白芸豆莲子杂粮粥的做法

      莲子、白芸豆加小米、糙米、玉米糁和苦荞米 煮粥，营养平衡，口感特别好。春季食用这款粥，有调养的作用。 莲子的营养成分丰富，常作为一种药材应用到临床医学中，莲子的营养价值之高由此可见一斑。莲子含有丰富的蛋白质、淀粉、磷脂、生物碱、类黄酮以及多种维生素等营养保健成分，莲子中铁的含量也非常丰富。 白芸豆营养丰富含蛋白质、脂肪、胡萝卜素及多种微量元素及丰富的维生素。作为家常菜蔬，健脾利水，用于治疗脾胃虚弱，水湿内停所致食欲不振、便溏、脚气病、水肿等具有不错的辅助治疗作用。

      一、主料

      主料用量
      白芸豆20颗
      莲子50颗

      二、辅料

      辅料用量
      小米100克
      玉米糁（细）50克
      糙米40克
      苦荞米15克
      纯净水3500毫升
      调和油几滴

      三、做法

      步骤
      1.

      莲子和芸豆提前浸泡，泡好后把莲子掰开，去掉残留的芯。

      |
      |
      2.

      盛出小米和玉米糁。

      |
      |
      3.

      盛出糙米和苦荞米。

      |
      |
      4.

      锅里加入纯净水，放入莲子和芸豆开火。锅开后加入洗干净的米，加入几滴食用油（防溢出），大火烧开后改小火煮至成熟既好，中间不要揭盖。

      |
      |
      5.

      煮熟的粥。

      |
      |
      6.

      装碗的粥。

      |

      四、功效

      治慢性病，养发，强身健体，抵抗力，防癌，降压，安神，益精。
