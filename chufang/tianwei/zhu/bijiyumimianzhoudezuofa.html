      荸荠玉米面粥的做法

      荸荠也称为马蹄。甜甜的味道，脆脆的口感无论是生食还是入菜，或是入汤都是美味。这次用荸荠与玉米面组合煮粥，也非常好喝。看似简单的家常粥，有着润肺，清热，止咳，滋阴润燥，润肠等功效。虽然剥荸荠皮很麻烦，但家人喜欢，又利于身体健康，麻烦些也值得。

      一、主料

      主料用量
      荸荠300克
      玉米面80克

      二、辅料

      辅料用量
      水1500毫升

      三、做法

      步骤
      1.

      荸荠洗干净

      |
      |
      2.

      将荸荠皮剥干净用冷水浸泡

      |
      |
      3.

      取适量玉米面用冷水调制成面糊

      |
      |
      4.

      荸荠切成块

      |
      |
      5.

      煮锅加水，荸荠入锅煮

      |
      |
      6.

      开锅后面糊入锅同煮,其间要适时搅动，以免粘锅,煮至食材熟，关火，焖5、6分钟后盛出享用。喜欢甜，可以加适量糖

      |

      四、功效

      促发育，降压，通便排毒，清热解毒，减肥瘦身，防癌，降血压，保护视力，润肠通便，健脑，抗衰老。
