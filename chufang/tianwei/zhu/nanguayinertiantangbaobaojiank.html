      南瓜银耳甜汤 宝宝健康食谱的做法

      你好亲爱的，我是食谱君，给宝宝制作的近400道辅食，都收录在了我的公众号《宝宝健康食谱》里，关注即可查看噢！包括主食、汤羹、菜品、烘焙、点心、食疗等分类，再也不用为宝宝吃啥而发愁啦！ 我第一次做给宝宝喝的时候，他不喜欢银耳嚼在口中格叽格叽的口感，都是把银耳捞出，只喝汤，那营养价值就折去一半。 索性我后来就尝试像今天这样打成羹，银耳无味，配上有甜味的南瓜再加上冰糖，浓郁润滑的口感让他一饮而尽，非常喜欢。 尤其在冬季，能帮助宝宝健脾、润肺去燥止咳，把食材都打成糊，8个月的小宝宝也能喝。

      一、主料

      主料用量
      南瓜一小块
      银耳适量
      冰糖选用

      二、辅料

      辅料用量
      无无

      三、做法

      步骤
      1.

      参考月龄：8个月以上
      食材准备：南瓜、银耳、冰糖2-3块
      烹饪时间：40-50分钟
      难度分析：初级

      |
      |
      2.

      南瓜清洗干净，去掉皮和籽，切成小块或者小丁备用。

      |
      |
      3.

      银耳提前放入冷水中，浸泡1-2个小时，涨发后摘小朵洗干净。

      |
      |
      4.

      锅内放适量水，水量要全部没过银耳，加入银耳和冰糖大火煮开转小火，煮20-30分钟左右，让银耳的胶质完全软化。

      |
      |
      5.

      加入南瓜丁继续煮。

      Tips：煲这类养生甜品，一定要又耐心哦，慢慢地将食材中有益物质炖出来，不可心急，用文火慢慢炖最佳。

      |
      |
      6.

      煮至南瓜丁完全软烂熟透就可以关火了。

      |
      |
      7.

      煮好后放入料理机中。
      打成细腻的羹，这样即使是8个月的宝宝也可以喝啦，而且很容易消化吸收。

      |
      |
      8.

      浓浓的南瓜银耳羹，味道是甜甜的，有南瓜的浓郁醇香，也有银耳的润滑，宝宝很喜欢。

      |

      四、功效

      防癌，治胃病，降糖，治前列腺，保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体，清热解暑，润肺化痰，化痰止咳，清热去火，高血压。
