      猪肝瘦肉苦瓜汤的做法

      又一奇葩的搭配，喜欢猪肝，瘦肉，苦瓜全部放在一起，然后煮汤。味道甘甘甜甜也很不错哦

      一、主料

      主料用量
      苦瓜200克
      瘦肉150克
      猪肝150克

      二、辅料

      辅料用量
      油20克
      盐2克
      酱油5克
      生粉5克
      姜片2片

      三、做法

      步骤
      1.

      猪肝瘦肉洗净，用生粉，酱油，油，盐腌一下

      |
      |
      2.

      苦瓜切好

      |
      |
      3.

      锅中放入适量水，放入苦瓜烧开

      |
      |
      4.

      烧开后，倒入猪肝瘦肉重新烧开

      |
      |
      5.

      肉煮开后，再煮8分钟，猪肝熟透即可出锅

      |

      四、功效

      清热祛暑，强身健体，健脾开胃，保护视力，防治痢疾，祛瘀消肿，缓解中暑，防治痱子，治结膜炎，防坏血病，软化血管，防心脏病，防癌，清热去火，抵抗力，强健身体，消肿散瘀，降血糖，降血脂。
