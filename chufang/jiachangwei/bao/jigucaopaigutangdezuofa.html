      鸡骨草排骨汤的做法

      最近冒出好几颗痘痘，听说鸡骨草清热解毒，就去药店买来试试吧

      一、主料

      主料用量
      鸡骨草50克
      排骨一人份
      胡萝卜一根

      二、辅料

      辅料用量
      响螺片一小片
      盐适量

      三、做法

      步骤
      1.

      响螺片选小片的，加热水泡软，切小块备用

      |
      |
      2.

      胡萝卜洗净去皮切片

      |
      |
      3.

      鸡骨草洗干净后放入煲汤砂袋里

      |
      |
      4.

      排骨焯水后捞出，全部放进锅里煮

      |

      四、功效

      补肾，润肺，抵抗力，降糖，保护视力，助消化。
