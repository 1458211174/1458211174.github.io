      萝卜炖筒骨的做法

      萝卜俗称菜头，过年吃萝卜，讨个好彩头，预祝大家新年快乐！冬吃萝卜夏吃姜，冬天的萝卜赛人参！萝卜含有能诱导人体自身产生干扰素的多种微量元素，可增强机体免疫力，并能抑制癌细胞的生长，对防癌，抗癌有重要意义。萝卜中的芥子油和膳食纤维可促进胃肠蠕动，有助于体内废物的排出。常吃萝卜可降低血脂、软化血管、稳定血压，预防冠心病、动脉硬化、胆石症等疾病。

      一、主料

      主料用量
      萝卜1根
      筒骨1根

      二、辅料

      辅料用量
      生姜6片
      大蒜6瓣
      盐2茶匙
      料酒3大勺
      茶叶水1碗

      三、做法

      步骤
      1.

      天气好好适合出门拔萝卜，于是我就去地里拔萝了，哈哈！

      |
      |
      2.

      拔了萝卜就要把它好好的吃掉
      那就炖锅筒骨汤吃吃吧，暖胃暖心
      首先把该洗的材料都洗干净，该去皮的去皮

      |
      |
      3.

      冷水下骨头、3片生姜、3瓣蒜、1勺半料酒

      |
      |
      4.

      煮沸后再煮两分钟，把血水都煮出来

      |
      |
      5.

      然后捞出骨头用温水冲洗干净（用冷水还是热水随便你，反正都能炖烂）

      |
      |
      6.

      重新下锅炖（加入姜片、大蒜、料酒、茶叶水）水开后转小火

      |
      |
      7.

      炖差不多1.5～2个小时点时候把萝卜去皮切块

      |
      |
      8.

      然后把萝卜加入骨头汤里和骨头一起炖（水不够的话可以稍微加点水）

      |
      |
      9.

      萝卜炖透了就关火，撒点葱花就OK啦

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体。
