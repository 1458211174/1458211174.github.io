      红糖煮鸡蛋的做法

      红糖煮鸡蛋可让身体温暖，增加能量，活络气血，加快血液循环，月经也会排得较为顺畅。

      一、主料

      主料用量
      红糖1.5茶匙
      鸡蛋2个
      红枣5颗

      二、辅料

      辅料用量
      水500克

      三、做法

      步骤
      1.

      在锅里放入适量的清水烧开

      |
      |
      2.

      红枣洗净去核

      |
      |
      3.

      把鸡蛋打到煮开的水里，转小火

      |
      |
      4.

      待水再开时，放入红枣、红糖，用小火煮15分钟即可使用！

      |

      四、功效

      痛经，经期保健，止痛，驱寒暖身，滋补强身，心血管病，健脑益智，保肝，动脉硬化，防癌，延年益寿，美容护肤，治流鼻血。
