      简单小鸡汤的做法

      最近压力大，终于有个休息的时间，不想出门，炖锅鸡汤滋补一下吧。非常简单，单身狗也能轻易驾驭。

      一、主料

      主料用量
      鸡翅根6块
      山药一小段

      二、辅料

      辅料用量
      白玉菇半颗
      姜两片
      魔芋节（丝）少量
      红枣四颗
      枸杞子少量
      醋一勺
      料酒2勺

      三、做法

      步骤
      1.

      清洗以上所有材料，山药削皮滚刀切块，翅中入沸水3分钟，撇去浮末，捞出

      |
      |
      2.

      砂锅中加至足够的水，不要太满，会溢出，加入姜片，翅中，山药，开着盖，中火煮沸

      |
      |
      3.

      加一点醋，一点料酒，盖上盖子，小火煮20分钟之后再放入红枣，白玉菇，魔芋节（丝），继续褒20分钟

      |
      |
      4.

      加入少量枸杞子，放一点盐～十分钟后关火。可以焖一会。出锅

      |

      四、功效

      滋肾益精，降血糖，延年益寿，助消化。
