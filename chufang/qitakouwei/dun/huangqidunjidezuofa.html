      黄芪炖鸡的做法

      女人补气血一定要喝它！

      一、主料

      主料用量
      笨鸡一只

      二、辅料

      辅料用量
      黄芪20克
      西洋参20克
      当归20克
      白芍20克
      大枣六个
      生板栗六个
      葱白适量
      姜两片
      盐适量

      三、做法

      步骤
      1.

      将准备好的药材切片备用！鸡洗净切块，将鸡体内的内脏和油脂去掉留下能吃的部分！

      |
      |
      2.

      锅家适量的水，鸡块洗净下锅！放上准备好的材料开火炖！先不要放板栗

      |
      |
      3.

      炖半小时后家去皮的板栗和适量的盐在炖30分钟或者一小时，鸡炖烂就好！

      |
      |
      4.

      我炖了一个半小时鸡肉才烂，我用的是养了约三年的笨鸡！烫很漂亮吧！

      |

      四、功效

      助消化，骨质疏松。
