      草莓布丁的做法

      布丁软绵绵的口感甜甜的，是很多人必备的下午茶甜点。可是大家会在想外面买的布丁安全吗？嘿嘿，别担心，今天我来教大家做草莓布丁，这款布丁做起来非常的简单，材料费也很便宜，做出来也非常的好看，作为下午茶的甜点或是生日上的甜品，是非常不错的。自己做不仅口感好做出来还有一种满满的成就感，大家可以拿去跟亲朋好友去好好分享。是不是很有成就感？很骄傲，也很开心呢。嘻嘻，可以去炫炫啦！

      一、主料

      主料用量
      好妈妈牌布丁粉一盒
      清水450毫升

      二、辅料

      辅料用量
      水果罐头可有可无
      巧克力可有可无
      果酱可有可无

      三、做法

      步骤
      1.

      材料很简单哦，只要一盒布丁粉，（口味随机，品牌随机）和一杯清水（由于不同的品牌的要求不一样，所以具体的大家可以看说明哦(´-ω-`)）

      |
      |
      2.

      这里要用到量杯，水多的话，布丁做不出来会变成奶昔，水少的话做出来口感也不是特别的好，所以建议大家使用量杯。如果没有量杯的话可以用矿泉水的水瓶或是纸杯的容积，或是周围水瓶子的容积进行换算，估计一下就可以了。别忘了，水必须是热水。

      |
      |
      3.

      将布丁粉全部倒入450毫升的热水中，并均匀的搅拌，直到变稠。

      |
      |
      4.

      刚刚加完布丁粉时水中会出现一些粉色的颗粒物状物质，这就是刚刚做的布丁粉，我们需要不停的搅拌直至布丁粉消失为止。

      |
      |
      5.

      最好的布丁液就是搅拌过程中会有一层胶装物质敷在上面

      |
      |
      6.

      做好后布丁液还是热的，这时应该冷却好再冷藏

      |
      |
      7.

      把布丁液装好后会有气泡，这是震一下，把气泡弄没才可以冷藏哦

      |
      |
      8.

      素不素美美哒
