      西红柿鸡蛋菠菜疙瘩汤（拌汤）的做法

      又饿又懒的时候，这是最快的！又美味又简单！还可以吃饱！比心

      一、主料

      主料用量
      面粉多半碗
      西红柿一个
      鸡蛋两个
      菠菜大的一颗，小的两三颗

      二、辅料

      辅料用量
      花生油适量
      花椒粉／鲜姜粉适量
      白胡椒适量
      香油少量
      生抽两小勺

      三、做法

      步骤
      1.

      菠菜切段

      |
      |
      2.

      西红柿切块

      |
      |
      3.

      锅里倒入少许花生油，把西红柿翻炒一下，放入粉状调料以及生抽，少许盐

      |
      |
      4.

      等到西红柿炒软出汤汁，就可以加水了！

      |
      |
      5.

      等锅烧开的时间可以拌面疙瘩了！

      |
      |
      6.

      我的疙瘩比较碎，煮出来会比较浓稠一点，这是成品

      |
      |
      7.

      面疙瘩拌好，锅也开了，把它倒入锅中，记得慢一点，小心烫手！

      |
      |
      8.

      再次烧开就放菠菜

      |
      |
      9.

      菠菜变软就可以加鸡蛋了，这时候小火就可以！大火鸡蛋容易老！

      |
      |
      10.

      鸡蛋倒入拿筷子搅拌一下就可以出锅了！

      |
      |
      11.

      是不是很简单，很容易完成！

      |

      安神，防癌，高血压，消食，祛斑，延年益寿，血栓，健脑益智，保肝，动脉硬化，美容护肤，补铁，通便排毒，促发育，增强体质，美容养颜，抗衰老。
