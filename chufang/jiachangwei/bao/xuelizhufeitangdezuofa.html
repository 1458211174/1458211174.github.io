      雪梨猪肺汤的做法

      本汤以滋润肺燥、清化热痰为主。雪梨性味甘微酸凉，有润肺生津、清热化痰的作用，猪肺性味甘平，有补肺的作用，合而为汤，既可清肺之燥热，又可补肺之不足，既可镇咳，又可祛痰，共奏清燥润肺、化痰、止咳之功，是标本同治之汤品。

      一、主料

      主料用量
      猪肺600克
      无花果10颗
      雪梨2个

      二、辅料

      辅料用量
      姜2片
      盐适量

      三、做法

      步骤
      1.

      准备好材料。

      |
      |
      2.

      锅内烧开水放入猪肺，沸腾后再煮2，3分钟，煮出血沫后捞起冲洗干净

      |
      |
      3.

      热锅放油，放入两片姜、猪肺，大火炒一下。

      |
      |
      4.

      雪梨去皮切块。

      |
      |
      5.

      全部材料放入锅内大火煮开后小火煮一个半小时即可，用电锅的话直接按煲汤就行了。

      |
      |
      6.

      关火前放盐调味即可。

      |

      四、功效

      治流鼻血，润肺止咳，健脾，排毒，清热解暑，化痰止咳，润肺化痰，高血压，利尿，养颜护肤。
