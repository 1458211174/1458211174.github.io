      莲子百合草莓粥的做法

      莲子百合粥不仅具有滋补元气，止泻功效，生津液，畅胃气的功效。而且还有清理胃肠、润泽肝腑、平肝散火的效果，所以营养价值还是蛮高的。当然，粥里如果您再加一些营养滋补的水果一起吃，营养和口感就更佳了。今天的粥，我加了一些冻干草莓片，不但丰富了口感，而且草莓漂亮的颜色也为这碗粥提高了不少颜值，看着是不是很有食欲啊！

      一、主料

      主料用量
      大米50克
      冻干草莓片15克

      二、辅料

      辅料用量
      莲子30克
      干百合20克
      冰糖20克

      三、做法

      步骤
      1.

      干百合冷水提前浸泡半小时。

      |
      |
      2.

      大米洗净后放入冰箱冷冻一夜，砂锅水沸后下入大米。

      |
      |
      3.

      下入莲子。

      |
      |
      4.

      用汤勺贴锅底搅拌，防止糊锅，大火煮开后改小火慢煮。

      |
      |
      5.

      粥煮至米粒膨胀后放入百合、冰糖，小火继续煮15分钟。

      |
      |
      6.

      煮至米粒软烂，汤汁粘稠即可关火。

      |
      |
      7.

      冻干草莓片稍微掰碎一些，放入粥里，搅拌均匀即可。

      |
      |
      8.

      尝尝吧！

      |

      四、功效

      适合婴儿，强身健体。
