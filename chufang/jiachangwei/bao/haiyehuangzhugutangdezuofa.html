      海椰皇猪骨汤的做法

      “百草回芽，百病发作”这春季上火，就属于其中一种。所以春季饮食清淡,喝汤也颇讲究。春季煲汤不要太油腻、忌燥热。比如加了海椰皇的猪骨汤，鲜味浓郁，热气带着椰香蒸腾而来，入口细品，椰子的清甜缠绕于唇齿之间，配合无花果、莲子、茯苓等材料合而为汤，可清热去火和滋阴润肺。 海椰皇，亦称复椰子，属于棕榈科，是热带森林的棕榈树的果实，以籽小，味道浓香，汁丰润滑，功效显著而闻名，具有润肺止咳,清燥热,养颜滋阴之功效。

      一、主料

      主料用量
      猪骨400克
      海椰皇50克
      无花果（干）20克
      白茯苓10克
      莲子10克

      二、辅料

      辅料用量
      姜2片
      盐适量

      三、做法

      步骤
      1.

      准备好材料，海椰皇敲开壳后将椰肉对切，其他汤料稍作清洗。

      |
      |
      2.

      猪骨洗净飞水（放入凉水中，大火煮开沸滚1~2分钟，然 后将水倒掉，冲洗干净）；

      |
      |
      3.

      所有材料和姜片一起下瓦煲，加水2500毫升（约10碗量）按自己家庭情况；大火滚沸后改小火煲约1 个半至2小时.

      |
      |
      4.

      出锅前加入食盐即可。

      |

      四、功效

      骨质疏松，润肺，健脾，甲状腺，降压，抵抗力，防癌，安神，益精。
