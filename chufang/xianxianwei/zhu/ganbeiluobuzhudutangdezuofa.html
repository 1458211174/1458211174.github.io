      干贝萝卜猪肚汤的做法

      干贝富含蛋白质、碳水化合物、核黄素和钙、磷、铁等多种营养成分，具有滋阴补肾、和胃调中功能，常食有助于降血压、降胆固醇、软化血管、补益健身。 猪肚中含有大量的钙、钾、钠、镁、铁等元素和维生素A、维生素E、蛋白质、脂肪等成分，能补虚损，健脾胃，治虚劳羸弱，泄泻，下痢，消渴，小便频数，小儿疳积等。 中医认为，白萝卜味辛甘，性凉，入肺胃经，具有清热生津、凉血止血、下气宽中、消食化滞、开胃健脾、顺气化痰的功效，为食疗佳品。 此汤口感清淡鲜美，有滋阴补肾，健脾和胃，清肺理气的功效。

      一、主料

      主料用量
      猪肚500克

      二、辅料

      辅料用量
      干贝10粒
      白萝卜半个
      甜玉米半个
      姜1块
      枸杞5克
      盐2茶匙
      胡椒粉1/2茶匙
      料酒1汤匙
      清水适量

      三、做法

      步骤
      1.

      准备好食材。

      |
      |
      2.

      猪肚洗净切丝。

      |
      |
      3.

      甜玉米剥粒，白萝卜切丝，姜切丝。

      |
      |
      4.

      锅里加入适量清水，姜丝，料酒，烧滚后下入猪肚氽熟，然后捞出备用。

      |
      |
      5.

      重起锅，下入姜丝，干贝，猪肚。

      |
      |
      6.

      大火烧开后，盖上盖子，转中大火炖1个半小时。

      |
      |
      7.

      炖好猪肚加入萝卜丝，玉米粒，枸杞。

      |
      |
      8.

      因为猪肚没什么油，加入一勺猪油，再炖半小时。（不喜欢猪油的可以加植物油）

      |
      |
      9.

      加入适量盐，胡椒粉调味，即可出锅。

      |
