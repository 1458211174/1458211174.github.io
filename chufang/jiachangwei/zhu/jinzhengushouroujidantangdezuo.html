      金针菇瘦肉鸡蛋汤的做法

      快手营养汤，很适合夏天食用。

      一、主料

      主料用量
      金针菇200克
      瘦肉150克
      土鸡蛋两个

      二、辅料

      辅料用量
      茶籽油30克
      盐1克
      生姜1块

      三、做法

      步骤
      1.

      金针菇切掉根部，切成两段，撕开洗净；姜去皮洗净切薄片

      |
      |
      2.

      瘦肉洗净切薄片，放盐搅拌均匀，腌制片刻；土鸡蛋打散

      |
      |
      3.

      锅里倒油，放姜片煸香

      |
      |
      4.

      倒入足够的清水煮开

      |
      |
      5.

      放瘦肉煮开

      |
      |
      6.

      放金针菇煮开

      |
      |
      7.

      倒入鸡蛋液

      |
      |
      8.

      再煮三分钟，起锅装盘即可

      |

      四、功效

      抵抗力，降血脂，过敏，补肾，润肺。
