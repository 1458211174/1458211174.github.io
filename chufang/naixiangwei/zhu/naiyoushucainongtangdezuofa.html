      奶油蔬菜浓汤的做法

      众所周知西兰花是养生界的宠儿，防癌抗癌、增强机体免疫力等功效多到数不胜数。通常我们烹调西兰花的方式为清炒或搭配蒜蓉来炒，今天给大家一个略微偏西式的烹调方式：奶油蔬菜浓汤。蔬菜浓汤搭配吐司面包来做早餐，既开胃又健康，还可以补充各种维生素。家里有小朋友的妈妈都知道，小朋友本能的会对一些蔬菜产生抗拒和排斥而挑食，但用蔬菜和奶制品搭配起来做成浓汤则可以很好的解决这个问题，让宝宝告别挑食，营养吸收更全面。 西兰花的口味很多时候在小朋友眼里不太淘喜，所以这道蔬菜浓汤我又加了土豆和甜玉米粒进来。土豆我选择的是颜色偏黄色的那种，口感更加绵密，加进浓汤里面可以省了家面粉来调整汤的浓稠度。甜玉米粒我觉得是这道菜吸引孩子的一个亮点，可以让浓汤的口感更加的清甜。当然，如果家里的宝宝对坚果不过敏的话也可以适量加一些坚果进来，口感更香浓，营养上更上一个台阶。

      一、主料

      主料用量
      西兰花250克
      土豆1个
      甜玉米100克
      洋葱1/4个
      淡奶油200毫升
      高汤200克
      水400毫升

      二、辅料

      辅料用量
      黄油8克
      盐0.5克
      黑胡椒0.5克

      三、做法

      步骤
      1.

      材料图。

      |
      |
      2.

      洋葱切丁。土豆削皮洗净后同样切丁，用冷水冲洗一下表面的淀粉预防氧化。

      |
      |
      3.

      西兰花掰成小朵用盐水浸泡15分钟，然后用淡盐水焯半分钟后捞出冲凉、控水备用。

      |
      |
      4.

      平底锅烧热后放入黄油，待黄油融化后下洋葱丁煸炒出香味。

      |
      |
      5.

      下土豆丁，调入盐和黑胡椒碎，翻炒至土豆丁变得半透明。

      |
      |
      6.

      调入高汤和水，煮开（没有高汤全部用水也是可以的）后关火。

      |
      |
      7.

      将煮好的高汤土豆等转移到破壁料理机的搅拌杯中，加入焯水后的西兰花、甜玉米粒和淡奶油。接通电源，选择“浓汤”功能，按下启动键。

      |
      |
      8.

      待机器发出“滴滴”的提示音后好喝的奶油蔬菜浓汤就做好了。

      |

      四、功效

      防癌，瘦身减肥，促进生长，软化血管，清热解渴，抗衰老，强身健体，美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，抵抗力，消食，降血压，补钙，延年益寿。
