      淮山排骨汤的做法

      淮山其中功效之一，适合男性：防止“老胃病”复发。在“老胃病”病人中，有相当多属于溃疡病，而秋季是溃疡病容易发生或复发的季节。现代药理研究证实，山药中所含尿囊素有助于胃黏膜的修复。

      一、主料

      主料用量
      淮山药200克
      排骨半斤
      胡萝卜一只
      姜几片

      二、辅料

      辅料用量
      盐适量

      三、做法

      步骤
      1.

      材料准备好。

      |
      |
      2.

      排骨洗净掉血水。

      |
      |
      3.

      姜切片。

      |
      |
      4.

      淮山，胡萝卜滚刀切块。

      |
      |
      5.

      锅中先放入掉水后的排骨和切好的姜片，加入足量的清水。

      |
      |
      6.

      大火烧开。

      |
      |
      7.

      倒入切好的淮山和胡萝卜。

      |
      |
      8.

      大火煮沸，中火煲40分钟。

      |
      |
      9.

      看到汤汁显乳白色，即可加盐。

      |
      |
      10.

      关火。

      |
      |
      11.

      出品。

      |

      四、功效

      补肾，润肺，抵抗力，降糖，保护视力，助消化，预防感冒，散瘀止疼，强身健体，治肠炎，开胃，预防中暑，缓解腹痛，治吐泻。
