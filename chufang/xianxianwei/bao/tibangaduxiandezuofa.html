      蹄膀腌笃鲜的做法

      浙江人喜欢吃笋，最近菜场看到了春笋，买回来尝个时令新鲜。去年做腌笃鲜笋有涩味，今年特意用开水放了一点白酒焯了，中午不涩了。单独的蹄膀汤略显单调和油腻，放咸肉和笋做成腌笃鲜不仅不单调鲜美度也提升了不少。

      一、主料

      主料用量
      猪蹄膀2斤
      咸肉150克
      春笋5根

      二、辅料

      辅料用量
      青菜2棵
      白酒2毫升
      色拉油8毫升
      菜籽油5毫升
      青葱1把
      生姜1块
      青蒜1根

      三、做法

      步骤
      1.

      将春笋剥皮，并将每棵均匀竖切成四小条。

      |
      |
      2.

      放开水里焯熟，放两毫升白酒。据说可以去除笋的涩味，实验下来果真是可以去涩的

      |
      |
      3.

      焯好笋后，接着将剁好的蹄膀放入刚才的开水中焯一下。

      |
      |
      4.

      准备生姜葱和青蒜一根

      |
      |
      5.

      倒入色拉油8毫升，菜籽油5毫升。小贴士：菜籽油可以使汤浓白，但是全都放菜籽油汤会黄黄的，并且菜籽味很重，所以拿捏好量。油开后将葱姜蒜放入锅中煸出香味。

      |
      |
      6.

      葱姜蒜煸炒出香味后放入焯过的蹄膀，继续大火翻炒半分钟。

      |
      |
      7.

      倒入一点水，继续翻炒五分钟。

      |
      |
      8.

      锅中倒满水，烧到沸腾。

      |
      |
      9.

      将锅中的汤和肉全部倒入电饭锅中，按煲汤功能煲两小时。

      |
      |
      10.

      将鲜肉放进冰箱冻成半硬状，大概冻两小时，冻完拿出来切薄片。冻硬后比较好切薄片。

      |
      |
      11.

      两小时结束后把咸肉和笋一起倒入锅中煮二十分钟到半小时

      |
      |
      12.

      快盛的时候放一两颗青菜入锅烫熟，青菜也可以不放，为了拍照色彩好看点我还是放了一颗，嘻嘻

      |
      |
      13.

      大功告成

      |

      四、功效

      助消化，化痰止咳，补血，保护视力，健脾，养颜护肤。
