      灵芝药材骨头汤的做法

      灵芝具有补气安神，保肝解毒、抗肿瘤、抗衰老的功效，用于眩晕不眠，心悸气短，虚劳咳喘。

      一、主料

      主料用量
      猪大骨头250克
      灵芝10克
      黄芪10克
      党参10克
      枸杞10克
      红枣10克
      当归10克
      红枣5颗

      二、辅料

      辅料用量
      盐1勺

      三、做法

      步骤
      1.

      灵芝药材配好。

      |
      |
      2.

      骨头洗净。

      |
      |
      3.

      准备好炖锅。

      |
      |
      4.

      将所有食材放入炖锅内。

      |
      |
      5.

      加入3碗清水。

      |
      |
      6.

      外锅加入最大水量。

      |
      |
      7.

      隔水炖3个小时。

      |
      |
      8.

      还剩下几分钟。

      |
      |
      9.

      开锅下盐。

      |
      |
      10.

      到时间自己保湿。

      |
      |
      11.

      出锅，得凉。

      |
      |
      12.

      出品。

      |

      四、功效

      骨质疏松，安神助眠，散瘀止疼，心血管病，降压，降血脂，糖尿病，抵抗力，贫血，通便，抗休克，强心，神经衰弱，保护视力，保肝，降三高，治流鼻血，血栓，补血。
