      猪肝粉肠豆苗汤的做法

      豆苗是我很喜欢吃的一种蔬菜，而猪肝粉肠是我家那位十分喜欢吃的。所以一个菜里，放入三种食材，满足两个人的喜好，味道也一点都不冲突。让重味，也能带点小清新的感觉哈

      一、主料

      主料用量
      猪肝100克
      粉肠100克
      豆苗200克

      二、辅料

      辅料用量
      油20克
      酱油5克
      生粉5克
      盐2克
      姜片3克

      三、做法

      步骤
      1.

      处理好的粉肠猪肝，放入姜片，酱油，油，盐，生粉腌一下

      |
      |
      2.

      豆苗洗好备用

      |
      |
      3.

      锅中放入适量水，烧开

      |
      |
      4.

      烧开后倒入猪肝粉肠

      |
      |
      5.

      大火重新烧开，并在开时去浮沫

      |
      |
      6.

      猪肝粉肠烧开两三分钟后，倒入豆苗。重生煮开

      |
      |
      7.

      水开一分钟，豆苗熟透，加入盐，油调味即可出锅

      |
      |
      8.

      清新上桌

      |

      四、功效

      保护视力，补血，排毒，抵抗力，利尿，缓解腹泻，健胃，利尿消肿。
