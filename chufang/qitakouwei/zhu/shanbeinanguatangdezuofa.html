      扇贝南瓜汤的做法

      浓厚的南瓜汤配上鲜嫩的扇贝，有多好吃我就不说了，大家懂。

      一、主料

      主料用量
      扇贝4只
      南瓜250克
      松子25克
      橄榄油50毫升
      淡奶油50毫升
      洋葱1/4个

      二、辅料

      辅料用量
      黄油50克
      净水500毫升
      海盐少许
      黑胡椒少许

      三、做法

      步骤
      1.

      食材图

      |
      |
      2.

      加热平底锅，放入黄油，炒香松子直到变成金黄色，放置一边备用；

      |
      |
      3.

      另取一锅内倒入橄榄油，加入南瓜和洋葱，炒至半熟；

      |
      |
      4.

      倒入水煮到南瓜变软，加入盐和胡椒调味，煮到南瓜酥烂后加入淡奶油，等汤烧开后倒入料理机打碎；

      |
      |
      5.

      打开扇贝，用盐和胡椒腌制，放入平底锅，用黄油煎至金黄后装盘即可。

      |

      四、功效

      健脑，健脾，养颜护肤，防癌，治胃病，降糖，治前列腺，抗衰老，健脾养胃，健脑益智，滋阴润肺，延缓衰老，软化血管，美容养颜，抵抗力，消食，降血压，补钙，延年益寿。
