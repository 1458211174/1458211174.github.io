      桂花金桔炖雪梨的做法

      秋冬季节天气干燥，很容易上火引起感冒咳嗽等症状，所以平时在生活饮食上需要多加注意。 天冷了小朋友喝水的频率越来越低，经常是每天早上装一杯到带到学校，下午放学回来时原封不动给背回来，回家后不停提醒喝水，也经常是不端杯子喝，经常回一句，我现在不渴，不想喝水，等会再渴吧，等一会后水杯中的水照样原封不动放在那边，因而每隔几天就会准备点清热去火的食物给，做到防患于未然。 今天给大家分享的这款美食非常适合这个季节给小朋友们炖着吃。

      一、主料

      主料用量
      雪梨1个
      金桔5个

      二、辅料

      辅料用量
      糖桂花1勺
      冰糖5粒

      三、做法

      步骤
      1.

      准备好食材

      |
      |
      2.

      用刷子清洗干净梨表面后用刀在顶部横切一刀，用勺子掏空梨的核

      |
      |
      3.

      冰糖放入去核的梨中

      |
      |
      4.

      放入金桔

      |
      |
      5.

      盖上梨的顶端，用牙签固定梨的顶端和底部

      |
      |
      6.

      将固定好的梨放入炖芯中

      |
      |
      7.

      往水箱中倒入适量水，盖上盖子，接通电源

      |
      |
      8.

      按功能键选择“隔水炖”，当电炖盅指示灯跳至保温状态时，关掉电源，取出炖好的金桔雪梨，淋上糖桂花
