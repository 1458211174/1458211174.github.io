      棒骨山药玉米汤的做法

      虽然生活在东北，但是家人也有每天喝汤的习惯，浓汤也好，清汤也罢，只要有个汤就行。东北的夏季，转眼即瞬，立秋时节带来阵阵凉意，这时煮点养胃健脾的汤喝喝，既能生津止渴、又能补养身体。玉米不仅含有丰富的植物纤维素，可以抑制脂肪吸收，降低血脂水平，预防和改善冠心病、肥胖、胆结石症的发生。山药性平，能润肺、健脾、益肾，它含有淀粉酶、多酚氧化酶等物质，有利于脾胃消化吸收功能，是平补脾胃的药食两用之品。

      一、主料

      主料用量
      棒骨半个
      玉米1根
      铁棍山药1根

      二、辅料

      辅料用量
      葱50克
      姜20克
      玉竹10克
      茶树菇10克
      香菇10克
      姬松茸10克
      黄芪15克
      莲子5克
      枸杞5克
      清水1500克
      盐适量

      三、做法

      步骤
      1.

      准备所需食材，并清洗干净；

      |
      |
      2.

      棒骨洗净，锅里放冷水，放入筒骨，开大火，水焯；

      |
      |
      3.

      焯水后的棒骨冲洗干净；

      |
      |
      4.

      砂锅中加入清水，将棒骨、玉米及葱、姜、玉竹、茶树菇、香菇、姬松茸、黄芪、莲子放入锅中；

      |
      |
      5.

      大火煮开；

      |
      |
      6.

      调中小火，保持微沸状态，盖砂锅盖，煲煮70-80分钟；

      |
      |
      7.

      加入山药，继续煲煮15分钟；

      |
      |
      8.

      加入枸杞和盐，关火；

      |
      |
      9.

      盛出即可食用。

      |

      四、功效

      减肥瘦身，防癌，降血压，保护视力，润肠通便，健脑，抗衰老。
