      杜仲黑豆猪尾汤的做法

      冬天为自己和家人炖上一锅，营养丰富，不要看这汤黑乎乎的，可都是滋阴补肾的佳品。经常喝这汤，男人补肾健腰，女人滋阴益髓！！ 用猪尾炖汤比猪蹄好吃多了，而且也都是满满的胶原蛋白，汤肉吃起来也不油腻。配上黑豆，补肾健腰一步到位，想要效果更佳的话，也可以加点杜仲！

      一、主料

      主料用量
      杜仲10克
      猪尾巴1条
      黑豆70克
      红枣6颗

      二、辅料

      辅料用量
      姜3片
      盐少许

      三、做法

      步骤
      1.

      准备所有好材料，猪尾巴去毛斩成小段。

      |
      |
      2.

      材料清洗干净，黑豆稍微浸泡一会。

      |
      |
      3.

      红枣去核，主要是为了没那么上火。

      |
      |
      4.

      猪尾巴凉水下锅烧开1分钟，猪尾巴后去掉浮沫血水洗干净备用。

      |
      |
      5.

      所有准备好的食材与姜片一起放进去，（按个人习惯可放少许料酒）大火烧开后转小火慢炖1个半小时。

      |
      |
      6.

      出锅前调入食盐即可。

      |

      四、功效

      降压，骨质疏松，慢性病，流产，延年益寿，补钙，排毒，健脑，养发，软化血管，治流鼻血。
