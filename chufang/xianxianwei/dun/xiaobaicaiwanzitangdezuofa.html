      小白菜丸子汤的做法

      羊肉小白菜汤很好喝，但是一般很少会吃里面的肉，自从把羊肉做成了丸子就不一样了，一口汤一口丸子，真是美呆了。

      一、主料

      主料用量
      小白菜1把
      羊肉200克

      二、辅料

      辅料用量
      盐适量
      花椒粉适量
      姜一块
      大葱一段
      生抽一勺
      料酒一勺
      淀粉一大勺

      三、做法

      步骤
      1.

      羊肉一块，要这种有肥有瘦的，味道和口感更好

      |
      |
      2.

      先简单的剁一下肉

      |
      |
      3.

      放入姜，料酒，生抽调味

      |
      |
      4.

      剁均匀以后，放盐，花椒粉调味，继续剁均匀，这样边加调料边剁肉，肉更容易入味

      |
      |
      5.

      加入淀粉继续剁，剁到看不见浮粉

      |
      |
      6.

      此时的肉馅很有劲儿，放置片刻，让味道充分融合

      |
      |
      7.

      锅里放水，冒小泡的时候下丸子，丸子从虎口挤出来放到锅里

      |
      |
      8.

      撇去浮沫

      |
      |
      9.

      继续煮

      |
      |
      10.

      放葱姜和盐调味

      |
      |
      11.

      小白菜洗净

      |
      |
      12.

      下入小白菜

      |
      |
      13.

      大火烧开即可

      |
      |
      14.

      很鲜呢

      |

      四、功效

      软化血管，防癌，润肺化痰，强身健体，抵抗力，美容养颜，延年益寿，健胃，温补肝肾，温经止痛，保护视力。
