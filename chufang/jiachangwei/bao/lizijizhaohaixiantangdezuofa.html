      栗子鸡爪海鲜汤的做法

      我们广东人都喜欢喝汤了。鸡爪、猪展肉、响螺片、栗子、大蜜枣、云苓土茯苓、淮山、玉竹、莲子，祛湿健脾。大火煮沸腾后慢火煮1.5个小时即可，汤味浓郁。

      一、主料

      主料用量
      猪展肉200克
      响螺干片50克
      鸡爪6只
      去皮栗子15粒

      二、辅料

      辅料用量
      蜜枣2个
      淮山10克
      云苓10克
      玉竹10克
      土茯苓10克
      姜片3片
      莲子10克
      盐2小勺

      三、做法

      步骤
      1.

      汤料洗干净，响螺片提前浸泡1个小时。

      |
      |
      2.

      将除肉类外的所有汤料全部放入玻璃锅内，加入清水，大火煮沸。

      |
      |
      3.

      烧一锅水，放入肉类，焯煮。

      |
      |
      4.

      焯好的肉类，冲洗干净血沫。

      |
      |
      5.

      把肉全部放入玻璃锅内，一同煮。

      |
      |
      6.

      先用大火，煮至沸腾。

      |
      |
      7.

      然后转为小火，慢慢煲约1.5个小时。

      |
      |
      8.

      熄火前，加入2勺子食盐调味。

      |
      |
      9.

      这老火汤熬好了，盛一碗尝尝，汤汁浓郁甜蜜。

      |

      四、功效

      高血压，养颜护肤。
