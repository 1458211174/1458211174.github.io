      鸡肉碎米香粥的做法

      这道粥，非常的好喝，清爽不油腻，特别能激起食欲，并且做法简单。食材里用到的大米我选的是碎米，爱烂还非常香，再配以鸡肉和香菇，营养又味美。香菇提高免疫力、防癌抗癌、延缓衰老；鸡肉强身健体、促进智力发育

      一、主料

      主料用量
      五常碎米70克
      鸡胸肉248克

      二、辅料

      辅料用量
      香菇1小朵
      盐2茶匙
      葱花少许
      香油2滴
      清水3小碗

      三、做法

      步骤
      1.

      备好食材，香菇洗净

      |
      |
      2.

      碎米洗净入电饭锅内胆，加入3小碗清水

      |
      |
      3.

      电饭锅设置煮粥功能，米熟了还不黏稠的时候放入切好的香菇丁和鸡肉丁，加入葱花，搅匀，合上盖子继续熬制

      |
      |
      4.

      粥粘稠就熬好了，加入盐、香油搅匀即可
