      海带骨头汤的做法

      清热去火

      一、主料

      主料用量
      海带适量
      猪扇骨适量

      二、辅料

      辅料用量
      姜1块
      牛丸10颗
      料酒1勺

      三、做法

      步骤
      1.

      将扇骨焯水，焯水捞出用清水冲洗一下渣沫，控干水分后入锅，放入姜片、料酒加水大火煮开

      |
      |
      2.

      将海带清洗干净，泡发后撕成小段并打结，备用

      |
      |
      3.

      牛肉丸装盘备用

      |
      |
      4.

      骨头大火煮开后转小火炖30分钟，后加入海带牛肉丸食盐，继续炖30到50分钟，最后起锅

      |

      四、功效

      利尿，防病，御寒，防癌，防甲亢，抗辐射，骨质疏松。
