      猪脚黄豆汤的做法

      猪脚黄豆汤，一口砂锅，奉献一点时间，一道美容养颜的好汤，营养价值高，胶原蛋白多，绝对是女士滋补佳品，过程操作简单，在家轻松就能搞定，偷偷告诉你们，清水煮面条后捞在汤碗里，再浇上一些猪脚黄豆汤，撒上蒜苗末，香味扑鼻而来……还在等什么，快快去尝试吧！期待你们的成品图哦

      一、主料

      主料用量
      猪脚1只
      黄豆100克

      二、辅料

      辅料用量
      食盐1.5勺
      生姜适量
      料酒1勺
      香葱2颗
      鲜红朝天椒1个
      干花椒适量

      三、做法

      步骤
      1.

      买回来的主角清洗干净，沥水

      |
      |
      2.

      黄豆提前一晚用水泡好，泡之前需要将黄豆清洗干净

      |
      |
      3.

      生姜片，干花椒，鲜红朝天椒，料酒，香葱，没有鲜红朝天椒用干辣椒替代也是可以的。

      |
      |
      4.

      取锅放入猪脚，加水盖过猪脚，放入生姜片，干花椒，鲜红朝天椒，料酒，香葱，大火煮开后转小火，再煮10分钟关火，取出猪脚。(没有鲜红朝天椒用干辣椒替代也是可以的)

      |
      |
      5.

      将飞水后的猪脚清洗干净沥水放置

      |
      |
      6.

      取5颗大蒜，3片生姜

      |
      |
      7.

      取砂锅放入猪脚，黄豆，生姜，大蒜，倒入3勺黄酒，开大火煮开转小火煲90分钟，加入适量盐调味即可

      |
      |
      8.

      出锅前放入少量葱花，喜欢吃蒜苗的朋友们可以放点蒜苗末，味道会更赞

      |

      四、功效

      美容养颜，抗衰老，促进生长，防冠心病，软化血管，益智，糖尿病，降脂，美白，防癌，脂肪肝，更年期，抵抗力。
