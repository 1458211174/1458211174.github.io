      瑞典小丸子

      一直以为只有中国人才吃肉丸子，没有想到瑞典人也会用猪肉做小丸子啊！

      一、材料

      【食材】西红柿 1个，苹果 半个 // 山药 半根 // 洋葱 半个 // 瘦肉 一块 // 盐、黑胡椒 // 黄油//面包糠

      二、做法

      步骤
      1、首先熬酱和蒸山药；
      2、下黄油炒香洋葱碎，加入西红柿丁翻炒，再加入苹果丁继续翻炒；
      3、加入盐、糖和清水煮至西红柿苹果软烂粘稠，就算是熬好了；
      4、熬煮酱汁时，可以剁肉糜；
      5、将蒸熟的山药压成泥和黄油炒香的洋葱碎末，还有面包糠一起加入肉糜，加盐和黑胡椒拌匀；
      6、将肉糜捏成小丸子，放在锡纸上，如上图，锡纸不会粘粘；
      7、下油锅小火将小丸子炸成焦黄；
      8、最后浇上苹果番茄酱
