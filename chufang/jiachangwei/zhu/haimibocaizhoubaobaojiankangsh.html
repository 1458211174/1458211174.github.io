      海米菠菜粥 宝宝健康食谱的做法

      你好亲爱的，我是食谱君，给宝宝制作的近400道辅食，都收录在了我的公众号《宝宝健康食谱》里，关注即可查看噢！包括主食、汤羹、菜品、烘焙、点心、食疗等分类，再也不用为宝宝吃啥而发愁啦！ 对于已经吃辅食的宝宝们来说，辅食粥是个相当不错的选择。 不过，要做出味道好又营养的粥来，要学会对搭配不同的食材，种类多样化、经常翻新花样，才能增加宝宝对于食物的兴趣，提高食欲。 今天这碗菠菜鸡肉海米粥，大小宝宝都适合，口感柔嫩细腻、鲜香软糯，有海米天然的咸鲜调味，就算不加盐，宝宝也很爱。 海米营养丰富，优质蛋白是普通鱼、蛋、奶的几十倍，而且肉质松软，宝宝很容易消化，加上鸡肉和菠菜，营养密度变得更高。冬季给宝宝来上一碗，非常温润滋养！

      一、主料

      主料用量
      大米30克
      鸡胸肉20克
      菠菜2颗
      海米3克

      二、辅料

      辅料用量
      油适量

      三、做法

      步骤
      1.

      参考月龄：10个月以上
      食材准备： 鸡胸肉20克，菠菜1颗，大米30克，海米3克
      烹饪时间：30分钟
      推荐指数：★★★★★

      |
      |
      2.

      大米冷水浸泡30分钟，提前准备好哈！

      |
      |
      3.

      海米冲洗干净，然后用冷水泡洗。
      Tips:
      泡洗的时候需要换3-4次水，这样吃起来口感就不会太咸。海米是海虾经过晾晒等工序做成的，营养非常赞，用来做粥也不需要额外加盐噢！

      |
      |
      4.

      泡洗好的海米体积会略微膨胀，味道有点咸味就可以啦，剁碎成末。
      Tips:
      这里要根据自己宝宝的咀嚼能力来做，可以剁成细碎的末末，也可以像我这样切成小丁。如果你不确定宝宝是否能咀嚼碎化，那就切丁放粥里，盛出来的时候要虾米挑出去，虾米的味道已经融入到粥里啦，依然会有鲜味哦！

      |
      |
      5.

      菠菜清洗干净焯水。
      Tips:
      焯水时间不要太久，一般是沸水入锅，再次沸水就可以捞出来啦！

      |
      |
      6.

      焯好水以后的菠菜切末。

      |
      |
      7.

      鸡胸肉放在砧板上，左手拉住肉的一侧，右手用刀刮肉泥。这样做出来的肉泥比用刀切的嫩。

      处理好的肉泥，加少许的水。

      |
      |
      8.

      再滴2-3滴食用油。
      搅拌均匀。

      |
      |
      9.

      锅里加入足量的清水，然后放入大米和海米一块焖煮。

      |
      |
      10.

      至大米粘稠感增加的时候放入鸡肉泥，用筷子迅速搅拌均匀，防止成团。

      |
      |
      11.

      米粥煮好以后放入菠菜搅拌均匀，然后关火焖1分钟。
      Tips:
      菠菜提前焯水的时候已经断生啦，这里用余温再次加热就可以熟透了，如果继续烧煮会导致菜叶变黄。

      |
      |
      12.

      焖好以后就可以盛给宝宝咯！

      |
