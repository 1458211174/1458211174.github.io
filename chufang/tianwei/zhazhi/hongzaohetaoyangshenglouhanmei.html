      红枣核桃养生露----汉美驰破壁机的做法

      冬天的清晨或下午来一杯热腾腾的饮品,胃暖暖的,又可以给身体补充营养能量,生活的小确幸! 汉美驰多功能破壁机的热汤功能,是这个冬日我用得最多的程序,用它制作好的成品可直接饮用,无需要加热!周末在家制作的这款核桃红枣养生饮,营养又美味,口感顺滑细腻

      一、主料

      主料用量
      红枣4个
      核桃适量
      枸杞适量

      二、辅料

      辅料用量
      糯米50克
      纯净水800毫升

      三、做法

      步骤
      1.

      准备好食材,备用。

      |
      |
      2.

      红枣去核.糯米用水泡2小时左右。

      |
      |
      3.

      将食材倒入汉美驰破壁机的搅拌杯内,倒入适量的纯净水,盖上盖子。

      |
      |
      4.

      开启开关键,选”热汤”功能。汉美驰破壁机主要有四种程序,热汤,果昔,纯果汁,刨冰。”热汤”功能,整个程序从低温(或室温)开始搅拌,并将浓汤加热至接近沸点的。

      |
      |
      5.

      热汤功能运行5分45分,透明的杯体可随时随地观察到食物的状态！搅拌杯桶用的是婴儿奶瓶级Tritan材质，帮而可耐高温、耐腐蚀，而且易清洗，不会残留异味，还能在高速工作中保证稳定性。
