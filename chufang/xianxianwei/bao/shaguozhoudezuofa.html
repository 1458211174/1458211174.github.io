      砂锅粥的做法

      砂锅粥是广东的名菜，营养不上火。

      一、主料

      主料用量
      鸡半只
      虾一斤
      米一杯

      二、辅料

      辅料用量
      香菜3棵
      油一小勺
      盐一小勺
      酱油一小勺

      三、做法

      步骤
      1.

      先煲粥，多少人吃看情况放米，洗干净后放去煲，粥容易粘底，煲的时候要注意拌动！

      |
      |
      2.

      选用甜脆金黄色的萝卜干，把萝卜干剁碎，直接放进粥里一起煲。

      |
      |
      3.

      虾洗干净，把头切掉，从中间划开，把黑色线挑走，放油盐酱油捞一下倒进粥里。(嫌麻烦的黑线也可以不管他，虾也可以切开两边，自己喜欢。)

      |
      |
      4.

      把鸡洗干净，切小，放油盐酱油一起捞一下放进粥里一起煲。

      |
      |
      5.

      香菜等粥煲好，放盐调好味后再放香菜，一翻转就可以出锅了。

      |

      四、功效

      下奶，安神助眠，动脉硬化，抵抗力，助睡眠。
