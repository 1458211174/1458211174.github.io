      山药养生粥的做法

      煮粥我喜欢放点小米，小米和西兰花绝配，营养丰富。山药养胃，加点胡萝卜，虾仁和袖珍菇，营养又美味，让营养更全面一些。

      一、主料

      主料用量
      大米半碗
      小米适量
      山药半根

      二、辅料

      辅料用量
      西兰花1小个
      胡萝卜1小个
      生姜1小块
      虾仁20只
      袖珍菇60克
      食用油适量
      料酒1/3勺
      盐适量

      三、做法

      步骤
      1.

      大米，小米淘洗干净放锅里。

      |
      |
      2.

      加适量清水，加加油，盖上锅盖，煮熟。

      |
      |
      3.

      袖珍菇洗干净对半切片，大个的可以切三等分；西兰花洗干净，掰小朵；胡萝卜洗干净，切小粒；生姜洗干净切片，虾仁洗干净；分别装在碗里，备用。

      |
      |
      4.

      山药去皮洗干净，切片，装在碗里备用。

      |
      |
      5.

      锅里烧水，水开后，袖珍菇倒入锅里焯30秒，捞出备用。

      |
      |
      6.

      锅里加油加热，姜片和虾仁倒入锅里煸炒出香味。

      |
      |
      7.

      胡萝卜倒入锅里煸炒几下，至油变成浅橙色。

      |
      |
      8.

      山药片倒入锅里煸炒几下。

      |
      |
      9.

      料酒倒入锅里，加盐调味，翻炒，盛碗备用。

      |
      |
      10.

      将盛在碗里的山药胡萝卜倒入煮粥的电饭锅里，翻炒均匀。盖锅盖继续煮。

      |
      |
      11.

      西兰花倒入锅里煸炒。煸炒几下加盐调味，翻炒均匀，盛出备用。

      |
      |
      12.

      粥煮至浓稠时，西兰花和袖珍菇一起倒入锅里，拌匀。加盐调味，可以加点香油。盖上锅盖，继续煮5分钟关火。

      |
      |
      13.

      盛碗。

      |

      四、功效

      适合婴儿，强身健体，开胃消食，抗感染，预防流产，维持性功能，祛斑美容，促进生长，助胎儿发育，滋肾益精，降血糖，延年益寿，助消化。
