      白薯粥之懒人版的做法

      初春，甩肉的季节，白薯是粗纤维的减肥佳品，晚上喝一碗暖暖的白薯粥，既有饱腹感又可以清肠，制作简单，厨房小白们快动手吧！

      一、主料

      主料用量
      米饭一碗
      白薯1个

      二、辅料

      辅料用量
      冰糖适量

      三、做法

      步骤
      1.

      白薯洗净削皮。

      |
      |
      2.

      白薯切成滚刀块儿。如果会用刀劈成不规则的形状更好比切出来的口感好。米饭就可以用吃剩的米饭，如果没有可以直接用生米。白薯和米饭的比例是2:1。这样白薯粥里的白薯会很多，很香。

      |
      |
      3.

      加凉水，如果喜欢吃稠一点的就少放水。我用的是电压力锅选择煮粥模式。放的水沒过了白薯，最后煮出来的汤还会有一些，不稠不稀。上气儿后三十分钟做好。

      |
      |
      4.

      开盖后，米饭和白薯都已经软烂了。可以适当的把白薯搅碎些，和粥充分的融合。

      |
      |
      5.

      白薯粥制作完成！很简单吧

      |

      四、功效

      抵抗力，健脾，保护视力，健脑安神，缓解腹泻，防癌，通便，抗衰老。
