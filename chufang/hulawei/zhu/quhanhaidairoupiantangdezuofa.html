      驱寒海带肉片汤的做法

      比起鲜味的汤，这个做法是不一样的味道哦

      一、主料

      主料用量
      肉200克
      干海带丝60克

      二、辅料

      辅料用量
      姜适量
      干辣椒4颗
      花椒适量
      蒜三瓣
      葱一根
      盐适量
      油几滴

      三、做法

      步骤
      1.

      处理材料：肉切片或者丝；姜切丝；如果用的干海带，海带提前用温水泡2-3小时泡发，泡发后切成方便咀嚼的小段；蒜切末葱切段

      |
      |
      2.

      炒：锅内烧油，小火，入姜丝炒香后下肉片

      |
      |
      3.

      炒：肉片半熟后加入干辣椒花椒，翻炒熟后倒入盛了清水的汤锅里

      |
      |
      4.

      煮：倒入肉片

      |
      |
      5.

      煮：倒入海带丝，中小火煮25分钟左右（海带煮软入味）

      |
      |
      6.

      捞：汤锅水开后可能会有些浮泡，捞掉后继续盖锅煮10分钟

      |
      |
      7.

      添：快煮熟时加适量的盐，再盖锅煮一会儿

      |
      |
      8.

      添：最后倒入蒜末开盖煮五分钟

      |
      |
      9.

      洒：滴几滴油，洒葱花起锅

      |
