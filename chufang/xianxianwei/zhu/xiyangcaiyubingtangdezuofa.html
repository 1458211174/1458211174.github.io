      西洋菜鱼饼汤的做法

      鱼饼是我们家的特产，用鱼肉剁碎加面粉油炸而成。可以炸好就吃，也可以用来炒菜，我喜欢用来做汤，吃着一点都不油腻，口感也爽滑

      一、主料

      主料用量
      鱼饼100克
      西洋菜500克

      二、辅料

      辅料用量
      盐2克

      三、做法

      步骤
      1.

      食材准备

      |
      |
      2.

      西洋菜洗好

      |
      |
      3.

      锅中放入适量水，烧开

      |
      |
      4.

      然后放入鱼饼

      |
      |
      5.

      倒入西洋菜，大火烧开

      |
      |
      6.

      烧开至西洋菜全部变软，然后加入盐调味即可

      |
      |
      7.

      丰盛上桌

      |

      四、功效

      化痰止咳，利尿。
