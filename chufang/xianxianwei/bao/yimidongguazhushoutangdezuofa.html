      薏米冬瓜猪手汤的做法

      这个天气比较潮湿，熬点汤水滋润一下，冬瓜是减肥的好东西，加点薏米，眉豆去一下湿气，对于春季还不适合用太凉的冬瓜熬汤，记得加多点姜，和放几个红枣，中和一下，领这道汤不会太凉。　冬瓜味甘、淡、性凉，入肺、大肠、小肠、膀胱经；具有润肺生津，化痰止渴，利尿消肿，清热祛暑，解毒排脓的功效；可用于暑热口渴、痰热咳喘、水肿、脚气、胀满、消渴、痤疮、面斑、脱肛、痔疮等，还能解鱼、酒毒。冬瓜皮以利尿见长；冬瓜子以健脾养颜、止咳化痰见长。

      一、主料

      主料用量
      薏米30克
      花生50克
      冬瓜500克
      红枣两个
      猪手500克
      眉豆30克

      二、辅料

      辅料用量
      姜50克
      盐两勺
      油一勺
      鸡粉少许

      三、做法

      步骤
      1.

      准备好食材，冬瓜，薏米，花生，眉豆，猪手，红枣姜

      |
      |
      2.

      食材豆放入锅里用少许油盐爆一下

      |
      |
      3.

      下入适量的是慢火熬煮

      |
      |
      4.

      熬上40分钟后放入去皮的冬瓜

      |
      |
      5.

      熬煮到冬瓜软烂下少许味精即可，如果不够味再下少许盐调味

      |
      |
      6.

      已经做好的成品

      |

      四、功效

      防癌，美白美容，抵抗力，糖尿病，解热镇痛，治脚气病，清热去火，助消化，强身健体，抗衰老，润肺，止血，心血管病，补虚，保护心脏，促进发育，益智，减肥瘦身，滋养肾气，化痰，润肤美容，降。
