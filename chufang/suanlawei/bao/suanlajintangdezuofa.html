      酸辣金汤的做法

      这道菜可以放肥牛、肥羊，酸辣开胃，每次都吃的撑到饱，第一次做的时候我家先生很给面子的吃个精光！

      一、主料

      主料用量
      午餐肉200克
      千张2张
      绿豆芽100克

      二、辅料

      辅料用量
      植物油适量
      味精适量
      清水适量
      葱一颗
      蒜2个
      青辣椒1个
      泡椒1个
      海南黄灯笼辣椒酱2茶匙
      海米一小把
      白醋4茶匙
      食盐适量

      三、做法

      步骤
      1.

      葱、蒜、青辣椒、泡椒切好备用。

      |
      |
      2.

      碗底菜我用的绿豆芽、千张，焯水备用。

      |
      |
      3.

      锅内入油

      |
      |
      4.

      将葱花蒜末青辣椒末泡椒末放入，爆香煸炒

      |
      |
      5.

      加入海南黄灯笼辣椒酱，煸炒出黄油。

      |
      |
      6.

      加入清水，加入白醋，自己根据自己的情况加盐

      |
      |
      7.

      水开后加入海米、午餐肉片，然后倒入事先铺好碗底的汤碗内，撒上香菜末，即可。

      |

      四、功效

      心血管疾，清热解暑，舒缓神经，防溃疡，通便排毒，去胆固醇。
