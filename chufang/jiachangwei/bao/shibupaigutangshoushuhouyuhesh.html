      食补排骨汤（手术后愈合伤口）的做法

      家里有老人刚刚做完手术，所以根据协和医生的叮嘱把需要的食材适合煲汤的全部都放在一起，可以说是术后最好的爱心补汤了。

      一、主料

      主料用量
      排骨1000克

      二、辅料

      辅料用量
      小香葱10克
      姜片10克
      香菜10克
      鲜芦笋50克
      魔芋100克
      山药100克
      海带75克
      香菇75克
      鹌鹑蛋75克

      三、做法

      步骤
      1.

      买的排骨防冷水5分钟散散血水。

      |
      |
      2.

      香菜小葱最后再放，先备好

      |
      |
      3.

      鹌鹑蛋肯定是先煮好了剥开

      |
      |
      4.

      山药去皮，切块，泡水

      |
      |
      5.

      海带一定要提前拿热水过一下，要不会串味，而且会不好掌握汤的咸度，因为海带自带盐分

      |
      |
      6.

      海带抄好后切片

      |
      |
      7.

      香菇泡发开后一部分切片，一部分整块。

      |
      |
      8.

      鲜芦笋一定要最后出锅前两分钟放就可以了

      |
      |
      9.

      魔芋很好洗，但要切成片后在水里先泡一会，要不会汤太稠

      |
      |
      10.

      这样切很好，既容易熟，又好入味

      |
      |
      11.

      凉水下锅把排骨煮熟，一定要凉水啊！这样煲汤的排骨很烂，现在我已经开始煲汤了，最好稍微点一点白醋，这样可以把骨头的营养释放出来，香菇这时候就可以放了，开以后再把所有配料按照海带，魔芋，山药，鹌鹑蛋，芦笋，香菜香葱的顺序入锅就可以了。

      |

      四、功效

      补肾，润肺。
