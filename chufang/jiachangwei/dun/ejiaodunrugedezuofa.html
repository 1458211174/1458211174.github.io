      阿胶炖乳鸽的做法

      老话说的好：大雪补得当，一年不受寒。说的当然不说“大雪”这一天了。而是大雪节气这段时间。大雪节气一过，这时大部分地区的最低温度都降到了0℃或以下。天气寒冷，正是人体收藏精气之时，这时候进行温补最容易使人体吸收。大雪节气养生应重温补，养生要防燥。 食补正当时，今天，我也买了一只鸽子，准备做一道温补的阿胶炖乳鸽。阿胶，是传统的滋补、补血上品，是以驴皮为主要原料，放阿井之水而制成，带琥珀色，透明，无臭味。亦称驴皮胶。特别适合女性，因它补血滋阴，润燥，止血。用于血虚萎黄，眩晕心悸，心烦不眠，肺燥咳嗽。阿胶温补，不易上火。但还是建议服用阿胶的时候适当的多吃点蔬菜水果。 都说一鸽胜九鸡，以阿胶炖乳鸽，绝对食补圣品。能补血补气，能温补肾阳。男女适用

      一、主料

      主料用量
      乳鸽1只
      东阿阿胶1片

      二、辅料

      辅料用量
      红枣6颗
      姜三片
      龙眼干肉10克
      盐适量

      三、做法

      步骤
      1.

      准备食材

      |
      |
      2.

      洗理干净的乳鸽洗净冷水下锅，放几片姜，水开后焯烫2-3分钟捞出洗净浮沫。

      |
      |
      3.

      焯好洗净的乳鸽放入砂锅，加入枣，龙眼干，东阿阿胶切成小块备用

      |
      |
      4.

      往砂锅内注入没过所有食材的水。

      |
      |
      5.

      盖上砂锅盖，大火煮开。

      |
      |
      6.

      加入切碎了的阿胶块，转文火慢炖，大约2个小时左右。喝之前，再把乳鸽撕开，根据口味调入适量盐即可。
