      香桃奶酪火锅的做法

      我把RIO白桃白兰地鸡尾酒加入奶酪火锅中，吃起来奶酪有着芬芳的香桃味，儿子说没想到这么好吃，我还在拍照，他就老在催好了没有……

      一、主料

      主料用量
      rio白桃白兰地鸡尾酒半碗
      马苏里拉奶酪1碗
      牛奶1碗
      土豆200克
      西兰花50克
      圣女果50克
      面包100克
      红绿甜椒1碗
      蒜片5克

      二、辅料

      辅料用量
      黑椒1小匙
      盐1小匙
      橄榄油1匙
      生粉1匙

      三、做法

      步骤
      1.

      备RIO白桃白兰地鸡尾酒与其他材料

      |
      |
      2.

      土豆、西兰花用水煮熟，面包烤脆

      |
      |
      3.

      用橄榄油爆香蒜片

      |
      |
      4.

      倒入牛奶

      |
      |
      5.

      加进RIO白桃白兰地鸡尾酒

      |
      |
      6.

      下盐、黑椒

      |
      |
      7.

      煮沸后加入生粉与小许水调成的浆

      |
      |
      8.

      所有材料煮成浓稠状

      |
      |
      9.

      奶酪浆小火煮着，把食物拌上即可食用

      |
