      茶树菇老鸭汤的做法

      不知道有没有小伙伴跟我一样，爱吃鸭胜过爱吃鸡，总觉得鸭肉怎么做都超香的。老鸭汤的汤汁澄清香醇，滋味鲜美，鸭脂黄亮，肉酥烂鲜醇。传统的老鸭汤一般都配金华火腿一起炖煮，我用老家的川味腊肠来代替，如果是用腊肉是比较接近火腿的作用的，这是炖好更香老鸭汤的秘密哟。 我的个人美食微信公众号：吴小厨的厨房秘密，欢迎关注，每天更新一道精美食谱哟！

      一、主料

      主料用量
      老鸭500克
      干茶树菇1小把
      干香菇2朵
      腊肠1/2根

      二、辅料

      辅料用量
      姜适量
      小葱1根
      大葱6段
      料酒1勺
      盐适量

      三、做法

      步骤
      1.

      干茶树菇和干香菇温水泡30分钟，除去根部，再微微揉搓冲洗2遍，沥干备用。腊肠洗净切段备好。

      |
      |
      2.

      鸭肉斩切成大块洗净，尽量保留鸭腿。冷水锅内放入鸭块、姜片、大葱段和料酒，大火煮开后再煮1分钟即可捞出。

      |
      |
      3.

      焯水完后的鸭块冷水洗净油脂、冲掉渣滓，沥干备用。

      |
      |
      4.

      处理好的茶树菇和香菇铺在高压锅锅底，上面铺鸭块和腊肠，加入大葱段、姜片和足够盐，大火盖锅烧开（即高压锅大声叫的时候），转小火炖煮40分钟左右，撒上香葱即可上桌。没有高压锅的可以用电饭锅代替，多掺些水，炖煮40分钟也是可以的。
