      莴笋叶肉片粥的做法

      莴笋叶的营养远远高于莴笋茎，具有清热安神、清肝利胆、养胃的功效，含有的莴笋素还能增强胃液，刺激消化，增进食欲。此外，秋冬季爱患咳嗽的人，多吃莴笋叶还可平咳。 这是一道老少皆宜的养生粥品，既营养，又美味，我们家两孩子每次都能吃上两碗。如果你家也有不爱吃 ，不爱吃肉，经常生病、咳嗽的孩子，不防试试这道莴笋叶肉片粥吧！

      一、主料

      主料用量
      莴笋叶80克
      里脊肉180克
      大米200克

      二、辅料

      辅料用量
      花生油25克
      盐1克
      生抽15克
      蚝油3克

      三、做法

      步骤
      1.

      食材准备

      |
      |
      2.

      大米洗净放入电饭锅内胆里，倒入适量清水，接通电源，选择煮粥键

      |
      |
      3.

      里脊肉去筋膜，洗净切薄片，放油、盐、生抽、蚝油搅拌均匀，腌制片刻

      |
      |
      4.

      莴笋叶洗净切小段，沥干水份

      |
      |
      5.

      待米粥浓稠时，加入肉片搅散

      |
      |
      6.

      煮开后，放入莴笋叶再次煮开即可出锅享用
