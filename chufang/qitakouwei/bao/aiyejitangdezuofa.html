      艾叶鸡汤的做法

      母鸡艾叶汤 作用：补气摄血，健脾宁心；适用于体虚不能摄血而致月经过多，心悸怔仲，失眼多梦，小腹冷痛，胃痛，胃寒，据老中医说还有保胎作用哦！

      一、主料

      主料用量
      母鸡肉1只
      艾叶/干15克

      二、辅料

      辅料用量
      盐2勺

      三、做法

      步骤
      1.

      干的艾叶，15克左右

      |
      |
      2.

      一只鸡

      |
      |
      3.

      鸡切块，放入高压锅，加入艾叶。艾叶可以泡一泡，也可以洗了直接放进去哈。水末过鸡肉就可以了，调20分钟。

      |
      |
      4.

      转入汤锅，用小火炖10分钟就可以了

      |
      |
      5.

      做好了
