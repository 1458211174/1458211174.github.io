      莲藕雪梨羹的做法

      微信公众号“宝宝营养辅食”，收录了我给宝宝做过的500道营养辅食，有了这500道辅食菜谱，不再为宝宝吃啥而发愁。小思妈妈会在这个公众号分享6-24个月的宝宝辅食。 多吃白色食物可以润肺，莲藕就是其中的代表。莲藕生吃和熟食功效大不相同。生藕具有清热解渴、止血化痰的功效。莲藕煮熟后，对脾胃有益，可以帮助宝宝养脾胃。 但是莲藕的口感比较硬，宝宝不好咀嚼，所以我将莲藕搅成汁熬煮，这样做出来的汤羹适合宝宝喝，也更容易消化和吸收。莲藕汁通过加热搅拌的方式会慢慢变浓稠，就像加了糯米粉一样，口感非常好喝。再加上雪梨、奇异果和橙子，味道又变得酸酸甜甜，十分清香。 冬天天气比较冷，室温下的水果也有点凉，如果宝宝经常吃室温下的水果，肠胃适应得很好，那直接吃水果是没问题。但对于一些肠胃比较娇嫩的宝宝，可以把水果做成这样的一道热乎甜品。

      一、主料

      主料用量
      莲藕80克
      雪梨35克
      奇异果15克
      橙子15克

      二、辅料

      辅料用量
      冰糖少许

      三、做法

      步骤
      1.

      准备食材：莲藕80g、雪梨35g、奇异果15g、橙子15g。

      PS： 奇异果和橙子也可以用其他水果代替。

      |
      |
      2.

      将莲藕去皮。

      PS： 莲藕的淀粉含量非常丰富，给宝宝煮藕羹只需要一小块，像这么大一块莲藕，大概需要1/3就可以了。

      |
      |
      3.

      切莲藕前，一定要仔细观察一下藕孔中是否还有泥沙，如果有的话要用清水冲洗干净。

      |
      |
      4.

      然后将莲藕切碎。

      PS： 莲藕去皮后如果来不及马上搅拌，可以先放入水中，这样就可以防止莲藕氧化变色，等搅拌的时候可以将莲藕同浸泡莲藕的水一起倒入搅拌杯搅拌。

      |
      |
      5.

      将莲藕倒入搅拌杯中。

      |
      |
      6.

      添加200ml左右清水。

      PS： 清水的比例可以自己掌握一下，如果宝宝平时喜欢喝比较稀一点的汤羹，可以再多添加一点清水。

      |
      |
      7.

      用搅拌机将莲藕搅成细腻的莲藕汁。

      |
      |
      8.

      搅拌后的莲藕汁倒入碗中。

      PS： 莲藕比较容易搅拌，搅拌后的莲藕汁就已经非常细了，很适合宝宝喝。但是如果想要更细腻一点或者给小宝宝喝的话，还可以将藕汁过滤一下，去除藕渣。

      |
      |
      9.

      雪梨去皮后，切成小丁。

      |
      |
      10.

      奇异果用小勺挖出后，切成小丁。

      PS： 奇异果的味道酸酸甜甜，可开胃、助消化，而且它的维生素C含量非常高，在水果中名列前茅。

      |
      |
      11.

      橙子去皮后，用手撕成小块。

      PS： 橙子用刀一切就很容易碎掉，而且汤汁也会流出来，所以处理橙子的时候最好用手撕成小块。

      |
      |
      12.

      将莲藕汁倒入锅中。

      PS： 煮莲藕的时候，最好用不锈钢锅或者不粘锅，不要用铁锅煮，以免引起藕羹发黑。

      |
      |
      13.

      先大火煮沸，然后小火继续煮。

      |
      |
      14.

      煮沸后，将雪梨丁倒入锅中，搅拌均匀。

      PS：因为橙子、奇异果煮后会有一点点发酸，所以我只煮了梨块儿。可以稍候放到莲藕羹里面泡一下，温一下。

      |
      |
      15.

      周岁以上宝宝可以根据口味，加入一小颗冰糖调味。

      |
      |
      16.

      期间不断用勺子搅拌。

      PS： 煮好的莲藕羹比较粘稠，所以在熬煮的过程中要经常搅拌，避免糊锅。

      |
      |
      17.

      最后熬煮成这种粘稠的状态就可以关火出锅啦！

      PS： 藕羹很容易熬粘稠，期间一定要随时观察，大概煮到这种程度就可以了，如果时间长了口感就不好了。

      |
      |
      18.

      莲藕羹盛出后，放上切好的奇异果和橙子，酸酸甜甜的莲藕雪梨羹就做好啦，好喝又十分开胃呦~

      PS： 奇异果和橙子放入藕羹后，会变得温温的，不用担心宝宝吃得太凉呦~
