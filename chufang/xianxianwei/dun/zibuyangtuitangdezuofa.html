      滋补羊腿汤的做法

      最爱喝羊肉汤呢，古时称羊肉为羖肉、羝肉、羯肉。它既能御风寒，又可补身体，对一般风寒咳嗽、慢性气管炎、虚寒哮喘、肾亏阳痿、腹部冷痛、体虚怕冷、腰膝酸软、面黄肌瘦、气血两亏、病后或产后身体虚亏等一切虚状均有治疗和补益效果，最适宜于冬季食用，故被称为冬令补品，深受人们欢迎。

      一、主料

      主料用量
      羊腿1000克
      白萝卜400克
      山药300克

      二、辅料

      辅料用量
      香葱50克
      花椒2克
      八角1克
      香叶2克
      丁香1克
      桂皮1克
      肉蔻2克
      草果2克
      陈皮2克
      干姜2克
      料酒20毫升
      枸杞5克
      鸡精5克

      三、做法

      步骤
      1.

      此步骤用料步骤用时：5分钟
      羊腿1000克

      羊腿剁成小块清洗备用。

      |
      |
      2.

      此步骤用料步骤用时：15分钟
      羊腿1000克

      羊腿入锅焯水，去掉多余的血水

      |
      |
      3.

      此步骤用料
      羊腿1000克

      把焯好的羊腿肉捞出，水倒掉，

      |
      |
      4.

      洋葱切小块

      |
      |
      5.

      把香料放入布袋当中系好袋子

      |
      |
      6.

      另起锅放入清水或者高汤，放入料包，葱头、葱结大火烧开，小火慢炖2小时以上

      |
      |
      7.

      期间翻动锅底放入15克食盐，继续慢炖

      |
      |
      8.

      放入5克鸡精，尝尝咸淡，如果味道不足可以自己在添加食盐

      |
      |
      9.

      炖至1.5小时放入山药和白萝卜慢炖半小时即可，出锅放入枸杞和香葱

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体，滋肾益精，降血糖，延年益寿，助消化。
