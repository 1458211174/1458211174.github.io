      鸡煲蟹的做法

      秋风起，蟹脚肥，又到了吃蟹的季节，大家专称为大闸蟹的，在羊村我们叫毛蟹，小的时候老爸在河里抓到的都会留给我吃，那时候只是清蒸，都觉得好好吃，今天我要做的是我们羊村吃法，鸡煲蟹，因为这次的蟹实在太小了，不管用什么方法做起来都是没肉的，所以选择做汤，喝的就是鲜甜

      一、主料

      主料用量
      大闸蟹400克
      三黄鸡400克
      虾200克
      花蛤蜊250克

      二、辅料

      辅料用量
      生蒜2颗
      芹菜1颗
      姜10克
      小葱2颗
      盐适量
      广东米酒30毫升
      花生油适量

      三、做法

      步骤
      1.

      小葱头，蒜头切出来，其余切段，姜切片，如图

      |
      |
      2.

      清洗鸡肉，切块

      |
      |
      3.

      清洗虾，去虾头虾肠，清洗花蛤

      |
      |
      4.

      水烧开，放花蛤

      |
      |
      5.

      拿个蝶子跟漏勺站在锅边，花蛤一边煮开一边马上捞起来，过冷水，花蛤汤拿碗装好，不要锅底的，会有沙子

      |
      |
      6.

      清洗蟹，还没有解开绳子前先把蟹肚翻开，去掉掩，用牙刷刷掉泥污，再掰掉大腿（这个需要技巧，小心被夹，最好是左手按住一边，右手掰另一边）再解开绳子，用牙刷清洗，蟹腿上的毛用刀刮干净，所有处理完以后，蟹身切两段，蟹腿拍爆，方便出味

      |
      |
      7.

      起油锅，放葱头，蒜头，姜片，爆香

      |
      |
      8.

      先炒鸡肉，放盐，米酒放一半

      |
      |
      9.

      5分钟后加入蟹，继续翻炒，加入剩下的米酒，炒至蟹壳变红，蟹膏都会流出来进到汤汁中，加入花蛤汤，继续炒2分钟关火

      |
      |
      10.

      把材料倒入沙锅，上面铺上花甲，虾，再撒上芹菜，蒜梗（叶子留到最后），加入开水，以刚好跟材料齐平为准

      |
      |
      11.

      加盖，大火烧开，揭开锅盖，转中小火煮15分钟

      |
      |
      12.

      加入蒜叶，煮到微软，如图

      |
      |
      13.

      关火，撒上葱段，如图，这样一锅鸡煲蟹就算完成了
