      鹰嘴豆番茄糊的做法

      微信公众号“宝宝营养辅食”，收录了我给宝宝做过的500道营养辅食，有了这500道辅食菜谱，不再为宝宝吃啥而发愁。小思妈妈会在这个公众号分享6-24个月的宝宝辅食。 鹰嘴豆富含丰富的蛋白质，被称为“豆中之王”，富含多种它还是一种很好的植物氨基酸补充剂，有较高的医用保健价值，对儿童智力发育、骨骼生长有很好的作用。 那么问题来了，这样高营养的鹰嘴豆，我们怎么做给宝宝吃呢？因为豆子本身比较大粒，不适合3岁以前的宝宝直接吃。所以，我一般都会打成米糊来跟宝宝一起喝，喝起来香香的栗子味。 这次我又换了一种吃法，将鹰嘴豆和番茄搭配起来，没想到味道也非常赞呦~煮好的鹰嘴豆，自带一种微甜的香气，再加上核桃碎，喝起来就更香了，鹰嘴豆完全可以作为小宝宝的一顿主食，也可以作为大宝宝的早餐。

      一、主料

      主料用量
      鹰嘴豆60克
      番茄30克
      核桃2个

      二、辅料

      辅料用量
      水适量

      三、做法

      步骤
      1.

      准备食材：鹰嘴豆60g、番茄30g、核桃2个，水适量。

      PS： 我是在麦德龙超市买的鹰嘴豆，一般小的超市可能会买不到。如果你那里实在找不到的话，也可以在万能的网上购买哈！一次买一大袋，够全家人吃很久。
      文中这些食材仅代表制作量，不是宝宝一顿的饭量，可以跟妈妈一起分享。
      第一次给宝宝添加鹰嘴豆一定要注意观察宝宝对其是否过敏哦。

      |
      |
      2.

      鹰嘴豆提前用清水浸泡一夜。

      PS： 鹰嘴豆即使用高压锅煮也比较难煮烂，所以一定要提前浸泡噢！浸泡的时候可以放在冰箱中，避免因温度太高而发酵。

      |
      |
      3.

      鹰嘴豆浸泡好后，放入高压锅中，选择豆类模式，将鹰嘴豆煮至软烂。

      PS： 大概煮到鹰嘴豆膨胀爆裂，用手轻轻捏就能捏碎的程度就可以了。煮鹰嘴豆的水千万不要扔掉噢，营养价值也很高，可以直接用来搅拌成糊。

      |
      |
      4.

      在番茄表面划一个十字，然后倒入热水浸泡至表皮涨开，然后去掉表皮。

      |
      |
      5.

      取一小半番茄，切成小丁。

      PS： 番茄不要全部用掉噢，否则味道会有些酸，宝宝不爱喝，酌情少用一点即可。

      |
      |
      6.

      将核桃夹碎，取出核桃仁。

      PS： 核桃是一种营养价值很高的坚果，含丰富的蛋白质、不饱和脂肪酸、磷脂、维生素E、钾、锌等营养成分。但是核桃含有较多的脂肪和能量，宝宝不可多吃得过多。

      |
      |
      7.

      然后把核桃倒入搅拌机中磨成粉。

      PS： 如果没有搅拌机的话，直接用擀面杖将核桃碾成碎末就可以了。因为核桃比较坚硬，如果比较大的话，宝宝很容易卡到嗓子，所以给宝宝吃核桃的时候一定要弄成粉末状。

      |
      |
      8.

      核桃搅成粉后倒入碗中备用。

      |
      |
      9.

      将高压锅中压好的鹰嘴豆盛出，倒入小奶锅中煮沸。

      |
      |
      10.

      然后加入番茄继续熬煮5分钟。

      |
      |
      11.

      将煮好的鹰嘴豆番茄连同汤汁一同倒入搅拌机中，搅拌成细腻的糊。

      |
      |
      12.

      鹰嘴豆番茄糊搅拌好后，再次倒入锅中加热，然后加入核桃粉末，搅拌均匀，待全部煮熟后即可出锅。

      |
      |
      13.

      最后麻麻还可以用酸奶画出一个小心形，宝宝看了一定会更喜欢呦~
