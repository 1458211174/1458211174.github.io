      腊排骨煲玉米松茸汤的做法

      表哥前两周，去北京工作前，到我这里玩了3天，来的时候给带了干松茸和竹荪，是他去年在云南工作时候认识的朋友，给他寄过来的。他知道我喜欢捣鼓吃的，所以就拿了一些给我，谢谢你，亲爱的表哥，愿你在北京的工作，生活，一切都安好！！！ 松茸，菌肉肥厚，具有香气，味道鲜美，常食松茸能强精补肾，恢复精力，另具有益胃补气，强心补血，健脑益智、理气化痰，驱虫以及治糖尿病和抗癌等作用。 周末想喝碗汤，想着家里有风干的腊排骨，配上干的松茸片，加上清甜的玉米，汤水略带甜味，混合咸香菌香，喝汤啃排骨，还有贴心的你陪伴，真好！！！

      一、主料

      主料用量
      腊排骨750克
      干松茸片60克
      甜玉米1根

      二、辅料

      辅料用量
      干辣椒1个
      生姜15克
      精盐3克
      桂皮2克

      三、做法

      步骤
      1.

      腊排骨剁小块，加水浸泡8小时。（精灵是提前一晚浸泡的。腊排骨太硬，让俺爹帮忙剁的）

      |
      |
      2.

      干松茸片加水浸泡3小时。

      |
      |
      3.

      浸泡好的排骨，加姜片焯水。

      |
      |
      4.

      泡发好的松茸片，洗净挤干水份；生姜切片，辣椒，桂皮冲洗浮尘。

      |
      |
      5.

      电锅中加入焯水洗净的腊排骨，松茸片。

      |
      |
      6.

      加入姜片，干辣椒，桂皮。

      |
      |
      7.

      加入热水，接通电源炖煮。

      |
      |
      8.

      甜玉米洗净剁小段。

      |
      |
      9.

      2小时后加入玉米，盐调味后再炖煮1小时即可。
