      牛奶花生汤

      自从官老师有了一支免费使用的快锅时. 于是只好来想想有哪些食谱可以让快锅发挥其特色 / 好让广大的美食爱好者或者说是快锅爱用者能够知道家里的快锅除了炖肉，卤猪脚外当然也可以来炖花生 在写这篇稿子的同时.其实一旁官老师也正在煮着花生呢…. 想想大概再过20分钟就可以完成了…就当做是今晚的甜点吧!

      一、材料

      用的顺手压力锅一支

      花生600g

      水1.5公升

      牛奶0.5公升

      细白砂糖150g

      二、做法

      步骤
      首先是花生的选择….这次官老师选择的是带皮的花生 /当然您也可以选择去皮的花生仁 /只是要注意别买到了加塩的就是了……哈!就看个人的喜好囉
      对于带皮的花生也许您会觉得使用前是否应该静泡一段时间较可缩短烹调时间….当然 /我不反对..理论上来说也合理 / 但偏偏官老师喜欢想吃什么就立刻做 / 所以要我等到静泡完成 /我想届时可能就没那个热忱了 /不知您是否也跟我一样呢
      相信只要有快锅来帮您煮…其实也就不用太担心煮好后的花生会否还是会有不熟的问题了而对于已经洗净的花生 / 官老师另外会再将浮在表层的花生给捞除 /因通常这类的花生多为空心或者里头有腐坏了..所以还是建议捞除较为保险
      将洗净的花生过滤干净
      将滤干净的花生放入快锅里
      加入配方中等比例的水 / 在这边官老师使用的是过滤的纯水 / 相信煮好后的花生汤也会较为清甜家里如有的话也可以使用 / 但也不一定非纯水不可 / 就看每个人对水质的感受度囉
      到此 / 那就来加盖吧
      火力的大小我是一开始就转大火 / 之后再转小火 / 至于何时才是之后呢?
      每家的压力锅设计都不一样 / 就以官老师这支锅来说的话 /其最上头会有一个称做为配重头的配件 /当配重头开始发出777的声音时就可以转小火了 / 同时开始计时40分钟
      同时也会发现旁边的红色压力阀升起如上图黄色圆圈处就是了 /只要持续的保持小火但也别让红色压力阀下降，然后持续的保持烹调40分钟关火，等待压力阀下降后总时间约为一小时 ,别了压力阀下降的同时也是有在烹调的 / 所以时间也要算进去
      小火火量的大小大约是这样..但还是要视该品牌压力锅的设计来决定 / 官老师使用的营业用的瓦斯炉，一般家庭的话转到炉心火应该就可以了
      时间到..这就是打开后的样子囉!表层还会浮上层浅浅的花生油…我想这应该也只有压力锅才能淬取的出来吧
      看看刚炖好的花生 /入口后的口感可以说是相当的绵密
      加入500CC的牛奶
      同时再加入150公克的白砂糖，再开小火持续煮到滚并确定砂糖溶化后 / 试试看老师的甜度比例是否喜欢再做微调整
      搅拌直到牛奶煮沸糖也溶化就可以了
      成品完成
      完成的牛奶花生汤除可热热的喝的以外 /也可以冰凉的食用，风味不减 /官老师是觉得热的喝有养生的感觉 /冰的喝则会因为油脂较为凝固所以口感上较为不油腻总之都不错 / 就看各人喜好囉.
      〒不得不说对于压力锅的使用，各家厂牌的设计或有不同，当然烹调时间也或有些许差异，实无法同一比较，建议还是要多练习看看，注意安全就是了
