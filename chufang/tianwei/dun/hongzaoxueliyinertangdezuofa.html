      红枣雪梨银耳汤的做法

      味道甜美，止咳润肺，养胃生津，滋阴润肺，春秋季大人孩子都多喝点吧～

      一、主料

      主料用量
      银耳2小朵
      红枣4颗
      雪梨1个

      二、辅料

      辅料用量
      冰糖3块
      水适量

      三、做法

      步骤
      1.

      准备所需的食材。备注：琪琪妈没有拍水的图片

      |
      |
      2.

      银耳温水中泡15分钟，去除根，撕成小片，待用。

      |
      |
      3.

      梨清水洗干净，切成小块，待用。

      |
      |
      4.

      红枣洗干净，去核，一个枣撕成两三瓣即可。

      |
      |
      5.

      起锅，加4小碗水，依次放入梨、银耳、红枣，大火烧开后，转小火，加入冰糖慢炖30分钟。此步以砂锅为宜

      |
      |
      6.

      美味甜汤做好啦～你学会了吗？和琪琪妈一起做起来吧～

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体，治流鼻血，清热解暑，化痰止咳，润肺化痰，高血压，利尿，养颜护肤。
