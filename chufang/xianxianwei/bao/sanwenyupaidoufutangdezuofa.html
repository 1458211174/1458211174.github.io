      三文鱼排豆腐汤的做法

      冬日清晨来一碗，鲜的不要不要的，身上暖暖哒。

      一、主料

      主料用量
      三文鱼排300克
      豆腐100克

      二、辅料

      辅料用量
      青菜100克
      生姜30克

      三、做法

      步骤
      1.

      煎炸金黄

      |
      |
      2.

      开锅加豆腐

      |
      |
      3.

      加生姜片

      |
      |
      4.

      叶菜都可以

      |
      |
      5.

      大火10分钟，转小火20分钟即可

      |

      补钙，缓解更年期，预防疾病，防乳腺癌，有益大脑。
