      香蕉粥的做法

      香蕉具有较高的药用价值。它可以清肠胃，治便秘，有清热润肺、止烦渴、填精髓、解酒毒等功效。香蕉性寒，咱们女生尽量少吃。脾胃虚寒、胃痛、腹泻、胃酸过多的筒子也尽量少吃。但煮过的香蕉，就没事啦。煮好的香蕉粥比较香甜，如果你喜欢吃香蕉，过一个不上火的春天，来试试【香蕉粥】吧。

      一、主料

      主料用量
      大米80克
      香蕉1根

      二、辅料

      辅料用量
      清水适量

      三、做法

      步骤
      1.

      电饭煲洗干净后，倒入适量清水烧开；

      |
      |
      2.

      大米淘洗干净，备用；

      |
      |
      3.

      香蕉去皮，切薄片；香蕉要选熟软度比较高的；

      |
      |
      4.

      清水烧开时，将大米放入锅里；

      |
      |
      5.

      多搅拌几次，以免粘锅；

      |
      |
      6.

      米粒煮至8分熟；

      |
      |
      7.

      放入香蕉片；

      |
      |
      8.

      多搅拌几次，这样香蕉粥会比较浓稠；香蕉粥煮至软烂时，即可断电；

      |
      |
      9.

      香蕉粥盛入碗里，趁热食用。

      |

      四、功效

      适合婴儿，强身健体，抵抗力，健胃，降压，通便排毒，助眠。
