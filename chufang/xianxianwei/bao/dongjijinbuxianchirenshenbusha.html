      冬季进补鲜吃人参不上火【鲜人参鸡汤】的做法

      　今年广州冬天进补又有个好选择，那就是鲜人参。鲜人参浆水充足，富有独特的参香，保留了人参的所有活性成分，营养成分更全面。没有任何加工、无农药残留，无漂白、熏蒸、上色、嫁接等非法二次加工。最关键的是食用鲜人参不会引起上火，可以象吃水果一样吃，大补却不上火。 　　鲜人参可强身健体，还能改善老年人心脑血管状况；对于女性来说，人参可以改善体寒怕冷、冻手冻脚的症状，调理女性气血，改善生理期不适；对于经常吸烟喝酒的男士来说，人参能改善肺气不足、气短喘促、少气乏力等症状，还可解酒护肝，降低酒精对肝脏的损耗；冬季给孩子服用一些人参汤，能改善其肠胃功能、增强体质，提高人体免疫力和抵抗力。这么多的好处，亲们快来试试最天然的进补方式。

      一、主料

      主料用量
      5年鲜人参2支
      土鸡1只
      枸杞5克
      红枣15克

      二、辅料

      辅料用量
      姜3克
      葱白3克
      盐5克
      清水适量

      三、做法

      步骤
      1.

      材料备好。

      |
      |
      2.

      土鸡去除内脏清洗干净，放入电饭煲里，加入最高水位线的清水。

      |
      |
      3.

      启动电饭煲的“煲汤”功能，盖上盖子，时间为120分钟。

      |
      |
      4.

      这时我们把鲜人参、红枣洗干净，枸杞浸泡好，葱白和姜切好待用。

      |
      |
      5.

      水煮至将滚时用勺子捞出鸡汤表面的浮沫。

      |
      |
      6.

      浮沫捞干净后，放入鲜人参、枸杞、红枣、姜和葱白。

      |
      |
      7.

      盖上盖子，让电饭煲继续煲煮。

      |
      |
      8.

      煮至程序快结束前放入枸杞和盐，如果汤味道不够浓郁，可再加一个程序煲煮。

      |
      |
      9.

      煮好的鸡汤放至微温时即可开动啦。
