      龙利鱼丸子汤的做法

      做给宝宝吃，味鲜没刺首选

      一、主料

      主料用量
      龙利鱼一片

      二、辅料

      辅料用量
      马铃薯淀粉适量
      鸡蛋清1个
      葱花儿适量
      精盐少量

      三、做法

      步骤
      1.

      去筋膜

      |
      |
      2.

      辅食机把龙利鱼搅碎～或者用刀都可以，依个人口味添加葱花，是给宝宝做，盐只放了一点点，所以用葱花调鲜

      |
      |
      3.

      分离蛋清蛋黄，只用蛋清

      |
      |
      4.

      蛋清淀粉放入龙利鱼泥里，顺时针搅拌，多搅拌一会，口感更好

      |
      |
      5.

      小火温水下锅，全部挤好后调温煮沸，煮至漂浮膨胀即可，特别好熟

      |
      |
      6.

      不做汤直接使用也可以滴，宝宝特别爱吃

      |
      |
      7.

      放了少许冬瓜片煮沸，因为是宝宝吃，所以只放了一点点盐，没放别的调料

      |
