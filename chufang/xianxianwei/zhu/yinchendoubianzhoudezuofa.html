      茵陈豆扁粥的做法

      在春天，喝点蔬菜粥，清爽又美味。

      一、主料

      主料用量
      茵陈50克
      玉米面2把
      豆扁2把

      二、辅料

      辅料用量
      盐半勺
      水适量

      三、做法

      步骤
      1.

      刚采摘的茵陈去根去黄叶

      |
      |
      2.

      浸泡一会后清洗干净沥水

      |
      |
      3.

      切成小段

      |
      |
      4.

      豆扁用水淘洗干净

      |
      |
      5.

      倒入锅中，加适量水

      |
      |
      6.

      大火煮开后会出现大量浮沫

      |
      |
      7.

      用饭勺将浮沫搅一下，使浮上来的豆扁下沉，将浮沫撇出

      |
      |
      8.

      下入玉米面，沿锅底搅拌几下

      |
      |
      9.

      大火煮开转中火慢煮15分钟即可

      |
      |
      10.

      倒入茵陈再煮2分钟

      |
      |
      11.

      出锅前加盐调味

      |

      四、功效

      清热去火，减肥瘦身，防癌，降血压，保护视力，润肠通便，健脑，抗衰老。
