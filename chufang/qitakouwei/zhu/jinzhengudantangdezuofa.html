      金针菇蛋汤的做法

      简简单单的汤，很好喝哦

      一、主料

      主料用量
      鸡蛋3个
      金针菇200克

      二、辅料

      辅料用量
      葱花儿少许
      盐1勺

      三、做法

      步骤
      1.

      去根

      |
      |
      2.

      洗净焯水

      |
      |
      3.

      捞出控水

      |
      |
      4.

      起油锅，稍微炒下，加盐

      |
      |
      5.

      加水煮

      |
      |
      6.

      鸡蛋加盐打散

      |
      |
      7.

      水开倒入蛋液，少许盐调味

      |
      |
      8.

      出锅前洒葱花

      |

      四、功效

      健脑益智，保肝，动脉硬化，防癌，延年益寿，美容护肤，抵抗力，降血脂，过敏。
