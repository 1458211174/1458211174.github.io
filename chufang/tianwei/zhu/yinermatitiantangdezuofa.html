      银耳马蹄甜汤的做法

      北方冬天的天气干燥，容易上火，这几天更不用说了，又开启了雾霾模式，女儿班上感冒发烧的同学不少，赶快炖些简单的荸荠糖水，清清肺泻泻火。

      一、主料

      主料用量
      银耳20克
      荸荠10个

      二、辅料

      辅料用量
      冰糖适量

      三、做法

      步骤
      1.

      食材：荸荠、银耳、冰糖

      |
      |
      2.

      银耳提前泡发，泡发好的因而撕成小朵，摘去老根

      |
      |
      3.

      将泡好的银耳去蒂撕成小朵放入电饭煲中，加适量清水。（手边有三粒红枣，随手放里了，家里如果有红枣，多放几颗，好吃

      |
      |
      4.

      启动汤粥键炖制。

      |
      |
      5.

      煮银耳的时候马蹄去皮后切小块

      |
      |
      6.

      如果不立刻用 先放在淡盐水中浸泡几分钟防止氧化变黑。

      |
      |
      7.

      汤粥程序进行到一半，放入切好的马蹄和冰糖，盖上盖子继续炖煮。

      |
      |
      8.

      电饭煲汤粥程序结束即可。软糯清甜，清热去火，有时间也给家人煮上一碗吧！

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体，促发育，降压，通便排毒，清热解毒。
