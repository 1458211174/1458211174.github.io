      米伴侣玉米粥的做法

      阳春三月不乏搞点营养粥，八宝粥当然还是要五谷杂粮种类多一点的好，今天我们推出一款营养又美味的米伴侣玉米粥。

      一、主料

      主料用量
      米伴侣60克
      黄玉米1根
      黑玉米1根
      小米苦荞米40克

      二、辅料

      辅料用量
      枣6颗
      水5L

      三、做法

      步骤
      1.

      将准备好的米伴侣淘洗一遍，今天所选的米伴侣是经过预加工处理过的，所以不用泡，玉米切块；

      |
      |
      2.

      将米伴侣、黄金粥（小米+苦荞）、玉米和枣放入电饭煲内，加水，按下煲粥键；

      |
      |
      3.

      35分钟后即可食用

      |
      |
      4.

      乘入碗内，搭配包子，馒头即可享用美味早餐米伴侣玉米粥
