      宝宝小米油 宝宝健康食谱的做法

      中医上一直认为米油是很好的营养品，还有“米油赛参汤”的说法。能把粥油和人参相比，可见它补元气的功能是不容小觑的。 虽然现在的宝宝可选择的辅食种类很丰富，不必再像几十年前用米油喂婴儿了。 但对于辅食添加初期的宝宝们来说，米油比其他食物更容易消化吸收！ 还有容易消化不良和生病的大小宝宝们，胃肠功能紊乱、便秘、腹泻、呕吐是经常的事儿，这时候选择给宝宝喝几天小米粥油做辅食，能调整宝宝肠胃，帮助顺利度过生病期。 很多妈妈跟我说，煮小米粥的时候煮不出粘稠的感觉，也没有米油怎么办，其实熬它还是有讲究的，所以写了今天这个详细的步骤帖，一步步跟着做出好喝的养胃米油吧。

      一、主料

      主料用量
      小米30克

      二、辅料

      辅料用量
      水180克

      三、做法

      步骤
      1.

      参考月龄：6个月以上
      食材准备：小米30-50克
      烹饪时间：30分钟
      推荐指数：★★★★★

      |
      |
      2.

      小米浸泡15分钟以上。

      冷水入锅，米和水的比例是1:5，也就是一杯米加五杯水的比例。
      如果刚添加辅食初期的4-6个月的宝宝，米和水的比例可以是1:7。

      |
      |
      3.

      米下锅烧开后，转小火慢熬。

      |
      |
      4.

      煮至过程中，可以在锅盖处放一个勺子帮助透气，防止米汤溢出。

      |
      |
      5.

      等到米粒熬制膨胀状态以后，勤用勺子不断搅动，帮助米粒脱粉，也能防止糊锅。

      |
      |
      6.

      熬20-30分钟以后的样子，这个时候如果喜欢稀一点的就可以喝了。
      如果给宝宝吃，或者喜欢更浓稠的，可以继续熬了10-15分钟。小米和水已经完全融合在一起了。

      |
      |
      7.

      凉一点的时候，传说中的米油就飘起来了。这个特别有营养。

      也可以直接用滤网过滤一下，就盛出米油啦。

      |
      |
      8.

      粥为第一补物，而米油就是粥的灵魂。

      |

      四、功效

      开胃消食，抗感染，预防流产，维持性功能，祛斑美容，促进生长，助胎儿发育。
