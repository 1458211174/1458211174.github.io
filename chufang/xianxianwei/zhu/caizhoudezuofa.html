      菜粥的做法

      第一次发菜谱呢，好多不足的地方希望大家谅解，有不懂的地方尽管问我，我一定知无不言哟

      一、主料

      主料用量
      米适量
      小油菜适量
      猪肉适量
      水适量

      二、辅料

      辅料用量
      盐1勺
      葱1段
      油1勺
      花生碎适量
      干蘑菇适量
      豆刨适量
      味极鲜适量
      蚝油适量
      鸡精适量

      三、做法

      步骤
      1.

      小油菜浸泡一会儿

      |
      |
      2.

      蘑菇干豆刨泡软，豆刨切条，蘑菇切丁，豆刨可以不用

      |
      |
      3.

      牡蛎干泡软，把水倒掉

      |
      |
      4.

      把花生米擀成花生碎，然后最好把皮去掉，我太懒没有去皮

      |
      |
      5.

      肉之前忘了拍，瘦肉肥肉没有局限，我用的是肥肉，锅里倒少量油，小火，把肥肉熬出油

      |
      |
      6.

      然后下葱末，豆刨，蘑菇翻炒

      |
      |
      7.

      翻炒几下，倒入洗好的米，放入酱油和蚝油翻炒一会儿

      |
      |
      8.

      倒入清水，把花生碎，牡蛎干放进去，然后盖上锅盖，先大火后小火慢慢把米煮熟

      |
      |
      9.

      等米熟了放入小油菜，小油菜熟了尝一下汤的味道，淡了加盐，不够鲜加鸡精，感觉都正好就不用放这两样了，根据自己的口味调

      |

      四、功效

      止疼，清热解毒，排毒，抵抗力，美容养颜。
