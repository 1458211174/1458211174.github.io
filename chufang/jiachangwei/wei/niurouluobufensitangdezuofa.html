      牛肉萝卜粉丝汤的做法

      牛肉性味甘平，含有丰富的蛋白质，脂肪、维生素B族、烟酸、钙、磷、铁、胆甾醇等成分。具有强筋壮骨、补虚养血、化痰熄风的作用。同时牛肉的脂肪含量很低，但它却是低脂的亚油酸的来源，含有的锌是一种有助于合成蛋白质、能促进肌肉生长的抗氧化剂，牛肉中对防衰防癌具有积极意义；牛肉中含有的钾对心脑血管系统、泌尿系统有着防病作用；含有的镁则可提高胰岛素合成代谢的效率，有助于糖尿病的治疗。增长肌肉、增加免疫力、促进康复、补铁补血、抗衰老，也是每天所需要的铁质的最佳来源。 我家虽不是回民，但那小子不知何故从小就爱吃牛羊肉胜过传统的猪肉呢；吃的最多的属红烧和干切牛肉，炖牛肉汤还真是头一回；而我又在牛肉汤里加了通气清火的萝卜；所以在我炖牛肉汤的过程中，那位时不时瞥过一丝质疑的眼神；当我将炖好的牛肉萝卜汤端上桌请他品尝后，只听到他跟儿子说：“儿子，你妈这牛肉汤炖的还真是又香又嫩又好喝呢”！牛肉萝卜汤：好吃不上火~~

      一、主料

      主料用量
      白萝卜498克
      牛键子645克
      山芋粉丝1小把

      二、辅料

      辅料用量
      小葱2根
      生姜4片
      料酒2汤匙
      食盐3小勺
      鸡粉1/2小勺

      三、做法

      步骤
      1.

      准备好所有食材

      |
      |
      2.

      山芋粉丝提前浸泡变软

      |
      |
      3.

      牛肉洗净后冷水入锅焯水

      |
      |
      4.

      洗净后与葱姜、清水再次入锅

      |
      |
      5.

      加盖大火煮开

      |
      |
      6.

      沿锅边滴适量料酒去腥转小火慢炖

      |
      |
      7.

      白萝卜去皮清净切滚刀块

      |
      |
      8.

      待牛肉炖至8成熟时加入萝卜，加盖煮开后转小火

      |
      |
      9.

      待萝卜浮起时放入山芋粉丝

      |
      |
      10.

      再次煮开后调入食盐

      |
      |
      11.

      加入适量鸡粉

      |
      |
      12.

      拌匀后关火即可

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体。
