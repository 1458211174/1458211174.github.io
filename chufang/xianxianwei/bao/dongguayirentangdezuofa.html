      冬瓜薏仁汤的做法

      这是一款经典的祛湿汤，春天喝它最合适了。

      一、主料

      主料用量
      冬瓜100克
      薏仁100克
      赤小豆50克
      猪大骨500克

      二、辅料

      辅料用量
      食盐1茶匙
      生姜3片

      三、做法

      步骤
      1.

      清洗薏仁赤小豆，放入锅里。

      |
      |
      2.

      大骨头清洗干净，放入电饭锅。

      |
      |
      3.

      冬瓜清洗干净，把表皮清洗干净，然后皮切成大块，一并放入锅里，冬瓜籽去掉。最后再把冬瓜切块。

      |
      |
      4.

      把所有食材放入电饭锅，加适量清水，然后加生姜和盐，盖上盖子，按煲汤功能即可。

      |

      四、功效

      减肥瘦身，滋养肾气，化痰，防癌，润肤美容，降血糖，降胆固醇，降血脂，软化血管，助减肥，降压，美白美容，抵抗力，糖尿病，解热镇痛，治脚气病，清热去火，助消化，强身健体，通便排毒，消除水肿，健脾，下奶。
