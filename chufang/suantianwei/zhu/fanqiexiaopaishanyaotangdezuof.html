      蕃茄小排山药汤的做法

      山药又称为淮山药、怀山药、薯蓣、山芋、白苕等，是药食俱佳的珍品。山药中的黏液多糖物质与无机盐结合后可形成骨质，让软骨具有弹性，同时能增强骨骼强度与密度，故骨质疏松者可以多吃山药，预防和改善骨质疏松。 在冬天里适合润补，最佳食物是山药。山药中含有大量的粘液蛋白，维生素和微量元素，有效预防血脂在血管壁中沉淀，预防心血管疾病。中医认为秋天吃山药有健脾益胃，滋肾益精和益肺止咳的功效。因此在当季给孩子们煲上一锅暖暖又好喝的汤水，相信宝妈们一定是非常乐意的一件事。

      一、主料

      主料用量
      小排骨250克
      蕃茄1只
      山药1段

      二、辅料

      辅料用量
      料酒10克
      盐2克

      三、做法

      步骤
      1.

      先将小排骨洗净，山药及蕃茄备好

      |
      |
      2.

      山药去皮切小块浸到水中

      |
      |
      3.

      蕃茄切成小块

      |
      |
      4.

      热锅放下少许油，下排骨煸炒几下，淋入料酒，放入蕃茄炒出汁水

      |
      |
      5.

      加入二碗水，开始炖上15分钟。

      |
      |
      6.

      将山药放入汤水中再煮上10分钟，等山药绵软后，盐调味就可以了。

      |

      四、功效

      消食，清热解毒，清热解暑，补血，消食开胃，滋肾益精，降血糖，延年益寿，助消化。
