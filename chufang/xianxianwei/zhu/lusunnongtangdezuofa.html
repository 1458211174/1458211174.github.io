      芦笋浓汤的做法

      被称为“蔬菜之王”的芦笋，不仅营养价值非常丰富，而且味道鲜美，还可以抗癌，所以受到许多人的喜爱。今天做了芦笋浓汤，配上烤脆的蒜香吐司，好吃！

      一、主料

      主料用量
      土豆1个
      培根2片
      芦笋250克

      二、辅料

      辅料用量
      牛奶2勺
      黑胡椒碎少许
      浓汤宝半小盒
      盐少许
      黄油10克

      三、做法

      步骤
      1.

      芦笋洗净，把尾部老的部分切掉

      |
      |
      2.

      土豆去皮切薄片

      |
      |
      3.

      把芦笋和土豆上笼蒸10分钟左右

      |
      |
      4.

      培根切小块

      |
      |
      5.

      放入锅中煸出油煸干

      |
      |
      6.

      捞出用厨房纸巾吸干多余油份

      |
      |
      7.

      准备一碗高汤：锅中加入适量的清水和半盒浓汤宝煮至融化

      |
      |
      8.

      这是蒸熟的土豆和芦笋，把芦笋嫩头切下摆盘用，其他部分切小段

      |
      |
      9.

      把蒸熟的土豆和芦笋段放入料理机中

      |
      |
      10.

      加入熬好的高汤

      |
      |
      11.

      启动机器搅打

      |
      |
      12.

      把打好的芦笋浓汤倒回锅中加热，加点黄油

      |
      |
      13.

      加入适量的牛奶、盐和黑胡椒碎拌匀即可关火

      |
      |
      14.

      盘中放入芦笋嫩头和少许的培根粒

      |
      |
      15.

      加入适量的芦笋浓汤即可

      |

      四、功效

      美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，补钙，心脏病，增强体能，抵抗力，增进食欲，消食，防癌，清热利尿，健脑。
