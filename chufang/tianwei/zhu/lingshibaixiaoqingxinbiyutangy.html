      零失败小清新碧玉汤圆的做法

      中国人的节日素来和吃有剪不断的联系，元宵节到了，南汤圆北元宵又要掐起来了。汤圆和元宵其实没有什么大差别，汤圆是包出来的，汤色清亮些，元宵是滚出来的，汤色浑浊些。可是不管怎样，喜欢汤圆的还是吃汤圆，喜欢吃元宵的还是吃元宵，大家开开心心的过节，一年顺利吉祥！

      一、主料

      主料用量
      黑芝麻110克
      白砂糖85克
      猪油75克

      二、辅料

      辅料用量
      水磨糯米粉180克
      温水适量

      三、做法

      步骤
      1.

      黑芝麻炒熟，不需要放油

      |
      |
      2.

      加入白砂糖

      |
      |
      3.

      放料理机打成粉，越细越好

      |
      |
      4.

      加入猪油，拌匀

      |
      |
      5.

      搓成3克每个的小圆球，入冰箱冷藏

      |
      |
      6.

      准备好糯米粉

      |
      |
      7.

      加入适量的温水，揉成团

      |
      |
      8.

      此时的粉团延展性差，一掰就碎

      |
      |
      9.

      取大约十分之一的粉团，压扁，入锅煮熟

      |
      |
      10.

      加入原来的粉团

      |
      |
      11.

      揉搓均匀，延展性极佳

      |
      |
      12.

      将粉团分成两份，一份加入色素

      |
      |
      13.

      搓成两个长条

      |
      |
      14.

      掰成5克每个的小段，为了美观我搓圆了

      |
      |
      15.

      包入冷藏好的黑芝麻馅

      |
      |
      16.

      搓圆

      |
      |
      17.

      水开后煮汤圆，入锅后轻轻滑动锅铲防止粘锅，待汤圆浮起就熟了

      |
      |
