      土豆玉米浓汤的做法

      超好喝，又营养的西式浓汤，并且打破了通常勾芡的方法，直接用土豆代替了芡粉，更有质感。 材料：玉米一盒净重198克、洋葱40克、土豆180克、培根50克、淡奶油50毫升

      一、主料

      主料用量
      玉米粒198克
      洋葱40克
      土豆180克
      培根50克
      淡奶油50毫升

      二、辅料

      辅料用量
      盐2克
      油10克

      三、做法

      步骤
      1.

      备好材料：玉米一盒净重198克、洋葱40克、土豆180克、培根30克、淡奶油50毫升

      |
      |
      2.

      洋葱和土豆切丁。

      |
      |
      3.

      锅里少倒点油，油温热，先将培根炒出油，略焦盛出备用。

      |
      |
      4.

      用原油原锅将洋葱和土豆炒变色炒出香味。

      |
      |
      5.

      然后倒入玉米粒，加水漫过，盖锅盖煮开。

      |
      |
      6.

      连汤带水倒入调理机中打成泥。

      |
      |
      7.

      再倒回锅中，煮开，要边煮边搅拌，如太稠加水调整，并加盐调味。

      |
      |
      8.

      盛到碗中撒上焙焦的培根粒享用。

      |

      四、功效

      抵抗力，消食，降血压，补钙，防癌，延年益寿，美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，心脏病，增强体能，增进食欲。
