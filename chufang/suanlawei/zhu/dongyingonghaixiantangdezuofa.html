      东阴功海鲜汤的做法

      今天开工不知道吃什么，想起家里有一包冬阴功酱，还有海鲜，那就煮一道红红火火的菜冬阴功海鲜汤。

      一、主料

      主料用量
      大虾4只
      沙白100克
      青口100克
      蘑菇5颗
      鲜香菇4颗
      鸡胸肉100克

      二、辅料

      辅料用量
      西红柿1个
      指天椒2颗
      冬阴功酱1包

      三、做法

      步骤
      1.

      清洗好名种海鲜，我准备了虾、青口、沙白放入盘中。今天家里没有鱿鱼，放一些鱿鱼味道更好。

      |
      |
      2.

      把蘑菇、西红柿切成块，香菇打上花刀，鸡肉切成片，指天椒切碎放入盘中。

      |
      |
      3.

      炒锅里加少许的油，加热后，放入东阴功酱用小火炒出香味。

      |
      |
      4.

      东阴功酱炒出香味后加入300毫升的水

      |
      |
      5.

      放入300毫升水后加入海鲜和其它所有菜品大火煮开后，在煮十分钟根据自己口味可加入少许的盐，如果有薄荷叶和柠檬汁可加一点味道会更佳。

      |
      |
      6.

      一碗美味的东阴功汤就做好了。

      |

      四、功效

      抵抗力，下奶，安神助眠，动脉硬化，助睡眠，散瘀止疼，止咳化痰，通便排毒，促进食欲，助消化，防癌，养肾，消食。
