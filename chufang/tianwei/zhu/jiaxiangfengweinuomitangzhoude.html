      家乡风味糯米糖粥的做法

      糯米是爷爷奶奶亲自种的，过年时他们特意拿给我的，某天无意中看到“糖粥”我的记忆之门被打开了，想起小时候吃的家乡糯米糖粥，当时家家户户“入伙”（搬新家）时会煮来吃，过年时则在粥煮好后加入“煎堆”，是软、瘪的那种“煎堆”，还有加“粉丝”一起煮的，红红火火的意头。 电话里跟妈妈“取经”，第一次自己煮这个粥，这里买不到家乡那种软煎堆，所以不加软煎堆。 控制着吃哟，毕竟糖多～

      一、主料

      主料用量
      糯米155克
      冰片糖53克

      二、辅料

      辅料用量
      水适量
      姜4克

      三、做法

      步骤
      1.

      食材准备：冰片糖、糯米。

      |
      |
      2.

      简单清洗1、2次糯米，不用洗太多次、太久。

      |
      |
      3.

      姜清洗、刨丝。

      |
      |
      4.

      依次把糯米、姜丝、冰糖放入电压力饭锅内胆，加入水，水量到“粥”刻度的“0.5～0.7”位置。

      |
      |
      5.

      开始高压锅自动烹调，完成后待稍凉盛出食用。
