      营养蔬菜粥的做法

      适合所有年龄层的一款营养粥，特别适合6岁以上小孩。

      一、主料

      主料用量
      大米2杯

      二、辅料

      辅料用量
      春菜200克
      玉米一个
      胡萝卜小的一个
      瘦肉200克
      猪骨头300克
      盐适量

      三、做法

      步骤
      1.

      把猪骨洗净，然后放水煮开，把水倒了，加入清水重新熬煮。

      |
      |
      2.

      骨头汤大概熬一个小时

      |
      |
      3.

      清洗蔬菜

      |
      |
      4.

      把菜洗好后跺碎

      |
      |
      5.

      萝卜洗净，削皮

      |
      |
      6.

      切条，然后剁碎。

      |
      |
      7.

      把玉米掰开，洗净

      |
      |
      8.

      把瘦肉沫加点水加点油搅拌均匀，等下煮的时候不会成团。

      |
      |
      9.

      骨头汤熬好后，把米，萝卜和玉米加进汤里煮。如果家里有小孩，就把骨头汤的汤倒进另一个锅煮，只倒汤，不下骨头预防粥里有小骨头。

      |
      |
      10.

      加了玉米，大米，萝卜的粥。

      |
      |
      11.

      大概煮10多分钟的样子

      |
      |
      12.

      大概煮20分钟，加入肉末。

      |
      |
      13.

      加肉末后煮大概10分钟，加入青菜。

      |
      |
      14.

      加入青菜后，大概煮十分钟左右，粥就煮好了。

      |
      |
      15.

      煮好的粥

      |

      四、功效

      适合婴儿，强身健体。
