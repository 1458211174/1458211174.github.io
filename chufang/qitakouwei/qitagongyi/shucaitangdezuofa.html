      蔬菜汤的做法

      有土豆、胡萝卜和西红柿做的汤很开胃哦

      一、主料

      主料用量
      土豆1个
      胡萝卜1/2棵
      西红柿1个
      芹菜1根

      二、辅料

      辅料用量
      水适量
      盐适量
      番茄酱2-3汤匙
      花生油适量

      三、做法

      步骤
      1.

      准备食材

      |
      |
      2.

      土豆、胡萝卜、芹菜、西红柿切丁。

      |
      |
      3.

      热锅加适量油，把西红柿炒到基本成酱，放入胡萝卜、土豆、芹菜翻炒一下。

      |
      |
      4.

      加入适量的水，我是喜欢把土豆炖的软软的，所以水假的稍微多些，直到基本收汤，加入适量番茄酱，翻炒几下就可以出锅了。

      |
      |
      5.

      红红黄黄的很好看吧，味道也很好哦！

      |
      |
      6.

      配上一份主食，一份早餐就做好了

      |

      四、功效

      美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，抵抗力，降糖，保护视力，助消化，防癌，高血压，消食，祛斑，延年益寿，血栓，肝火，镇静助眠，贫血，防皮肤病。
