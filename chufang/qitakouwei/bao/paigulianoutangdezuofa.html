      排骨莲藕汤的做法

      地地道道的北方人，却爱上了湖北菜，喜欢煲一锅汤，在北方寒冷的天气里暖暖的喝上一碗。

      一、主料

      主料用量
      排骨500克
      莲藕1节

      二、辅料

      辅料用量
      盐适量
      鸡精适量
      枸杞20颗

      三、做法

      步骤
      1.

      排骨洗净备用 莲藕切块 用清水泡2个小时

      |
      |
      2.

      先把排骨放在开水里焯一下 焯掉血水

      |
      |
      3.

      焯好的排骨和莲藕一起倒进砂锅里 加上清水 精盐 小火煲90分钟左右

      |
      |
      4.

      煲到莲藕颜色变成淡粉色以后加入枸杞 鸡精 接着在煲1分钟就可以关火出锅了

      |

      四、功效

      补肾，润肺，防治胃病，贫血，开胃消食，治流鼻血。
