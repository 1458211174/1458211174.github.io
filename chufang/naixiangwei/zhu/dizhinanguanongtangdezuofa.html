      低脂南瓜浓汤的做法

      可能天气冷了的缘故，最近爱上了 喝浓汤。前两天用西兰花和淡奶油等做了蔬菜浓汤，小铥同学评价说：“入口是好喝的，让味蕾瞬间觉醒的，但喝多了会觉得有一点点腻，还是爱不放淡奶油的浓汤。”好吧，这次就遵从他的意见来一款低脂南瓜浓汤。 南瓜浓汤我主要用到了老南瓜和土豆，土豆最好选择色泽偏黄的那种，口感比较绵密，和粉粉糯糯的老南瓜比较搭，还省去了加面粉调整浓汤浓稠度的步骤。至于材料中的高汤，如果家里没有的话完全可以直接用水代替，口味影响不是太大。 冬日早上，来一碗热乎乎的南瓜浓汤，让身体瞬间暖和起来，还可以缓解肠胃不适，排毒养颜，一举三得！

      一、主料

      主料用量
      南瓜320克
      土豆1个
      洋葱1/4个
      牛奶200毫升
      高汤300毫升

      二、辅料

      辅料用量
      黄油8克
      盐0.5克
      黑胡椒0.5克

      三、做法

      步骤
      1.

      材料图。

      |
      |
      2.

      南瓜（去皮瓤）、土豆（去皮）、洋葱洗净后切丁。

      |
      |
      3.

      平底锅烧热后下黄油烧至融化，然后下洋葱炒出香味。

      |
      |
      4.

      下土豆、南瓜，调入盐和黑胡椒碎，翻炒至土豆丁半透明。

      |
      |
      5.

      调入牛奶、高汤，煮开后关火。

      |
      |
      6.

      将所有材料转至破壁料理机的搅拌杯中，选择“浓汤”功能，按下启动键。

      |
      |
      7.

      待破壁机发出“滴滴”的提示音后好喝的南瓜浓汤就做好了。

      |

      四、功效

      防癌，治胃病，降糖，治前列腺，美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，抵抗力，消食，降血压，补钙，延年益寿，镇静助眠，益智。
