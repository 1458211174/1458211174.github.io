      腊鸭萝卜汤的做法

      广东人的快手菜

      一、主料

      主料用量
      腊鸭腿150克
      白萝卜250克

      二、辅料

      辅料用量
      白胡椒少许
      香菜少许
      姜片15克

      三、做法

      步骤
      1.

      白萝卜滚花切好备用，腊鸭腿切成块

      |
      |
      2.

      忘记拍照片了，具体就是烧热锅了以后将腊鸭腿放进去炒香，我是广东人，不怎么爱吃油腻的食物，所以用的是不粘锅，所以，这里就不解释啦

      |
      |
      3.

      不好意思，也没有照片，满手是油……具体也是简单的很。把萝卜和腊鸭腿还有姜片一起放进锅，姜片是去腥的，别忘了放白胡椒进去，多少就由你们自己定啦！

      |
      |
      4.

      高压锅果然是好！十几分钟后就可以开锅吃了，喜欢香菜的可以放点将香味带出来哦！

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体。
