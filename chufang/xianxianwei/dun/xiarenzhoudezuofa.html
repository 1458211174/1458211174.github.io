      虾仁粥的做法

      一阴天下雨身体就不舒服，不舒服的时候喝一碗暖粥下肚再好不过，不油腻好消化又有营养，因为太懒所以用了懒人做法，适合喜欢偷懒的人群哦～～

      一、主料

      主料用量
      大米200克
      虾仁10个
      生菜适量
      香菇三个

      二、辅料

      辅料用量
      盐适量
      芝麻油适量

      三、做法

      步骤
      1.

      准备材料，香菇先泡发，米泡20分钟，两种食材准备好之后洗净蔬菜和虾仁

      |
      |
      2.

      白米放4-8倍水（看你喜欢的口感）后加入高压锅中，选煮粥，其实在水里面我还加了一汤勺的泡香菇的水，因为我喜欢香菇的味所以用来提鲜

      |
      |
      3.

      趁着粥还煮着先切蔬菜和虾仁，我喜欢粥里面有碎的虾肉，所以也会把虾仁切碎，这样口感更好，这次我用的熟的虾人所以不用研制，如果用生虾仁请在下锅前用料酒葱姜腌制20分钟

      |
      |
      4.

      白米粥煮好以后换砂锅，加蔬菜香菇虾肉和盐，中火煮十分钟，注意搅拌

      |
      |
      5.

      喜欢不同口味的可以加调料进去，我加了芝麻油进去，也可以再加些胡椒粉，加完调味料以后煮两分钟就可以出锅了

      |
      |
      6.

      装碗后添上洗洗的榨菜丝进去，可以开动啦

      |

      四、功效

      适合婴儿，强身健体，减肥，助睡眠，利尿消肿，抵抗力，延年益寿，防癌，降压，降血脂，降胆固醇。
