      霸王超市| 什锦早餐粥的做法

      忙碌的上班族，没时间做早餐是常有的事，一上午的饥肠辘辘，不仅影响工作效率，久而久之胃痛也是常有的事，快试试这款快手什锦早餐粥吧，美味可口，关键是超级省事，5分钟轻松搞定，出门前带上一杯，一上午都能量满满。

      一、主料

      主料用量
      燕麦片25克
      纯牛奶200克
      饼干碎50克
      冻干草莓干15克
      冻干黄桃干15克
      酸奶150克

      二、辅料

      辅料用量
      白砂糖10克

      三、做法

      步骤
      1.

      准备好所需原料，

      |
      |
      2.

      燕麦片加入牛奶中拌匀，在杯子口上盖上保鲜膜，上面用牙签戳几个小洞，放入微波炉高火加热2分钟，

      |
      |
      3.

      取出后加入白砂糖搅拌至融化，

      |
      |
      4.

      然后取早餐杯，一层燕麦牛奶，一层饼干碎，一层酸奶，一层冻干黄桃干和草莓干，如此重复直到装满，盖上盖子就可以美美的带着上班去了，这样一杯下肚，一上午都有好心情！

      |

      四、功效

      防癌，镇静助眠，美容养颜，减肥瘦身，抗衰老，促消化，便秘。
