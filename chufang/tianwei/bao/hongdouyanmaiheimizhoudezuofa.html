      红豆燕麦黑米粥的做法

      一直喜欢红豆的特殊香气，有关红豆的美食总让我欲罢不能，最简单的莫过于煮一锅红豆沙。家里有袋燕麦米老久了都没有消化完，于是在一次煮红豆汤的时候加了一把燕麦米下去，煮出来的口感出奇地合胃口，口里充满了红豆的绵软和燕麦米的弹牙，吃了就忘不了的节奏啊！这次我又加了些黑米红枣和山药进去一锅混煲，营养更全面。

      一、主料

      主料用量
      红豆70克
      燕麦米20克
      黑米20克
      和田枣几颗
      山药半根
      水500克

      二、辅料

      辅料用量
      红糖适量

      三、做法

      步骤
      1.

      红豆、燕麦米、黑米洗净提前一天晚上泡好

      |
      |
      2.

      红枣去核，山药削皮和昨晚泡的材料一起放锅里加水和红糖

      |
      |
      3.

      高压锅压50分钟，喜欢吃软烂一点的可以适当延长时间，刚开始我是压了30分钟，红豆还不绵软又加压了20分钟

      |
      |
      4.

      这是压了30分钟的

      |
      |
      5.

      后来加压了20分钟

      |

      四、功效

      治流鼻血，美容养颜，防癌，保护心脏，心血管病，滋肾益精，降血糖，延年益寿，助消化。
