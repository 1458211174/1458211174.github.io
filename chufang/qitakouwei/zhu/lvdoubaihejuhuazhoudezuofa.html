      绿豆百合菊花粥的做法

      百合补中益气，性味甘微苦，能润肺止咳，清心安神，绿豆清热解毒作用，绿豆的豆香混合了百合和菊花的清香，会让你胃口大开哟。

      一、主料

      主料用量
      绿豆100克
      百合(干)30克
      菊花5克

      二、辅料

      辅料用量
      黄冰糖30克
      水600毫升

      三、做法

      步骤
      1.

      准备食材；

      |
      |
      2.

      此步骤用料步骤用时：3分钟
      绿豆100克

      绿豆洗净用冷水浸泡5小时或放入冰箱冷冻结冰备用；

      |
      |
      3.

      此步骤用料步骤用时：3分钟
      百合(干)30克
      菊花5克

      百合用冷水浸泡4-5小时，菊花洗净；

      |
      |
      4.

      此步骤用料步骤用时：1分钟
      绿豆100克
      水600毫升

      把绿豆放入锅中，加入水按下煮粥键；

      |
      |
      5.

      此步骤用料步骤用时：5分钟
      百合(干)30克
      菊花5克
      黄冰糖30克

      煮至20分钟后加入百合，菊花，冰糖，待自动提示煮好后即可。

      |
      |
      6.

      |

      四、功效

      清热解毒，抗菌抑菌，降血脂，保护肾脏，抗过敏，养阴润肺，降压，保护视力，健脑。
