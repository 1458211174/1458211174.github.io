      情人粉～～甜香玫瑰馅汤圆的做法

      都说女儿是爸爸前世的小情人，是妈妈贴心的小棉袄

      一、主料

      主料用量
      糯米粉300克
      汤圆馅儿（自制淡甜香）适量
      纯净水适量

      二、辅料

      辅料用量
      草莓粉适量，汤圆皮着色用

      三、做法

      步骤
      1.

      将糯米粉倒入容器内，锅内倒入水（可适量多一些，之后煮汤圆时可以用）待水开后，在缓慢的将水加入糯米粉内时，要用一只筷子将糯米粉搅成面絮状时就不用加水了，接着只需不断揉和成稍软但不粘手的面团，附上保鲜膜备用

      |
      |
      2.

      我用的自制草莓粉，将和好的面团取1/3与草莓粉混合均匀，草莓粉可以一点点的加，颜色深浅取决于放入的粉粉量。妈妈不怎么喜欢吃抹茶和巧克力，所以这次没有多彩的外皮，只有绵白里的那一抹红了……如果家里有其他颜色的粉粉，那就尽情释放你们的艺术魅力吧！

      |
      |
      3.

      接下来只需要按自己需要的汤圆大小取面团加入甜美的玫瑰馅包成圆形、红鼻小猪

      |
      |
      4.

      月牙状的最多了，因为……省时省力简单易做啊

      |
      |
      5.

      红鼻小猪

      |
      |
      6.

      花

      |

      四、功效

      抵抗力，健脾。
