      燕麦银耳莲子粥的做法

      燕麦，位全球健康谷物第一名。麦香米蛋白质含量平均高达15.6%，8种氨基酸组成平衡，维生素和磷、铁等物质含量丰富。燕麦还是预防动脉粥样硬化、高血压、冠心病的理想食物。它含丰富的亚汕酸，占全部不饱和脂肪酸的35%-52%，对糖尿病、脂肪肝、便秘、浮肿等有辅助疗效，对中老年人增进体力，延年益寿大有裨益。加上银耳、莲子、枸杞一起煮成糖水，营养更加一层。

      一、主料

      主料用量
      银耳20克
      燕麦米40克
      莲子30克
      枸杞10克

      二、辅料

      辅料用量
      白糖10克

      三、做法

      步骤
      1.

      准备好食材

      |
      |
      2.

      莲子洗干净，燕麦米掏洗干净，分别浸泡10分钟左右

      |
      |
      3.

      银耳加水泡发

      |
      |
      4.

      摘去根部不能食用部分，然后把泡发好的银耳撕成小块

      |
      |
      5.

      把洗干净后的燕麦米、莲子、银耳，放入到电饭锅内胆里

      |
      |
      6.

      加入适量的水

      |
      |
      7.

      选择“煲粥”功能键，启动电饭锅开始工作

      |
      |
      8.

      枸杞过水清洗一下，捞起备用

      |
      |
      9.

      待电饭锅进入倒计时5分钟时，加入枸杞，搅拌一下，继续煮，直至程序结束进入保温状态

      |
      |
      10.

      程序结束后，根据个人口味加入适量的白糖调味，即可食用

      |
      |
      11.

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体，治流鼻血，防癌，降压，安神，益精，保护视力，保肝，降三高。
