      生滚鱼片粥的做法

      味道鲜美，入口软糯，营养暖身的好粥

      一、主料

      主料用量
      大米半杯
      龙利鱼柳100克
      小葱2棵
      大叶生菜2片
      鲜香菇2个

      二、辅料

      辅料用量
      姜丝2克
      白胡椒粉1茶勺
      料酒2汤匙
      食盐1茶勺
      香油1茶勺

      三、做法

      步骤
      1.

      大的是汤匙，小的是茶勺。

      |
      |
      2.

      用电饭煲配的量杯取半杯大米，放入碗中，加入清水，淋入香油1茶勺，浸泡半小时以上。

      |
      |
      3.

      超市买来的龙利鱼柳，解冻后清洗干净。

      |
      |
      4.

      用刀将鱼柳斜着片成薄片，装入碗中备用。

      |
      |
      5.

      生姜两片切成姜丝备用。

      |
      |
      6.

      鱼肉中加入切好的姜丝，加入白胡椒粉半茶勺，食盐半茶勺，料酒2汤匙，抓匀以后腌制半小时以上。

      |
      |
      7.

      将泡米水倒掉，泡好的大米加入电饭煲。

      |
      |
      8.

      加入清水，水量按刻度加到半杯米熬粥的地方，按下熬粥程序开始熬粥。

      |
      |
      9.

      香菇用清水浸泡十分钟，之后剪去根部洗净。

      |
      |
      10.

      香菇切薄片备用。

      |
      |
      11.

      取大叶生菜的两片叶子，洗净撕碎备用。

      |
      |
      12.

      熬粥程序进行一小时后打开锅盖，加入腌制好的鱼肉，腌制的汤汁和姜丝一起加入。

      |
      |
      13.

      加入切好的香菇片。

      |
      |
      14.

      加入半茶勺食盐，半茶勺白胡椒粉，搅拌均匀后盖盖继续熬制。

      |
      |
      15.

      继续熬制半小时后加入撕碎的生菜叶，轻轻搅拌一下。

      |
      |
      16.

      小香葱摘洗干净后切碎备用。

      |
      |
      17.

      熬粥完成时加入香葱碎末，即可开始享用了。

      |

      四、功效

      适合婴儿，强身健体，去火，清肺化痰，促进食欲，抵抗力，防癌，助消化。
