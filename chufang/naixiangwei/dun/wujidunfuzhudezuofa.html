      乌鸡炖腐竹的做法

      乌鸡是滋补的好食材。乌鸡炖腐竹由乌鸡、洋葱、西兰花、胡萝卜、土豆、腐竹、奶炖小方等食材组成。能吃饱还不会长胖，还能使得冬天不冷。这么好的一道菜，操作非常简单。

      一、主料

      主料用量
      乌鸡300克
      洋葱200克
      西兰花100克
      胡萝卜100克
      土豆100克
      腐竹100克

      二、辅料

      辅料用量
      橄榄油2毫升
      奶炖小方块1盒

      三、做法

      步骤
      1.

      准备食材：乌鸡洗净切块、洋葱洗净切块、西兰花洗净切开、胡萝土豆卜去皮洗净切块、腐竹温水泡软

      |
      |
      2.

      热锅，倒入橄榄油，放入洋葱煸炒至透明；

      |
      |
      3.

      加入乌鸡块，煸炒至熟；

      |
      |
      4.

      放入土豆翻炒变软；

      |
      |
      5.

      加入胡萝卜，翻炒；

      |
      |
      6.

      加入适量的清水，水量没过食材即可；

      |
      |
      7.

      大火煮沸之后放入腐竹；

      |
      |
      8.

      再放入西兰花；奶炖小方，6块全放入，搅拌至完成溶解；

      |
      |
      9.

      转为小火炖至浓稠状即可。

      |

      四、功效

      延年益寿，预防贫血，抵抗力，消食，降血压，补钙，防癌，瘦身减肥，促进生长，软化血管，清热解渴，抗衰老，强身健体，降糖，保护视力，助消化，美容瘦身，美容养颜，益智安神，消除水肿，延缓。
