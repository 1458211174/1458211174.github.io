      砂锅炖酸萝卜老鸭汤的做法

      春夏季的必备汤水，清热去火，不喝你就亏大了！ 春季天干物燥，万物生发，容易耗伤阴血，所以要多喝汤水来滋补一下。鸭肉滋阴凉血，特别适合阴虚火旺的体质。酸萝卜老鸭汤最适合春夏食用了，老鸭汤中放入四川泡菜，汤变成酸味，特别可口开胃，春季容易肝火旺，酸味入肝，老鸭滋阴凉血有很好的补益作用！特别是经常熬夜的朋友，更要煲些来喝，否则阴血耗伤过重，再调就难了，平时要多注意食补，多爱惜自己！ 煲汤最好选用砂锅，可以保留食材的原汁原味，营养不流失。两年的老鸭，没有肥油，用坤博砂锅小火慢炖3个小时，老鸭的营养成分充分释放到汤中，非常美味！

      三、做法

      步骤
      1.

      老鸭半只洗净，斩块，入冷水锅，加料酒焯水

      |
      |
      2.

      酸菜、酸豇豆切段，姜切片

      |
      |
      3.

      锅中油烧热，放入姜片爆香

      |
      |
      4.

      酸萝卜、酸豇豆、酸菜入锅翻炒两分钟

      |
      |
      5.

      砂锅中放入足够的开水，放入炒过的酸菜、老鸭、料酒煮沸，

      |
      |
      6.

      盖上盖子小火煲3个小时即可

      |
      |
      7.

      可以配上米线，在老鸭汤快煮好的时候，云南米线下锅煮8分钟，闷8分钟，捞出过凉水，加入老鸭汤即可

      |
