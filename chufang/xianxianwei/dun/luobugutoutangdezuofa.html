      萝卜骨头汤的做法

      简单的食材，不简单的味道。 仅仅发挥食材本身的味道就很了不起！

      一、主料

      主料用量
      猪骨头400克
      白萝卜400克
      甜玉米1个
      海带丝40克

      二、辅料

      辅料用量
      姜适量
      食盐适量
      鸡精适量
      料酒2匙

      三、做法

      步骤
      1.

      添加姜、料酒，不停的撇沫

      |
      |
      2.

      切萝卜块

      |
      |
      3.

      玉米切块

      |
      |
      4.

      捞出姜片，然后加萝卜和玉米

      |
      |
      5.

      大火煮开转小火慢慢炖

      |
      |
      6.

      并加适量食盐和鸡精

      |
      |
      7.

      哒哒哒

      |
