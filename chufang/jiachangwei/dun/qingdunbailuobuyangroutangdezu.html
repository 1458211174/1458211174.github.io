      清炖白萝卜羊肉汤的做法

      羊肉性温，冬季常吃羊肉，不仅可以增加人体热量，抵御寒冷，而且还能增加消化酶，保护胃壁，修复胃粘膜，帮助脾胃消化，起到抗衰老的作用，清炖白萝卜羊肉汤是由羊肉、白萝卜配上葱姜等作料，加入适量的调味料炖制而成。白萝卜羊肉汤营养丰富，制作简单，具有补肾壮阳，健脾益胃，提高体能之功效

      一、主料

      主料用量
      羊肉350克
      葱适量
      白萝卜半个
      生姜几片
      香菜适量

      二、辅料

      辅料用量
      鸡粉适量
      料酒适量
      盐适量

      三、做法

      步骤
      1.

      羊肉切好放入水里浸泡两个小时，中间要换水，主要是去血水，浸泡好的羊肉洗净，起锅放入羊肉，生姜，料酒，加入没过羊肉的水量

      |
      |
      2.

      将羊肉大火烧开，焯一下去掉血水，捞起来，用冷水冲洗干净

      |
      |
      3.

      砂锅里先放入适量的水烧开，放入羊肉，葱，生姜，料酒，大火烧开 后转小火炖半个小时，因为我切得不是很大块所以半个小时的时间就可以把肉炖软了

      |
      |
      4.

      半个小时后放入白萝卜块，再炖半个小时，这样把肉就可以炖得很软了，炖好的汤加入调料一点盐和鸡粉，这样就算好

      |
      |
      5.

      出炉后放上一点点葱花和一点点香菜花，一碗鲜美的白萝卜羊肉汤就这样完成了这样一份香甜美味的白萝卜羊肉汤真的是暖到心窝里了

      |

      四、功效

      健胃，温补肝肾，温经止痛，保护视力，抵抗力，促进食欲，老年痴呆，防癌，感冒，降三高，延年益寿，消食，保护肠胃，预防疾病，强身健体，预防感冒，散瘀止疼，治肠炎，开胃，预防中暑，缓解腹。
