      萝卜老鸭汤的做法

      家养的老鸭就是鲜美，醇醇的汤汁，还有自家种的萝卜的清甜味，非常美味的大补汤哦～～

      一、主料

      主料用量
      鸭250克
      萝卜1个

      二、辅料

      辅料用量
      盐适量
      料酒1茶匙
      姜3片

      三、做法

      步骤
      1.

      先将老鸭剁成块，放入冷水锅中，焯烫一下。

      |
      |
      2.

      苏泊尔养生壶中加入适量清水，放入焯烫好的鸭肉。爱喝汤的话，可以适当多加一些清水。

      |
      |
      3.

      加入姜片，倒入适量料酒，开启养生汤模式，定时120分钟。

      |
      |
      4.

      将萝卜洗净，去皮切块待用。

      |
      |
      5.

      待养生汤模式还剩40分钟时，下入萝卜块，继续煮。

      |
      |
      6.

      最后2分钟时，加入适量盐调味即可。

      |

      四、功效

      消食，保护肠胃，预防疾病，强身健体。
