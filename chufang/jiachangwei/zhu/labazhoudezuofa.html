      腊八粥的做法

      农历十二月初八俗称腊八，也称之为腊八节，古人在这天会祭祀祖先和神灵、祈求来年丰收吉祥。现代很多地区习惯在这一天喝腊八粥。 腊八粥根据地区或家人的口味不同，制作过程中用料也有所不同，主要原材料以米类为主，再辅以豆类和适量的干果或坚果，制作出一款既营养又美味的粥。这款粥除腊八节喝外，平时也可以经常煮，粗细粮精心搭配制作出来的粥不光口感佳，还是一道养生粥，适合各种人群食用，对于脾胃不好的人群，在寒冷的季节喝上一碗还能养胃。 参考份量：5人份

      一、主料

      主料用量
      大米1/3杯
      糙米1/8杯
      糯米1/5杯
      红豆1/2杯
      百合20瓣
      花生米20粒
      红薯半个
      红枣5颗

      二、辅料

      辅料用量
      清水适量

      三、做法

      步骤
      1.

      准备好食材

      |
      |
      2.

      红豆用清水冲洗过后，加适量水放电压力锅中浸泡半小时左右

      |
      |
      3.

      红薯洗净随意切滚刀块，大米、糯米、糙米、百合、花生米、红枣等混合在一起清水洗过水后连同红薯一起放入电压力锅中，再加入适量水

      |
      |
      4.

      盖上压力锅盖子，接通电源，功能菜单中选择“煮粥”，口感上选择“软烂”，然后按“启动”

      |
      |
      5.

      50分钟后腊八粥制作完成

      |

      四、功效

      适合婴儿，强身健体，脂肪肝，散瘀止痛，治贫血，助睡眠，减肥瘦身，防癌，开胃，心血管病，防过敏，糖尿病，增强体质，健脾胃，缓解尿频，清热去火，安神美容，抵抗力，通便，抗衰老，治流鼻血。
