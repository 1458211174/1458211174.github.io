      蘑菇鸡蛋汤的做法

      蘑菇具有化痰止咳，通便，提高免疫力，抑癌抗瘤的功效。蘑菇鸡蛋汤营养价值很高，做起来很简单。我经常做着吃

      一、主料

      主料用量
      蘑菇500克
      鸡蛋2个
      油菜1小把

      二、辅料

      辅料用量
      花生油10克
      葱少许
      生抽3克
      盐半勺
      花椒5粒

      三、做法

      步骤
      1.

      清洗蘑菇，油菜，用开水焯一下。

      |
      |
      2.

      热锅凉油，放入花椒，葱，出香味后放入蘑菇。翻炒一会后放入水炖会。

      |
      |
      3.

      把搅拌好的鸡蛋倒入锅中成型后，再放入油菜。

      |
      |
      4.

      出锅前放入酱油，盐。盛入汤碗中待用。

      |

      四、功效

      散瘀止疼，安神助眠，止咳化痰，通便排毒，促进食欲，抵抗力，健脑益智，保肝，动脉硬化，防癌，延年益寿，美容护肤，脂肪肝，排毒，补钙。
