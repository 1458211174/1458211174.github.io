      adcdef汤的做法

      冬日里的一道暖心汤，富含丰富的维生素，营养又健康，大爱。

      一、主料

      主料用量
      土豆3个
      胡萝卜1个
      西红柿2个
      金针菇1把
      牛肉卷300克

      二、辅料

      辅料用量
      葱10克
      姜10克
      韩国甜辣酱2勺（个人口味）
      盐（个人口味）
      白砂糖（个人口味）

      三、做法

      步骤
      1.

      土豆切片（一定要厚），胡萝卜、西红柿、青椒切块，牛肉圈圈上金针菇。葱姜切丝备用。

      |
      |
      2.

      葱姜下锅炒香。

      |
      |
      3.

      放入胡萝卜，土豆翻炒，翻炒至表皮微变色

      |
      |
      4.

      放入西红柿，青椒，加入水（水量超过食材）

      |
      |
      5.

      放入糖，盐，辣酱，少许醋，开炖～，炖约20分钟，土豆出沙即可

      |
      |
      6.

      放入牛肉金针菇卷，炖5分钟，撒上少许鸡精或味精即可出锅。

      |
      |
      7.

      可以吃啦

      |

      四、功效

      美容瘦身，美容养颜，益智安神，消除水肿，延缓衰老，抵抗力，降糖，保护视力，助消化，防癌，高血压，消食，祛斑，延年益寿，血栓，降血脂，过敏。
