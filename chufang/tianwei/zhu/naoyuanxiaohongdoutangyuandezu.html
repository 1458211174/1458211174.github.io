      闹元宵，红豆汤圆的做法

      又是一年元宵节，元宵佳节最爱的当然是汤圆。今天推荐一款连汤圆汤都想多喝几碗的红豆汤圆，汤圆馅是浓香丝滑的花生酱，煮汤圆的汤是用红豆熬煮一小时而得。健康的祛湿，美味红豆汤圆。

      一、主料

      主料用量
      糯米粉200克
      红豆100克

      二、辅料

      辅料用量
      白砂糖60克
      花生酱100克

      三、做法

      步骤
      1.

      准备食材：糯米粉、红豆、白砂糖、花生酱；

      |
      |
      2.

      红豆浸泡洗净，熬煮一小时，煮成红豆粥；

      |
      |
      3.

      利用熬煮红豆粥的时间，把白砂糖加适量清水煮沸，倒入糯米粉中拌均揉成光滑米团；

      |
      |
      4.

      分成小团，压扁，中间包花生酱，锁紧口；PS：不能用擀面杖擀，糯米太黏，只能用手压扁在放花生酱。

      |
      |
      5.

      做好的圆滚滚的漂亮花生味汤圆；PS：馅可以选择其他口味入黑芝麻。

      |
      |
      6.

      适量的清水，放入一点点白砂糖煮沸，放入汤圆煮熟；

      |
      |
      7.

      把煮好红豆粥加入，小火熬煮1分钟即可。

      |

      四、功效

      抵抗力，健脾。
