      【冬瓜鱼丸汤】冬季暖身必备的做法

      寒冷的天儿，最希望能有一碗热气的汤羹来暖身。而这碗冬瓜鱼丸汤便是最好的选择，低脂还鲜美。

      一、主料

      主料用量
      鱼丸200克
      冬瓜150克
      海米10克
      葱1段
      姜2片

      二、辅料

      辅料用量
      芝麻油5克
      胡椒粉1克
      盐1克

      三、做法

      步骤
      1.

      备好所需食材；冬瓜去皮切片，洗净，海米提前一小时浸泡；

      |
      |
      2.

      打开美的电饭煲，依次加入鱼丸、冬瓜、海米、葱、姜、并加入足量的水；

      |
      |
      3.

      盖上电饭煲盖子，启动蒸煮程序；

      |
      |
      4.

      20分钟后，电饭煲停止工作，调入芝麻油、盐、胡椒粉即可。

      |

      四、功效

      甲状腺，下奶，清热解毒，减肥瘦身，滋养肾气，化痰，防癌，润肤美容，降血糖，降胆固醇，降血脂，软化血管，助减肥，降压，保护血管，骨质疏松，安神助眠，助睡眠，促进食欲，老年痴呆，抵抗力。
