      酸爽开胃汤的做法

      这几天没有胃口了！今天就做了这个金针菇番茄汤，真的好好喝哟！

      一、主料

      主料用量
      肉丸子半碗
      番茄两个
      小香葱五颗
      香菜一颗

      二、辅料

      辅料用量
      油适量
      生抽两汤勺
      耗油两汤勺
      盐适量
      鸡精一汤勺
      淀粉一汤勺

      三、做法

      步骤
      1.

      准备好的食材

      |
      |
      2.

      把金针菇切去根部冲洗干净，姜蒜切粒番茄切成小块

      |
      |
      3.

      锅中油热时加入姜蒜粒爆香

      |
      |
      4.

      接着加入番茄块大火翻炒均匀

      |
      |
      5.

      把番茄汁完全炒出来

      |
      |
      6.

      加入适量的热水烧开

      |
      |
      7.

      加入肉丸子烧熟

      |
      |
      8.

      加入耗油和生抽搅拌均匀

      |
      |
      9.

      接着加入金针菇烧开

      |
      |
      10.

      煮菜的同时把淀粉用水调开

      |
      |
      11.

      把调好的淀粉水加入大火烧开就可以了

      |
      |
      12.

      撒上鸡精和盐调好味道，加入葱段和香菜段就可以享用了！

      |
      |
      13.

      颜色诱人的番茄汤酸酸的很可口哟！

      |

      四、功效

      消食，感冒，祛除寒气。
