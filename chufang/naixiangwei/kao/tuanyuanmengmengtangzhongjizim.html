      团圆萌萌汤种鸡仔面包的做法

      2017年鸡年“鸡”祥

      一、主料

      主料用量
      汤种：高粉18克
      主面团:高筋面粉300克
      汤种：水60克
      鸡蛋1个（去壳51克）
      水130克
      奶粉20克
      细砂糖40克
      黄油15克
      酵母4克
      盐3克

      二、辅料

      辅料用量
      红曲粉2克

      三、做法

      步骤
      1.

      先把汤种制作好，水和高粉混合至看不见干粉

      |
      |
      2.

      放在煤气灶上小火加热，一边不停的搅拌至浓稠状态，离火，放凉即可使用

      |
      |
      3.

      启动面包机揉面程序，我选了揉面20分钟

      |
      |
      4.

      第1个20分钟揉好让面团静止10分钟，这一步是为了让面团更容易出膜

      |
      |
      5.

      静止好之后加入软化的黄油，再揉20分钟

      |
      |
      6.

      适时地检查面筋的强度，把面团小心的伸开，这时候面团已经可以形成一层薄薄的薄膜，但是薄膜并不是特别坚韧，还是比较容易破裂，破裂的洞口呈不规则的形状，而不是光滑的圆形，这个时候面团就达到了扩展阶段，这个时候做餐包是可以的了
      我是选择直接揉至如图扩展状态，破洞口处为光滑的圆形

      |
      |
      7.

      揉好的面团，裹圆，盖上保鲜膜
      室温发酵30分钟左右有1.5倍大（发酵不一定要按照时间来，这要看实际温来判断）

      |
      |
      8.

      发酵好的面团，用手机粘点面粉戳一个洞不回缩就是发酵好了

      |
      |
      9.

      把面团擀开排气

      |
      |
      10.

      这个方可做3个6寸模的面包先分割两个10克的面团出来，（如果只是做圆形的就分割一个10克的面团出来加红曲粉就好，我的是要做其它的造型所以才分割两个）

      |
      |
      11.

      一个小面团加入2克的红曲粉

      |
      |
      12.

      把分割好的面团揉圆码在中空模里

      |
      |
      13.

      把红曲面团擀开，用小模具按压出自己喜欢的形状，白色的面团我做了母鸡的翅膀还有小花

      |
      |
      14.

      如果没有小模具的就，把面团搓成长条状，搓成小圆形用刮板在圆面团中间切一刀，整形成鸡冠

      |
      |
      15.

      如图，这时先把烤箱上下火150度预热10分钟

      |
      |
      16.

      把按压好的红曲面团装饰在发酵好的面团上

      |
      |
      17.

      送进烤箱上火120度下次130度烤20分钟即可

      |
