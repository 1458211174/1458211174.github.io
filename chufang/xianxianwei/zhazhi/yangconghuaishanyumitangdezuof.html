      洋葱淮山玉米汤的做法

      这是一道咸味的汤，跟南瓜浓汤做法类似，没有淡奶油，如果喜欢，可把清水换成牛奶做奶香浓汤。

      一、主料

      主料用量
      玉米70克
      淮山40克
      洋葱30克
      清水1000毫升

      二、辅料

      辅料用量
      低钠盐2克

      三、做法

      步骤
      1.

      食材准备、称取，新鲜玉米清洗后，用刀把一排的玉米粒左右各划一刀，把玉米粒切下来。

      |
      |
      2.

      从切开的玉米粒旁开始，用手把玉米粒一粒粒掰下来即可。

      |
      |
      3.

      戴一次性手套或手套处理淮山，先用刨刀把皮去掉，切块再切小块。

      |
      |
      4.

      洋葱清洗，摘去最外边变黄的衣，切片，再切小块。

      |
      |
      5.

      玉米粒、淮山块、洋葱块、清水全部倒入自动豆浆机中，盖上盖，接上电源，启动米糊功能。

      |
      |
      6.

      制作完成，趁热加入低钠盐，用长柄汤勺搅拌一会让盐充分溶解，盛出享用。

      |

      四、功效

      减肥瘦身，防癌，降血压，保护视力，润肠通便，健脑，抗衰老，抵抗力，消食，补钙，延年益寿，美容养颜。
