      富含层次感的流沙汤圆的做法

      喜欢吃流沙包，突然想到，不如试试流沙汤圆。(汤圆皮每个约20克，流沙馅每个约15克，约14粒成品) （想要流沙更猛烈些，可增加流沙馅的克数，减少汤圆皮的克数。流沙馅放心做多点吧，剩下放冰箱冷藏，还可以做流沙包哦！） 1 搓时如果裂开漏馅，可以在裂开出补上一点面团，按紧，手心沾一点点温水，然后再搓圆。 2 搓汤圆皮时要用力，时间约20分钟，速冻成形后，就要收起来密封，防止水分流失。用密封袋、保鲜盒装起，再放入冰箱冷冻。 3 冷冻后的流沙汤圆才能煮，（不要搓好就煮，爆馅的）记得开水下锅。 4 汤圆吃时要小心，不要烫到口。 5 减肥大业未成功的，尽量少吃。 流沙汤圆煮熟沥干，放在椰蓉盘子滚一圈，华丽转身成为糯米糍，好吃到不要不要的

      一、主料

      主料用量
      水磨糯米粉160克
      开水140毫升（20毫升备用）

      二、辅料

      辅料用量
      咸蛋黄4个
      黄油50克
      纯牛奶（或水）20毫升
      砂糖30克
      生粉（玉米淀粉）20克
      炼奶约10克
      奶粉约15克
      吉利丁片1片约5克
      吉士粉10克

      三、做法

      步骤
      1.

      咸蛋黄在白酒里洗一下，烤箱预热5分钟180℃。烤盘铺上油纸，放咸蛋黄，烤约8分钟。

      |
      |
      2.

      用刀背把咸蛋黄压碎。

      |
      |
      3.

      吉利丁片用温水泡软后加入所用馅料。

      |
      |
      4.

      用打蛋器到30秒。

      |
      |
      5.

      放入冰箱冷冻3小时。

      |
      |
      6.

      取出，分成约14粒，每粒约15克。

      |
      |
      7.

      加开水

      |
      |
      8.

      先用筷子搅拌。

      |
      |
      9.

      用手揉搓约20分钟。

      |
      |
      10.

      搓成长条，切成约20克一块。

      |
      |
      11.

      大拇指按住中间。

      |
      |
      12.

      捏成杯状，塞入流沙馅。

      |
      |
      13.

      搓圆后，放在洒有糯米粉的盘子上。
