      香菇牛肉汤的做法

      香菇具有高蛋白、低脂肪、多糖、多种氨基酸和多种维生素的营养 牛肉健脾胃、益气血、强筋骨二者搭配实属宝宝营养菜品

      一、主料

      主料用量
      牛肉100克
      香菇6颗

      二、辅料

      辅料用量
      生姜片8块
      大蒜3颗
      葱花少许
      盐少于
      植物油少量

      三、做法

      步骤
      1.

      洗好香菇，分成小瓣儿

      |
      |
      2.

      把洗干净分小的香菇放汤碗里，加入四片姜片，烧开水，就是汤所需的水量，倒入汤碗泡着香菇

      |
      |
      3.

      把牛肉切成片，大蒜，姜，葱都准备好~~我用的是已经煮熟的牛肉，鲜牛肉做出来，味道会更鲜

      |
      |
      4.

      倒入油，放入大蒜，生姜爆香放入牛肉炒三分钟

      |
      |
      5.

      把刚刚用开水泡着的香菇，连水一起倒入国内开煮~煮20分钟，起锅之前放葱！香喷喷的汤出锅喽

      |

      四、功效

      强身健体，抵抗力，贫血，抗衰老，延年益寿，防癌，降压，降血脂，降胆固醇。
