      莲子桂圆腊八粥的做法

      腊八节喝腊八粥，所以就选用了几种食材早晨煮了这道粥。营养平衡，好喝香香滴。

      一、主料

      主料用量
      莲子(干)40颗
      桂圆13颗
      纯净水适量

      二、辅料

      辅料用量
      白芸豆20颗
      红小豆40克
      花生仁40颗
      红枣10个
      燕麦仁30克
      黑米60克
      江米40克
      小米60克

      三、做法

      步骤
      1.

      盛出红小豆和莲子。

      |
      |
      2.

      盛出白芸豆和花生仁。

      |
      |
      3.

      取红枣。

      |
      |
      4.

      盛出黑米和小米。

      |
      |
      5.

      盛出燕麦仁和江米。

      |
      |
      6.

      取出桂圆。

      |
      |
      7.

      锅里加入纯净水，将红枣、花生仁、红小豆、莲子和白芸豆洗净入冷水锅，开火煮。锅开后加入洗净的米和几滴油（防溢出）小火煮熟。

      |
      |
      8.

      煮好的粥。

      |
      |
      9.

      装碗的粥。

      |
