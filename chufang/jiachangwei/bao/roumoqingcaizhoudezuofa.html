      肉末青菜粥的做法

      粥的味道，不只是清淡，只要用心也能煲出营养美味的粥。肉末青菜粥制作方法简单，但口感香甜美味。

      一、主料

      主料用量
      瘦肉100克
      大米75克
      菜心85克

      二、辅料

      辅料用量
      食盐适量
      食用油1/2茶匙
      生抽1茶匙
      鸡粉2克
      白糖2克
      玉米淀粉少许
      胡椒粉2克

      三、做法

      步骤
      1.

      准备好食材

      |
      |
      2.

      瘦肉剥碎加入水淀粉、盐、食用油、白糖、生抽、胡椒粉、鸡粉调味

      |
      |
      3.

      把调味料抓拌均匀后，腌制1小时

      |
      |
      4.

      菜心洗净，切小粒

      |
      |
      5.

      大米洗净，放进适量的清水，选择电饭锅里的菜单按开始键

      |
      |
      6.

      粥煲至1小时左右，加入肉末

      |
      |
      7.

      最后10分钟左右，加入菜心粒，煲至断生

      |
      |
      8.

      粥煲好后，加上适量的食盐即可

      |

      四、功效

      补肾，润肺，利尿解毒，润肠通便。
