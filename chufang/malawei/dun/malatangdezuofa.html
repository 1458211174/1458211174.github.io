      麻辣烫的做法

      每次有朋友来家里吃火锅都会剩下很多调理，丸子和蔬菜，素材多种却分量小，用来做菜分量不够，丢掉了也可惜，不如做成麻辣烫，辣辣暖暖的一大碗，材料随性，好吃又省钱～

      一、主料

      主料用量
      鱼丸适量
      生菜6叶
      金针菇200克
      乌冬面一份
      海带200克
      白菜6叶
      蟹肉棒适量

      二、辅料

      辅料用量
      食盐适量
      火锅底料1瓶

      三、做法

      步骤
      1.

      剩下的麻辣火锅底料或者麻辣烫料子煮汤是最简单的，只是这些材料里都有很多花椒和辣椒，如果直接煮很影响口感，所以煮的时候把直接把料子放进过滤筛再放到锅里，水开了以后把渣滓丢掉

      |
      |
      2.

      趁着汤在煮着去洗菜，冰箱里很多边角料被我收集过来海带，金针菇，生菜，白菜，鱼丸，贡丸，蟹肉，虾仁，魔芋丝，都是吃火锅剩下的材料，同时我加了乌冬面，麻辣烫里没有主食的话是很容易饿的～

      |
      |
      3.

      汤好了就可以先下冻的丸子蟹肉和乌冬，这些不好煮的先煮，我喜欢吃煮的软乎乎的面所以也早下了进去，不喜欢吃软面的朋友可以晚些下，香菇这种不怕煮的东西也可以早下进去

      |
      |
      4.

      五分钟左右，丸子浮起来，面软了就可以放蔬菜菌菇和海带，海带是我的挚爱

      |
      |
      5.

      根据汤的味道放盐，放之前最好尝一尝，因为不同的锅底咸度也不一样

      |
      |
      6.

      菜熟了就出锅吧，喜欢酸的请放醋，喜欢麻将的请放麻将（火锅吃剩下的麻将也别浪费啦），都做好以后就开动吧～

      |

      四、功效

      甲状腺，下奶，清热解毒，减肥，助睡眠，利尿消肿，抵抗力，降血脂，过敏，利尿，防病，御寒，防癌，防甲亢，抗辐射，解渴利尿，润肠通便，消食。
