      花生猪脾汤的做法

      花生，味甘，性平；归脾、肺经。具有健脾养胃，润肺化痰的功效。 猪脾是猪的脾脏，又称联贴、猪横脷；性味甘平；能益脾胃，助消化；多用于小儿脾胃虚弱，饮食不化，食欲减退。

      一、主料

      主料用量
      新鲜花生米30克
      猪脾1条

      二、辅料

      辅料用量
      香油10克
      盐0.5克
      姜1块
      料酒10克

      三、做法

      步骤
      1.

      猪脾洗净

      |
      |
      2.

      剔除油脂

      |
      |
      3.

      锅里倒入适量清水，放姜片、料酒、猪脾焯水

      |
      |
      4.

      捞起切片，放水里洗至水清，浸泡半小时

      |
      |
      5.

      花生米洗净

      |
      |
      6.

      碗里放入洗净的猪脾、花生米，倒入适量清水，倒点茶籽油，上蒸锅大火蒸三十分钟

      |
      |
      7.

      放油盐调味，出锅即可
