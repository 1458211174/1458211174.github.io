      番茄紫菜瘦肉汤的做法

      每100克干紫菜含蛋白质25~30克，与大豆含量相当，还含有丰富的维生素等营养成分。紫菜中的蛋白质和其他营养成分容易被消化吸收，非常适合消化功能减退的老年人食用。加上瘦肉，营养更均衡，且不油腻，对老年人有延年益寿的作用。

      一、主料

      主料用量
      番茄1个
      紫菜8克
      瘦肉150克

      二、辅料

      辅料用量
      茶籽油20克
      盐1克
      姜1块

      三、做法

      步骤
      1.

      番茄、姜洗净改刀

      |
      |
      2.

      紫菜冷水泡发，洗净

      |
      |
      3.

      瘦肉洗净切薄片，放点盐搅拌均匀腌制片刻

      |
      |
      4.

      锅里倒入适量清水，放西红柿和姜片煮开

      |
      |
      5.

      放紫菜煮几分钟

      |
      |
      6.

      放瘦肉煮一分钟

      |
      |
      7.

      放油盐调味，起锅装盘即可

      |
