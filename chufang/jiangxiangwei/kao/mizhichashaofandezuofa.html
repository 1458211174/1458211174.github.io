      蜜汁叉烧饭的做法

      蜜汁叉烧饭——是一道营养丰富，色香味俱全的美味佳肴。搭配了蔬菜，给人清新喜人的感觉，那烤制好的叉烧肉味道非常的浓郁，吃起来外焦里嫩。让人越吃越爱吃~

      一、主料

      主料用量
      梅花肉500克
      青菜100克

      二、辅料

      辅料用量
      老抽1大勺
      糖少许
      蒜末少许
      米酒1勺
      叉烧酱2大勺

      三、做法

      步骤
      1.

      买回来的梅花肉洗净

      |
      |
      2.

      切成厚的大片，放入2大勺叉烧酱、1大勺老抽、糖、米酒、蒜末后

      |
      |
      3.

      拌匀腌制24小时

      |
      |
      4.

      将腌制好的梅花肉摆在烤架上。烤架放在烤盘上，烤盘上铺一张锡纸。将腌制好的生肉摆在烤架上。烤架放在烤盘上，烤盘上铺一张锡纸。先在梅花肉上刷一层蜂蜜，然后在刷一层腌肉的汁。放入预热的烤箱烤中，上下火200度，烤15分钟

      |
      |
      5.

      5分钟后取出翻面，再重复刷一层蜂蜜和一层腌肉汁继续烤15分钟就可以取出了

      |
      |
      6.

      放凉后

      |
      |
      7.

      斩片

      |
      |
      8.

      将腌制梅花肉剩下的汁倒在锅中，加适量的黄酒、蜂蜜、酱油、清水、白糖拌匀，煮开

      |
      |
      9.

      将米饭、烫熟的青菜和叉烧摆在盘上，然后再浇上煮开的酱汁，美味的蜜汁叉烧饭就可以享用了

      |

      四、功效

      软化血管，防癌，润肺化痰，强身健体，抵抗力，美容养颜，延年益寿。
