      山药莲藕鸡汤的做法

      山药质地细腻，味道香甜，含有多种营养成分，健脾益胃、助消化。藕是老幼妇孺、体弱多病者上好的食品和滋补佳珍。和鸡一起炖汤，而且放了枸杞，营养加倍哦。

      一、主料

      主料用量
      鸡腿2只
      莲藕1段
      山药1根

      二、辅料

      辅料用量
      食盐10毫升
      枸杞适量

      三、做法

      步骤
      1.

      食材准备。

      |
      |
      2.

      莲藕去皮切段。

      |
      |
      3.

      山药去皮切段。

      |
      |
      4.

      锅内倒入足量水，放入鸡腿和莲藕炖煮1小时

      |
      |
      5.

      加入山药再煮半小时。

      |
      |
      6.

      放入枸杞，加入适量盐即可。

      |

      四、功效

      抵抗力，养肾，防治胃病，贫血，开胃消食，治流鼻血，滋肾益精，降血糖，延年益寿，助消化。
