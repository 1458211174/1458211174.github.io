      豆腐脑的做法

      自家做的豆腐脑就是好喝！干净卫生！

      一、主料

      主料用量
      黄豆一斤

      二、辅料

      辅料用量
      油2汤匙
      黑木耳5朵
      酱油2茶匙
      盐2茶匙
      辣椒油适量
      老抽1茶匙
      内酯10克
      水淀粉适量
      过滤棉布1个

      三、做法

      步骤
      1.

      前一天晚上先用温水泡黄豆一斤。

      |
      |
      2.

      第二天将泡好的黄豆放入豆浆机内打磨成豆浆。

      |
      |
      3.

      将豆浆倒入过滤棉布中，过滤豆渣，留下豆浆

      |
      |
      4.

      将过滤好的豆浆放入不锈钢锅内煮开，期间要不停的搅拌，防止粘锅。

      |
      |
      5.

      将内酯用温水冲开，然后把煮开的豆浆放入盛有内酯的大碗中，静置30分钟左右

      |
      |
      6.

      将紫菜，木耳放入烧开水的锅中，再放入水淀粉勾芡，将汤汁放入豆腐脑内，新鲜美味的豆腐脑就出锅啦！

      |

      四、功效

      软化血管，益智，糖尿病，降脂，美白，防癌，脂肪肝，更年期，抵抗力。
