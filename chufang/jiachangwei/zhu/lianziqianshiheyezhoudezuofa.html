      莲子芡实荷叶粥的做法

      芡实又名鸡头米，《本草纲目》记载芡实“止渴益肾。”《神农本草经》也有记载“主湿痹腰脊膝痛。”《本草备要》记载莲子“清心除烦，开胃进食。”配上荷叶，煮出来的粥不仅养人，还口味清新哦。

      一、主料

      主料用量
      莲子60克
      芡实60克
      大米100克
      黄冰糖30克
      荷叶3克

      二、辅料

      辅料用量
      水1000毫升

      三、做法

      步骤
      1.

      食材准备好；

      |
      |
      2.

      此步骤用料
      芡实60克

      芡实用清水浸泡5-6小时或放入冰箱冷冻结冰备用；

      |
      |
      3.

      荷叶剪成小块；

      |
      |
      4.

      将剪好的荷叶、莲子、大米、芡实一起洗净；

      |
      |
      5.

      将所有食材一起放入高压锅中，加入水，盖上盖子，按下煮粥键即可。

      |

      四、功效

      防癌，降压，安神，抵抗力，益精，适合婴儿，强身健体，止咳化痰，养阴生津，润肺，清热解暑。
