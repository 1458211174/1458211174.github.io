      茶树菇乌鸡汤的做法

      茶树菇的营养价值比较高，含有人体所需的多种氨基酸，有抗衰老、降低胆固醇的功效，是心血管、高血压和肥胖症患者的最佳食品。其味道鲜美，具有补肾滋阴，健脾胃，提高人体免疫力，增强人体防病能力的功效。常食可起到抗衰老、美容等作用。 乌鸡性平、味甘，具有滋阴清热、补肝益肾、健脾止泻等作用。食用乌鸡，可提高生理机能、延缓衰老、强筋健骨，对防治骨质疏松、佝偻病、妇女缺铁性贫血症等有明显功效。

      一、主料

      主料用量
      乌鸡1/3只
      茶树菇50克
      红枣6-8粒

      二、辅料

      辅料用量
      姜片2-3片
      盐少许

      三、做法

      步骤
      1.

      准备好材料。

      |
      |
      2.

      将乌鸡焯水洗净。茶树菇洗净，加水浸泡。红枣去籽备用。

      |
      |
      3.

      锅中煮沸水，将姜片及所有材料加入，同时将浸泡茶树菇的水一起加入。

      |
      |
      4.

      大火煮20分钟后，改小火煲40分钟即可。出锅前可加少许盐调味。

      |

      四、功效

      延年益寿，预防贫血，防癌，抗衰老，治尿床，治流鼻血。
