      山药乳鸽汤的做法

      冬季适合进补。这款汤绝对的滋补，汤鲜、味美。

      一、主料

      主料用量
      乳鸽1只
      山药300克

      二、辅料

      辅料用量
      枸杞15克
      盐适量
      姜2片
      水适量

      三、做法

      步骤
      1.

      将鸽子清洗干净，剁成小块。

      |
      |
      2.

      炖锅添加适量的水，放入鸽子，姜片，烧沸。

      |
      |
      3.

      烧沸后用勺撇清泡沫，盖上盖上炖为8成熟。

      |
      |
      4.

      炖汤时间把枸杞清洗干净。

      |
      |
      5.

      鸽子汤炖8成熟放入枸杞，继续炖至鸽子软烂。

      |
      |
      6.

      炖鸽子的时间把山药清洗干净，切成小块放入锅里。

      |
      |
      7.

      依次放入适量的盐，炖至山药软烂就可以上桌开吃啦。

      |
      |
      8.

      成品图。汤很鲜美。

      |

      四、功效

      强身健体，滋肾益精，降血糖，延年益寿，助消化。
