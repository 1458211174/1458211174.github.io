      小米紫薯南瓜粥，冬季健脾养胃好粥道！--威厨艺的做法

      南瓜含有丰富维生素A、维生素E，可增强机体免疫力，对改善秋冬燥症状大有裨益，常吃能够预防哮喘、支气管炎等秋季多发病。用南瓜搭配同样有防秋燥的番薯煮成糖水，“暖笠笠”、甘甜润滑，在微寒的天气里喝上一碗，暖身又暖心！微信搜索关注“威厨房”，教你轻松烹饪健康家常大餐！

      一、主料

      主料用量
      紫薯3个
      小米100克
      南瓜1块

      二、辅料

      辅料用量
      大米50克
      红枣6个

      三、做法

      步骤
      1.

      食材准备

      |
      |
      2.

      小米、大米淘洗干净，放适量水大火烧开小火煮20分钟至米无硬心

      |
      |
      3.

      紫薯放入蒸笼中

      |
      |
      4.

      盖盖开大火，蒸熟至筷子可插入

      |
      |
      5.

      紫薯、南瓜去皮切小块

      |
      |
      6.

      米煮至无硬心时，倒入南瓜小火煮10分钟，至南瓜熟透

      |
      |
      7.

      倒入蒸熟的紫薯 煮两三分钟

      |
      |
      8.

      煮至紫薯变软

      |
      |
      9.

      加入少许红枣即可出锅

      |
