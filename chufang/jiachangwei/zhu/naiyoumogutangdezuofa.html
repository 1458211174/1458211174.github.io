      奶油蘑菇汤的做法

      因为今天做了牛排，所以做一道奶油蘑菇汤搭配，不是很难，宝宝们可以试一下。只是自己随意做的，并不是十分的专业。

      一、主料

      主料用量
      白蘑菇7~8个
      洋葱小半个
      脱脂纯牛奶100毫升
      黄油适量
      面粉适量
      熟牛肉一小块

      二、辅料

      辅料用量
      盐5克
      水1000克
      黑胡椒粉适量

      三、做法

      步骤
      1.

      把洋葱切碎，蘑菇切片，牛肉切丁。

      |
      |
      2.

      锅内加入适量的黄油融化，加入适量的面粉，把面粉炒香。加入纯牛奶，加入适量清水，开小火慢慢搅拌，直至无颗粒！

      |
      |
      3.

      另外用一个锅，加入适量的黄油融化，加入洋葱末，翻炒至金黄色，有香味出来，加入蘑菇片和牛肉粒，加入一点点盐，使蘑菇片入味一点，也可不加，看口味！

      |
      |
      4.

      将炒好的蘑菇片等一起加入锅内，中火加热，慢慢搅拌，直至汤汁变得浓稠！加入适量的盐和胡椒粉，一碗香喷喷的蘑菇汤就出炉啦！(哈哈，我做的比较多)

      |

      四、功效

      抵抗力，消食，降血压，补钙，防癌，延年益寿，促发育，安神。
