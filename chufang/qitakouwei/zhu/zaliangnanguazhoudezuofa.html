      杂粮南瓜粥的做法

      五谷杂粮含有人体必需七大营养素，尤其是矿物质和B族维生素，对于疾病的治疗、身体的保健、甚至美容等，都有莫大的疗效；五谷杂粮粥是由五谷杂粮熬煮出来的粥，是一种很好的养生营养粥。

      一、主料

      主料用量
      薏米20克
      小米20克
      红米20克
      燕麦米20克
      白糯米20克
      黄冰糖30克
      南瓜100克
      大米20克
      黑米20克

      二、辅料

      辅料用量
      水1200毫升

      三、做法

      步骤
      1.

      准备称重准备好；

      |
      |
      2.

      此步骤用料步骤用时：2分钟
      薏米20克

      薏米用冷水浸泡5-8小时，或者洗净放冰箱冷冻结冰；

      |
      |
      3.

      此步骤用料步骤用时：2分钟
      南瓜100克

      南瓜洗净，去皮、去瓢；

      |
      |
      4.

      此步骤用料步骤用时：2分钟
      南瓜100克

      南瓜切成小块状备用；

      |
      |
      5.

      此步骤用料步骤用时：2分钟
      薏米20克
      小米20克
      红米20克
      燕麦米20克
      白糯米20克
      大米20克

      粗粮混合洗净，倒入电饭煲中；

      |
      |
      6.

      此步骤用料步骤用时：1分钟
      水1200毫升

      加入水，盖上盖子，按下煮粥键；

      |
      |
      7.

      |
      |
      8.

      此步骤用料步骤用时：1分钟
      黄冰糖30克
      南瓜100克

      煮至1小时左右，加入切好的南瓜丁、黄冰糖；

      |
      |
      9.

      待自动煮好后即可食用啦。

      |

      四、功效

      防癌，美白美容，抵抗力，糖尿病，解热镇痛，治脚气病，清热去火，助消化，强身健体，开胃消食，抗感染，预防流产，维持性功能，祛斑美容，促进生长，助胎儿发育，治流鼻血，止咳化痰，养阴生。
