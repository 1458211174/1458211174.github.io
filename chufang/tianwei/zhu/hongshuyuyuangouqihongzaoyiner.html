      红薯芋圆枸杞红枣银耳羹的做法

      润肺美容，滋润暖身的一道简单的甜品，非常适合干燥又雾霾的冬天 ，而且还特别添加了自己手作的红薯芋圆，鲜芋仙那种，软软糯糯的口感，更加好吃，小朋友也会很喜欢，而且自制芋圆可以控制甜度和避免不好的添加剂，更健康，而且制作过程很简单，就是稍微花点时间。

      一、主料

      主料用量
      银耳3-4小块
      红枣3颗
      枸杞10颗
      红薯1个

      二、辅料

      辅料用量
      木薯淀粉6勺
      糖粉少量
      冰糖适量

      三、做法

      步骤
      1.

      看银耳大小，大概4块左右，用清水泡发，大概需要半个小时，下锅前撕成小块，顺便洗一下大枣和枸杞，主要是去去土，这两样不要泡，要不有效成分就泡出去了。

      |
      |
      2.

      趁着泡发银耳的时间蒸红薯，红薯洗净，切开或者切成小块，切的越小越好蒸软。

      |
      |
      3.

      把洗净的银耳撕成小块，洗净的红枣最好对半切开，方便味道和营养的溶出，放入砂锅中，多加点儿水，炖久一些更好喝，最好一次把水加够，加入冰糖，冰糖多少依个人口味而定就好。

      |
      |
      4.

      砂锅大火煮开后，加入洗净的枸杞，枸杞不用加太早，避免有效成分被破坏，换小火慢炖半个小时以上，会越炖越稠。

      |
      |
      5.

      在小火炖银耳的过程中，可以做红薯芋圆啦。将蒸好的红薯去皮，放凉捣碎，加入木薯淀粉和少量糖粉，拌匀，木薯淀粉多，口感就更糯更容易成型，但红薯味道就会变淡，相反，红薯味更浓，但就没那么弹牙。我是选择少量多次加木薯淀粉，到能成型就行，不要加太多淀粉，不要掩盖食材本身的味道更好。

      |
      |
      6.

      基本到这个程度就能成型了，不过我这样还是有点稀了，再干点儿会更好，更糯一些。

      |
      |
      7.

      另起一个灶，烧开水，下圆子，飘起来就行了，捞出来加入旁边的炖银耳的砂锅里。我圆子做得多，就没有都加入银耳羹里，留了一些，多煮了一会儿，单独盛出来也是很不错的一道甜品哦。

      |
      |
      8.

      加入圆子后，再煮几分钟红薯芋圆红枣枸杞银耳羹就完成了。看起来不错吧，吃不完的可以用保鲜盒盛好放冷藏，之后可以吃凉的，或者蒸一下吃都很方便哦。

      |

      四、功效

      保护肝脏，健胃消食，抵抗力，美容祛斑，减肥瘦身，强身健体，治流鼻血，保护视力，保肝，降三高，防癌，通便，抗衰老。
