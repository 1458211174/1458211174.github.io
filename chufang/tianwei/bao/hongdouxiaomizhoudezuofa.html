      红豆小米粥的做法

      红豆是补血的好食材，用来煮粥是我最喜欢的吃法。现在的电饭煲都有预约功能，晚上下米预约好，早上起床就能吃了，十分方便

      一、主料

      主料用量
      小米50克
      红豆50克
      黑米50克
      大米50克

      二、辅料

      辅料用量
      白糖20克
      水适量

      三、做法

      步骤
      1.

      红豆黑豆洗净，放入没过红豆的水。放入冰箱冷冻格，让其冷冻结冰

      |
      |
      2.

      这是冻过的红豆黑豆

      |
      |
      3.

      电饭煲中，放入大米小米

      |
      |
      4.

      再把红豆敲出来连同冰块起放入锅中，再加入适量水

      |
      |
      5.

      盖上盖子，选择煮粥功能，预约8小时，让其自动工作就好

      |
      |
      6.

      八小时后，粥已煮好，趁热加入糖调味即可

      |
      |
      7.

      熬好的粥~

      |
