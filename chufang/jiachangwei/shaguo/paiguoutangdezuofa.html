      排骨藕汤的做法

      排骨藕汤是一道家常菜，排骨含蛋白质补钙。藕可清热消痰、补血养颜的作用，若有贫血、心慌失眠的人吃了可以调节这些病。 在春节吃上这样的汤很是可口。祝愿大家鸡年如意。

      一、主料

      主料用量
      排骨500克
      浓汤宝1块

      二、辅料

      辅料用量
      藕500克
      姜2块

      三、做法

      步骤
      1.

      备好食材

      |
      |
      2.

      排骨焯下水去血腥

      |
      |
      3.

      捞出放砂锅

      |
      |
      4.

      藕切片放砂锅

      |
      |
      5.

      浓汤宝一盒放入砂锅加水淹过藕片多一点

      |
      |
      6.

      先大火烧开中小火二十分钟即可放点盐

      |

      四、功效

      补肾，润肺。
