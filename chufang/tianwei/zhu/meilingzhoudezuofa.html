      美龄粥的做法

      2016年的某一天，十余年后南京深秋的一晚热粥，可饮风霜，可温喉…念念不忘

      一、主料

      主料用量
      黄豆150克
      粳米100克
      百合20克
      山药150克
      冰糖20克

      二、辅料

      辅料用量
      枸杞子适量

      三、做法

      步骤
      1.

      黄豆提前浸泡两小时

      |
      |
      2.

      使用豆浆机湿豆浆功能，过滤豆渣后豆浆制成备用，豆浆1200ML。

      |
      |
      3.

      梗米提前浸泡两小时

      |
      |
      4.

      百合、山药洗净

      |
      |
      5.

      山药洗净蒸熟后捣成泥

      |
      |
      6.

      梗米、豆浆一同入锅，熬煮途中加入山药、百合，小火熬煮一小时，最后加入冰糖，整个厨房弥漫着香甜的味道！

      |
      |
      7.

      一碗暖胃香甜的热粥，倍感温暖。

      |

      四、功效

      软化血管，益智，糖尿病，降脂，美白，防癌，脂肪肝，更年期，抵抗力，通便排毒，助消化，防痔疮，散瘀止痛，降压，防糖尿病，治脚气病，去斑，体温，中风，贫血，清热去火，安神美容。
