      红枣银耳莲子羹的做法

      想炖个银耳莲子羹已经很久了，今天周末在家终于动手了

      一、主料

      主料用量
      红枣9颗
      莲子1把
      银耳半个
      干百合1把

      二、辅料

      辅料用量
      冰糖适量

      三、做法

      步骤
      1.

      材料备齐

      |
      |
      2.

      银耳温水泡开

      |
      |
      3.

      银耳撕开去蒂

      |
      |
      4.

      莲子、百合温水浸泡

      |
      |
      5.

      莲子去芯

      |
      |
      6.

      红枣洗净备用

      |
      |
      7.

      银耳、莲子放入炖盅，本体跟炖盅都加入烧好的开水，可以开始炖了哦⸜( ´ ꒳ ` )⸝♡︎

      |
      |
      8.

      银耳炖了一个半小时后加入百合继续炖半个小时；之后加入红枣、冰糖再同炖一小时

      |
      |
      9.

      出锅前加入枸杞即可

      |
