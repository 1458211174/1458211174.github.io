      罗宋汤的做法

      有条件的可以做做看，好吃到完全可以治愈各种不吃番茄不吃芹菜不吃洋葱的人～

      一、主料

      主料用量
      牛肉300克
      西芹4瓣
      洋葱半个
      土豆一个
      胡萝卜1个
      西红柿三个

      二、辅料

      辅料用量
      黄油1小块
      葱姜适量
      盐适量
      番茄酱半碗
      胡椒粉1茶匙

      三、做法

      步骤
      1.

      罗宋汤的水是要用煮好的牛肉汤来做，买碎牛肉丁用水浸泡二十分钟出血水，葱姜切片

      |
      |
      2.

      牛肉洗净放进高压锅内，加葱姜加水，水量是牛肉的四到五倍即可

      |
      |
      3.

      选煮汤

      |
      |
      4.

      罗宋汤最麻烦的就是切丁，三四个番茄，一个胡萝卜，半个洋葱（切的时候不要离太近会哭的），一个土豆，还有四五瓣芹菜（不要太老的芹菜），番茄丁切好单放一碗，其他材料另放一碗

      |
      |
      5.

      等牛肉汤做好以后去除葱姜加入煲汤锅，炒锅加油适量，大火到油开以后加番茄丁加两大勺番茄酱炒出红汁（要有一点耐心，番茄越多汤越酸，不喜欢酸的就少加一点也可以不加），然后加入牛汁的煲汤锅里，大火烧开，中火10分钟

      |
      |
      6.

      抄起他蔬菜丁要用黄油，黄油块下锅到完全融化，倒入其他蔬菜丁

      |
      |
      7.

      这个时候也要有一点耐心，要不停翻炒将芹菜和胡萝卜炒至断生才好，不然炖汤后会有一点硬

      |
      |
      8.

      炒好的蔬菜丁入锅，中火40分钟，加盐，再五分钟出锅

      |
      |
      9.

      可以开动啦

      |
