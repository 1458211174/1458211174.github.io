      红苋菜皮蛋汤的做法

      红苋菜它含有丰富的铁、钙和维生素K，具有促进凝血，增加血红蛋白含量并提高携氧能力，促进造血等功能。

      一、主料

      主料用量
      红苋菜一把
      皮蛋一颗

      二、辅料

      辅料用量
      大蒜子5颗
      食用油适量
      盐适量

      三、做法

      步骤
      1.

      红苋菜一把

      |
      |
      2.

      摘掉根部和老的地方，掐成段，清洗干净，多洗多漂，这个菜很容易有沙子

      |
      |
      3.

      皮蛋和大蒜

      |
      |
      4.

      切碎

      |
      |
      5.

      开大火，锅中倒入食用油，炒香蒜蓉

      |
      |
      6.

      倒入红苋菜翻炒一下

      |
      |
      7.

      放水煮

      |
      |
      8.

      水要莫过菜

      |
      |
      9.

      水开的时候加入皮蛋，加盐继续煮

      |
      |
      10.

      再次煮开，皮蛋熟了就可以盛出来了

      |
      |
      11.

      红色的汤，黑色的蛋，白色的蒜，别有一番风味，你试试

      |

      四、功效

      去火，抵抗力，利尿。
