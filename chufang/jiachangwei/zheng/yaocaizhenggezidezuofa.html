      药材蒸鸽子的做法

      自古以来就有”一鸽胜九鸡”的说法。可见鸽子的营养价值有多高。中医认为，鸽肉易于消化，具有滋补益气、祛风解毒的功能，对病后体弱、血虚闭经、头晕神疲、记忆衰退有很好的补益治疗作用。民间称鸽子为”甜血动物”，贫血的人食用后有助于恢复健康。乳鸽的骨内含有丰富的软骨素，可与鹿茸中的软骨素相媳美，经常食用，具有改善皮肤细胞活力，增强度肤弹性，改善血液循环，面色红润等功效。 这道药材蒸鸽子汤有补中益气，养血安神，健脾补益，清肝明目，提高身体免疫力等功效。 儿童要适当进补，提高免疫力，才能少生病。宝宝身体好，妈妈更放心！

      一、主料

      主料用量
      鸽子半只
      黄芪3克
      党参4克
      红枣10克
      鲜莲子10克
      枸杞1克

      二、辅料

      辅料用量
      盐0.5克

      三、做法

      步骤
      1.

      称好药材

      |
      |
      2.

      红枣去核，鲜莲子去皮去芯，党参切段，全部洗干净

      |
      |
      3.

      鸽子处理干净

      |
      |
      4.

      焯水，再冲洗一遍

      |
      |
      5.

      碗里倒入适量清水，放入鸽子、党参、黄芪、红枣、莲子，大火蒸三十分钟

      |
      |
      6.

      放枸杞再蒸十分钟

      |
      |
      7.

      放盐调味，出锅即可

      |

      四、功效

      降压，抵抗力，贫血，通便，心血管病，抗休克，强心，安神助眠，神经衰弱，治流鼻血，保护视力，保肝，降三高。
