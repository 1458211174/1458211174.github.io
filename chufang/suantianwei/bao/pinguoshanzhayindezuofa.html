      苹果山楂饮的做法

      周末的午后来一碗酸甜可口的苹果汤幸福感爆棚

      一、主料

      主料用量
      苹果3个
      山楂适量
      红枣适量

      二、辅料

      辅料用量
      冰糖适量
      玫瑰花适量

      三、做法

      步骤
      1.

      全部主料洗净备用，山楂有新鲜的最好，我没有买到所以用的是山楂干，口味差不多

      |
      |
      2.

      苹果削皮去核切块，红枣去核

      |
      |
      3.

      苹果块，红枣，山楂放入锅中倒入适量清水，水是想多喝点汤就多倒点

      |
      |
      4.

      我用的电饭锅煮40分钟，用汤锅就大火烧开再转中小火煮20至30分钟即可

      |
      |
      5.

      40分钟后加入冰糖及玫瑰花再煮10分钟即可

      |
      |
      6.

      起锅时搓碎一朵玫瑰花撒入汤内装饰

      |

      四、功效

      脂肪肝，防癌，中风，润肠通便，抵抗力，减肥瘦身，降血脂，开胃，止疼，腹泻，抗衰老，产后补虚，治流鼻血。
