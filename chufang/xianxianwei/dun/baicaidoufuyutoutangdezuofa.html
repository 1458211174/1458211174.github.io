      白菜豆腐鱼头汤的做法

      鱼头豆腐汤是道营养丰富且味道鲜美的家常菜，豆腐含钙量较多，而鱼肉中含有维生素D，两者合吃，借助鱼体内维生素D作用，使人体对钙的吸收率提高很多倍。 鱼和豆腐搭在一起很利于营养吸收，而且汤色鲜美，香而不腻。鲜香的鱼头加上嫩滑的豆腐，再有一点白菜味道的点缀，美美喝一碗，滋味妙不可言。 我的微信公众号【暖夫生活】，喜欢的小伙伴可以关注哦~

      一、主料

      主料用量
      鱼头1个
      豆腐200克
      白菜150克

      二、辅料

      辅料用量
      美人椒1个
      线椒1个
      葱15克
      姜5克
      蒜2瓣
      盐适量
      蚝油适量

      三、做法

      步骤
      1.

      准备食材：鱼头去腮洗净，豆腐切成大块，姜切成片，小葱洗净打结

      |
      |
      2.

      鱼头和葱姜蒜同时放进锅内，加水大火焖煮

      |
      |
      3.

      开锅后转中火慢炖，豆腐铺进锅里

      |
      |
      4.

      约15-20分钟后捞出葱结，下白菜

      |
      |
      5.

      加调味料：盐、蚝油，稍微炖煮后即可出锅，出锅前加适量葱花和辣椒末即可

      |

      四、功效

      软化血管，补钙，缓解更年期，预防疾病，防乳腺癌，有益大脑，抵抗力，解渴利尿，润肠通便，消食。
